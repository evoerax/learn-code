## 1940 年代

### Plankalkül (1943-1945)

- **诞生时间**: 1943-1945（发布于 1948）
- **创始人**: Konrad Zuse
- **诞生的初衷**: 作为 Z3 计算机的高级编程语言，是理论上最早的高级编程语言，旨在用数学形式描述计算过程
- **特点**: 包含赋值、循环、条件、函数等概念，从未被实际实现，但被视为现代编程语言的鼻祖
- **知名项目**: 仅存在于理论中，未被实现

### ENIAC 编码系统 (1943-1946)

- **诞生时间**: 1943-1946
- **创始人**: John von Neumann、John Mauchly、J. Presper Eckert、Herman Goldstine
- **诞生的初衷**: 为 ENIAC 计算机编写程序，解决弹道计算问题
- **特点**: 第一个使用存储程序概念的编程系统
- **知名项目**: ENIAC 弹道计算程序

### ENIAC Short Code (1946)

- **诞生时间**: 1946
- **创始人**: Richard Clippinger、John von Neumann
- **诞生的初衷**: 简化 ENIAC 编程，使用更接近数学表示法的语言
- **特点**: 第一个可运行的语言（非机器码），使用解释器执行
- **知名项目**: ENIAC 早期程序

### Short Code / Brief Code (1949)

- **诞生时间**: 1949
- **创始人**: John Mauchly、William F. Schmitt
- **诞生的初衷**: 为 UNIVAC I 计算机提供更易用的编程方式
- **特点**: 早期解释型语言，奠定了高级语言的基础
- **知名项目**: UNIVAC I 程序

## 1950 年代

### FORTRAN (1954-1957)

- **诞生时间**: 1954（概念），1957（实现）
- **创始人**: John W. Backus 及 IBM 团队
- **诞生的初衷**: 为 IBM 704 计算机提供高效的科学计算编程能力，解决手写机器码效率低下的问题
- **特点**: 第一个成功的高级编程语言，Formula Translation 的缩写，现代科学计算的基础
- **知名项目**: NASA 早期太空任务代码、天气预报模型、量子力学模拟

### LISP (1956-1958)

- **诞生时间**: 1956（概念），1958（实现）
- **创始人**: John McCarthy
- **诞生的初衷**: 用于人工智能研究，提供符号处理和递归计算能力
- **特点**: 第一个函数式编程语言，括号语法，奠定了 AI 语言的基础
- **知名项目**: Emacs（编辑器）、AutoCAD（早期版本）、NASA 专家系统

### ALGOL 58 / IAL (1958)

- **诞生时间**: 1958
- **创始人**: ACM/GAMM 委员会
- **诞生的初衷**: 作为国际算法语言，建立通用的编程语言标准
- **特点**: 现代编程语言的父亲，影响了 Pascal、C、Ada 等语言
- **知名项目**: 早期算法描述标准

### COBOL (1959)

- **诞生时间**: 1959
- **创始人**: CODASYL 委员会（Grace Hopper 等）
- **诞生的初衷**: 为商业计算设计通用的编程语言，实现跨平台业务数据处理
- **特点**: 面向商业数据处理，英文式语法，至今仍是金融系统主流语言
- **知名项目**: 银行核心系统、美国社会保障系统、信用卡系统

## 1960 年代

### BASIC (1964)

- **诞生时间**: 1964
- **创始人**: John Kemeny、Thomas Kurtz（Dartmouth College）
- **诞生的初衷**: 为大学生提供易于学习的编程语言，降低编程门槛
- **特点**: Beginner's All-purpose Symbolic Instruction Code，简单易学，影响了 VB、Python
- **知名项目**: Microsoft BASIC、Apple II BASIC、Visual Basic

### PL/I (1964)

- **诞生时间**: 1964（概念），1969（实现）
- **创始人**: IBM
- **诞生的初衷**: 融合 FORTRAN、ALGOL、COBOL 的优点，成为通用编程语言
- **特点**: 第一个多范式语言，支持过程式、面向对象编程
- **知名项目**: IBM OS/2 操作系统部分、航天控制系统

### Simula (1962-1967)

- **诞生时间**: 1962（概念），1967（实现）
- **创始人**: Ole-Johan Dahl、Kristen Nygaard（Norsk Regnesentral）
- **诞生的初衷**: 用于模拟和建模，面向对象编程概念的起源
- **特点**: 第一个面向对象语言，类和对象的概念源于此
- **知名项目**: 早期离散事件模拟系统

### APL (1962-1966)

- **诞生时间**: 1962（概念），1966（实现）
- **创始人**: Kenneth E. Iverson
- **诞生的初衷**: 为数学和数组操作提供简洁的表示法
- **特点**: 使用特殊符号的数组编程语言，数学表示力强大
- **知名项目**: 金融模型、量子计算模拟

### ALGOL 60 (1960)

- **诞生时间**: 1960
- **创始人**: ALGOL 委员会
- **诞生的初衷**: 作为 ALGOL 58 的改进版，建立编程语言标准
- **特点**: 结构化编程语言的基础，影响了 Pascal、C 等
- **知名项目**: 早期编译器设计、算法描述

### COBOL 61 (1960)

- **诞生时间**: 1960
- **创始人**: CODASYL 委员会
- **诞生的初衷**: 完善 COBOL 商业语言
- **知名项目**: 早期银行系统

### RPG (1964)

- **诞生时间**: 1964
- **创始人**: IBM
- **诞生的初衷**: 为 IBM i 系统提供报表生成能力
- **特点**: 报表生成语言
- **知名项目**: IBM i 系统商业应用

### Logo (1968)

- **诞生时间**: 1968
- **创始人**: Wally Feurzeig、Seymour Papert、Cynthia Solomon
- **诞生的初衷**: 为教育目的设计的编程语言，通过绘图教孩子编程
- **特点**: 海龟绘图，寓教于乐
- **知名项目**: LOGO 教学系统、海龟绘图

### Forth (1968)

- **诞生时间**: 1968
- **创始人**: Charles H. Moore
- **诞生的初衷**: 为天文望远镜控制提供高效、灵活的语言
- **特点**: 栈式编程语言，逆波兰表示法
- **知名项目**: 嵌入式系统、航天器控制系统

## 1970 年代

### C (1972)

- **诞生时间**: 1972
- **创始人**: Dennis Ritchie（贝尔实验室）
- **诞生的初衷**: 为 UNIX 操作系统开发提供高效的语言，结合了高级语言和低级语言的优势
- **特点**: 系统编程首选语言，现代编程语言的基础，影响了 C++、Java、C#
- **知名项目**: UNIX 操作系统、Linux 内核、Windows 核心组件、Git、Vim

### Smalltalk (1972)

- **诞生时间**: 1972
- **创始人**: Alan Kay、Adele Goldberg、Xerox PARC
- **诞生的初衷**: 实现"Dynabook"教育计算机的理念，创造面向对象的编程环境
- **特点**: 纯面向对象语言，图形用户界面的先驱，消息传递机制
- **知名项目**: Squeak（葡萄）、EToys（教育）、Seaside（Web 框架）

### Prolog (1972)

- **诞生时间**: 1972
- **创始人**: Alain Colmerauer
- **诞生的初衷**: 用于自然语言处理和逻辑编程，实现基于规则的推理系统
- **特点**: 第一个逻辑编程语言，声明式语言，用于 AI 和专家系统
- **知名项目**: IBM Watson（部分）、自然语言处理系统、专家系统

### SQL (1972)

- **诞生时间**: 1972
- **创始人**: IBM（基于 Edgar F. Codd 的关系数据库理论）
- **诞生的初衷**: 作为关系数据库的标准查询语言，实现数据的结构化查询
- **特点**: 结构化查询语言，所有关系数据库的标准接口
- **知名项目**: MySQL、PostgreSQL、Oracle、SQL Server

### Pascal (1970)

- **诞生时间**: 1970
- **创始人**: Niklaus Wirth
- **诞生的初衷**: 作为教学语言，提供清晰的结构化编程能力，用于编程教育
- **特点**: 结构化编程语言的典范，影响了 Ada、Modula
- **知名项目**: Turbo Pascal、Delphi、早期 Apple Lisa 系统

### ML (1973)

- **诞生时间**: 1973
- **创始人**: Robin Milner（爱丁堡大学）
- **诞生的初衷**: 作为元编程语言，用于定理证明和类型系统研究
- **特点**: 函数式语言，强大的类型推断系统，现代函数式语言的先驱
- **知名项目**: OCaml、SML、F#（受 ML 影响）

### Forth (1970)

- **诞生时间**: 1970
- **创始人**: Charles H. Moore
- **诞生的初衷**: 为天文望远镜控制提供高效、灵活的语言
- **特点**: 栈式编程语言，逆波兰表示法，广泛用于嵌入式系统
- **知名项目**: 嵌入式系统、NASA 航天控制

### Scheme (1975)

- **诞生时间**: 1975
- **创始人**: Gerald Jay Sussman、Guy L. Steele Jr.
- **诞生的初衷**: 作为 LISP 的教学语言，提供简洁、一致的方言
- **特点**: LISP 方言，Lambda 演算的完美实现，影响了 JavaScript
- **知名项目**: Emacs 配置语言、Git（部分）、Racket

### Modula-2 (1979)

- **诞生时间**: 1979
- **创始人**: Niklaus Wirth
- **诞生的初衷**: 作为系统编程语言，模块化和并发编程能力
- **特点**: 继承 Pascal，引入模块概念和协程
- **知名项目**: 早期操作系统、Apple Lisa 系统

### Ada (1980)

- **诞生时间**: 1980
- **创始人**: Jean Ichbiah（CII Honeywell Bull）
- **诞生的初衷**: 为美国国防部开发可信的嵌入式系统编程语言，替代数百种专用语言
- **特点**: 强类型、并发编程、高可靠性，用于航空航天和军事系统
- **知名项目**: F-35 战斗机软件、空中交通管理系统、NASA 火星探测器软件

## 1980 年代

### C++ (1983)

- **诞生时间**: 1983
- **创始人**: Bjarne Stroustrup
- **诞生的初衷**: 为 C 语言添加面向对象特性，同时保持效率和兼容性
- **特点**: 多范式语言，支持面向对象、过程式、泛型编程，系统/应用开发主流语言
- **知名项目**: Microsoft Office、Adobe Photoshop、Google Chrome、Mozilla Firefox、MySQL、Unreal Engine

### Objective-C (1983)

- **诞生时间**: 1983
- **创始人**: Brad Cox
- **诞生的初衷**: 为 C 添加面向对象能力和消息传递机制，用于 NextSTEP 操作系统
- **特点**: C + Smalltalk 语法，用于 macOS/iOS 开发（苹果系统）
- **知名项目**: macOS 系统、iOS 应用、NextSTEP 操作系统

### Perl (1987)

- **诞生时间**: 1987
- **创始人**: Larry Wall
- **诞生的初衷**: 为文本处理和系统管理提供更强大的工具，Unix 系统管理
- **特点**: 强大的正则表达式和文本处理能力，"有不止一种方法做到"的哲学
- **知名项目**: Wikipedia（早期）、Amazon、CGI 脚本、DevOps 工具

### Haskell (1990)

- **诞生时间**: 1990
- **创始人**: Haskell 委员会
- **诞生的初衷**: 作为纯函数式编程语言的学术研究平台，推广函数式编程
- **特点**: 纯函数式、惰性求值、强大的类型系统，函数式编程的代表
- **知名项目**: Cabal（包管理）、Xmonad（窗口管理器）、Pandoc（文档转换）

### Visual Basic (1991)

- **诞生时间**: 1991
- **创始人**: Microsoft
- **诞生的初衷**: 为 Windows 应用开发提供可视化编程能力，降低开发门槛
- **特点**: 可视化开发环境，事件驱动编程，微软生态主力语言
- **知名项目**: 早期 Windows 桌面应用、Office 宏（VBA）

## 1990 年代

### Python (1991)

- **诞生时间**: 1991
- **创始人**: Guido van Rossum
- **诞生的初衷**: 作为 ABC 语言的继承者，注重可读性和生产率
- **特点**: 简洁优雅的语法，多范式，AI 和数据科学首选语言
- **知名项目**: Instagram、YouTube、Dropbox、NASA、Spotify、Pinterest、Reddit

### R (1993)

- **诞生时间**: 1993
- **创始人**: Ross Ihaka、Robert Gentleman（奥克兰大学）
- **诞生的初衷**: 为统计计算和图形分析提供语言环境，S 语言的方言
- **特点**: 统计计算和数据分析的首选语言，丰富的统计包
- **知名项目**: Google Analytics、Tidyverse、ggplot2

### PHP (1995)

- **诞生时间**: 1995
- **创始人**: Rasmus Lerdorf
- **诞生的初衷**: 为个人主页提供动态内容生成能力，Personal Home Page Tools
- **特点**: Web 开发主流语言，服务端脚本，LAMP 栈核心
- **知名项目**: WordPress、Facebook（早期）、Wikipedia、Laravel

### Java (1995)

- **诞生时间**: 1995
- **创始人**: James Gosling（Sun Microsystems）
- **诞生的初衷**: 为消费电子设备提供跨平台编程能力，Write Once, Run Anywhere
- **特点**: 面向对象、跨平台、JVM 驱动，企业级应用主流语言
- **知名项目**: Android 应用（Kotlin 之前）、Hadoop、Minecraft、Eclipse、IntelliJ IDEA

### JavaScript (1995)

- **诞生时间**: 1995
- **创始人**: Brendan Eich（Netscape）
- **诞生的初衷**: 为网页提供动态交互能力，浏览器脚本语言
- **特点**: 原型继承、事件驱动、异步编程，Web 前端唯一语言（Node.js 后端）
- **知名项目**: Gmail、Google Maps、Facebook、Netflix、Node.js、React、Vue

### Ruby (1995)

- **诞生时间**: 1995
- **创始人**: Yukihiro Matsumoto
- **诞生的初衷**: 提供人性化、面向对象的脚本语言，注重开发者幸福感
- **特点**: 纯面向对象、简洁优雅、RailS 框架，Web 开发利器
- **知名项目**: Ruby on Rails、GitHub（早期）、Airbnb、Stripe

### C# (2000)

- **诞生时间**: 2000
- **创始人**: Microsoft（Anders Hejlsberg 团队）
- **诞生的初衷**: 作为 Java 的微软版本，提供现代编程特性，.NET 平台核心
- **特点**: 面向对象、强类型、现代语言特性，Windows 应用和游戏开发主流
- **知名项目**: .NET Framework、Unity 游戏引擎、Visual Studio、Windows 桌面应用

## 2000 年代

### Scala (2003)

- **诞生时间**: 2003
- **创始人**: Martin Odersky（洛桑联邦理工学院）
- **诞生的初衷**: 融合面向对象和函数式编程，运行于 JVM
- **特点**: 静态类型、函数式 + 面向对象、并发编程，分布式系统和大数据处理
- **知名项目**: Apache Spark、Apache Kafka、Twitter（早期）、LinkedIn

### Groovy (2003)

- **诞生时间**: 2003
- **创始人**: James Strachan
- **诞生的初衷**: 作为 Java 平台的脚本语言，简化 Java 开发
- **特点**: 动态语言，运行于 JVM，与 Java 无缝互操作
- **知名项目**: Gradle（构建工具）、Jenkins 脚本、Spock 测试框架

### ActionScript (2000)

- **诞生时间**: 2000
- **创始人**: Macromedia
- **诞生的初衷**: 为 Flash 动画和应用提供编程能力
- **特点**: 基于 ECMAScript，Flash/AIR 开发，RIA 应用
- **知名项目**: Flash 游戏、Adobe AIR 应用

### Lua (1993)

- **诞生时间**: 1993（广泛使用于 2000 年代）
- **创始人**: Roberto Ierusalimschy、Luiz Henrique de Figueiredo、Waldemar Celes
- **诞生的初衷**: 作为嵌入式脚本语言，用于游戏和应用程序扩展
- **特点**: 轻量级、嵌入性强、运行效率高，游戏脚本首选
- **知名项目**: World of Warcraft、Roblox、Lua（Adobe Lightroom、Nginx）

### Processing (2001)

- **诞生时间**: 2001
- **创始人**: Casey Reas、Ben Fry（MIT Media Lab）
- **诞生的初衷**: 为视觉艺术和设计提供编程环境，电子艺术和可视化
- **特点**: Java 方言，简化的图形 API，视觉艺术教育工具
- **知名项目**: 数字艺术作品、数据可视化

### F# (2005)

- **诞生时间**: 2005
- **创始人**: Microsoft Research（Don Syme）
- **诞生的初衷**: 为 .NET 平台提供函数式编程能力
- **特点**: 函数式 + 面向对象，.NET 平台函数式语言，数据科学和金融
- **知名项目**: 金融模型、微软内部工具

### Go (2009)

- **诞生时间**: 2009
- **创始人**: Robert Griesemer、Rob Pike、Ken Thompson（Google）
- **诞生的初衷**: 为 Google 基础设施提供高效、简洁的系统编程语言
- **特点**: 编译快速、并发原生、静态类型，云原生和微服务首选
- **知名项目**: Docker、Kubernetes、Terraform、Goole Cloud、Discord、Dropbox

### CoffeeScript (2009)

- **诞生时间**: 2009
- **创始人**: Jeremy Ashkenas
- **诞生的初衷**: "Python 语法"编译为 JavaScript，解决 JavaScript 语言设计问题
- **特点**: 语法糖，编译为 JavaScript，React 开发早期流行
- **知名项目**: 早期 Backbone.js 项目（已较少使用）

### Clojure (2007)

- **诞生时间**: 2007
- **创始人**: Rich Hickey
- **诞生的初衷**: 为 JVM 提供现代 Lisp，函数式编程和数据处理
- **特点**: Lisp 方言，运行于 JVM，函数式和并发编程
- **知名项目**: Apache Spark（部分）、Nerv（股票交易系统）

### Rust (2010)

- **诞生时间**: 2010
- **创始人**: Graydon Hoare（Mozilla）
- **诞生的初衷**: 提供内存安全且高效的系统编程语言，解决 C/C++ 的内存安全问题
- **特点**: 所有权系统、内存安全、零成本抽象，系统编程和 WebAssembly
- **知名项目**: Firefox（部分组件）、Discord、Ripgrep、Dropbox（核心组件）、Bolt

## 2010 年代

### TypeScript (2012)

- **诞生时间**: 2012
- **创始人**: Anders Hejlsberg（Microsoft）
- **诞生的初衷**: 为 JavaScript 添加静态类型系统，大型项目开发
- **特点**: JavaScript 超集，静态类型，IDE 支持，现代 Web 开发标准
- **知名项目**: VS Code、Angular、Vue 3、React（企业级）、Azure DevOps

### Swift (2014)

- **诞生时间**: 2014
- **创始人**: Apple（Chris Lattner 团队）
- **诞生的初衷**: 为苹果生态系统提供现代、安全、高效的编程语言，替代 Objective-C
- **特点**: 安全、快速、现代化，iOS/macOS 开发首选
- **知名项目**: iOS 应用、macOS 应用、SwiftUI、Airbnb iOS 应用

### Kotlin (2011)

- **诞生时间**: 2011
- **创始人**: JetBrains
- **诞生的初衷**: 为 JVM 提供更现代、更简洁的语言，解决 Java 冗长问题
- **特点**: 简洁语法、空安全、互操作，Android 开发首选
- **知名项目**: Android 应用（官方推荐）、Spring Boot、Kotlin Multiplatform

### Elixir (2011)

- **诞生时间**: 2011
- **创始人**: José Valim
- **诞生的初衷**: 利用 Erlang VM 的并发能力，提供更现代化的语法
- **特点**: 函数式、元编程、分布式，高并发和电信系统
- **知名项目**: WhatsApp（早期）、Discord、Pinterest、Erlang/Elixir 生态系统

### Julia (2012)

- **诞生时间**: 2012
- **创始人**: Jeff Bezanson、Stefan Karpinski、Viral B. Shah、Alan Edelman
- **诞生的初衷**: 为数值计算提供高性能语言，同时具备动态语言的生产率
- **特点**: 动态类型、JIT 编译、高性能，科学计算和机器学习
- **知名项目**: 气候模型、药物研发、机器学习框架、NASA

### Dart (2011)

- **诞生时间**: 2011
- **创始人**: Google（Lars Bak、Kasper Lund）
- **诞生的初衷**: 为 Web 应用提供结构化语言，替代 JavaScript
- **特点**: 类型安全、AOT/JIT 编译，Flutter 跨平台开发
- **知名项目**: Flutter 应用、Google Ads、BMW 应用、Alibaba 应用

### Hack (2014)

- **诞生时间**: 2014
- **创始人**: Facebook
- **诞生的初衷**: 为 PHP 添加静态类型，大型 PHP 项目
- **特点**: 类型标注、HHVM，Facebook 内部使用
- **知名项目**: Facebook（内部）、HHVM

### Crystal (2014)

- **诞生时间**: 2014
- **创始人**: Ary Borenszweig
- **诞生的初衷**: 提供 Ruby 语法和 C 性能，兼具开发效率和运行效率
- **特点**: 类型推断、编译语言、Ruby 语法，系统和高性能应用
- **知名项目**: 仍在发展中，Shrine、Crystal Web 框架

### Nim (2008)

- **诞生时间**: 2008
- **创始人**: Andreas Rumpf
- **诞生的初衷**: 提供高效、优雅的系统编程语言，Python 般的语法
- **特点**: 编译为 C/C++/JS，元编程、系统编程
- **知名项目**: Nim 社区项目、Angelos（Web 框架）

### Elm (2012)

- **诞生时间**: 2012
- **创始人**: Evan Czaplicki
- **诞生的初衷**: 为 Web 前端提供纯函数式、可靠的编程语言，前端可靠开发
- **特点**: 函数式、静态类型、编译器友好，前端架构影响深远
- **知名项目**: 早期 Elm 项目、Preact（受 Elm 影响）

## 附录三：补充语言

### AWK (1977)

- **诞生时间**: 1977
- **创始人**: Alfred Aho、Peter Weinberger、Brian Kernighan（贝尔实验室）
- **诞生的初衷**: 为文本处理和报表生成设计的领域特定语言
- **特点**: 行处理强大，正则表达式集成
- **知名项目**: Linux 工具脚本、日志分析、数据抽取

### REXX (1979)

- **诞生时间**: 1979
- **创始人**: Mike Cowlishaw（IBM）
- **诞生的初衷**: 为 IBM 主机的脚本语言，简化系统管理
- **特点**: 易读性强，跨平台脚本
- **知名项目**: IBM OS/2、z/OS 脚本

### MUMPS (1967)

- **诞生时间**: 1967
- **创始人**: Massachusetts General Hospital
- **诞生的初衷**: 为医疗信息系统设计的数据管理和事务处理语言
- **特点**: 嵌入数据库的文件系统
- **知名项目**: 医疗记录系统

### Icon (1977)

- **诞生时间**: 1977
- **创始人**: Ralph Griswold
- **诞生的初衷**: 作为 SNOBOL 的后继者，提供更强大的字符串处理
- **特点**: 描述性编程，字符串操作强大
- **知名项目**: 文本处理工具

### PowerShell (2006)

- **诞生时间**: 2006
- **创始人**: Microsoft（Bruce Payette、Jeff Snover）
- **诞生的初衷**: 为 Windows 系统管理提供强大的自动化框架
- **特点**: 对象管道、cmdlet 系统，Windows 管理首选
- **知名项目**: Windows 系统管理、Exchange Server、Azure 自动化

### MATLAB (1984)

- **诞生时间**: 1984
- **创始人**: Cleve Moler（MathWorks）
- **诞生的初衷**: 为工程和科学计算提供交互式数值计算环境
- **特点**: 矩阵运算、仿真、信号处理
- **知名项目**: MATLAB/Simulink、控制系统设计、图像处理

### Arduino (2005)

- **诞生时间**: 2005
- **创始人**: Massimo Banzi、David Cuartielles
- **诞生的初衷**: 为艺术家和设计师提供简单的微控制器编程工具
- **特点**: 简单易学、硬件集成、嵌入式入门
- **知名项目**: 物联网项目、机器人、教育套件

### Svelte (2016)

- **诞生时间**: 2016
- **创始人**: Rich Harris
- **诞生的初衷**: 编译时框架，无需虚拟 DOM，提供更快的运行时性能
- **特点**: 编译时优化、响应式、极小 bundle

### SCSS/SASS (2006)

- **诞生时间**: 2006
- **创始人**: Hampton Catlin
- **诞生的初衷**: 为 CSS 添加编程能力（变量、嵌套、混合）
- **特点**: CSS 预处理、模块化、函数

- **创始人**: Microsoft
- **诞生的初衷**: 为 ASP 和 Windows 脚本提供简单脚本能力

### Lasso (1996)
