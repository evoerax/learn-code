# Tooling & evaluation

**Scope:** how teams **know** a stack works: curated datasets, benchmark harnesses, regression tests, tracing, drift monitors, and release gates for prompts/config.

---

## Typical artifacts

- **Golden sets** — small, high-signal evaluation slices.
- **Traces** — per-request spans across router, retriever, tools, and model.
- **Reports** — score trends by model version and prompt hash.
