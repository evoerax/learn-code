## Featured Languages

Some languages that shaped the modern world:

- **C (1972)** — The foundation of operating systems. Still powering Linux, macOS, and Windows kernels today.
- **Lisp (1958)** — The oldest language still in active use. Pioneered garbage collection, dynamic typing, and metaprogramming.
- **Smalltalk (1972)** — Invented object-oriented programming as we know it. Inspired Java, Python, Ruby, and Swift.
- **Prolog (1972)** — Brought logic programming to life. Shaped AI research and influenced constraint-solving systems.
- **Python (1991)** — Became the lingua franca of data science and AI. The most taught language in universities worldwide.
- **JavaScript (1995)** — Created in 10 days, now runs on 99% of websites and powers server-side, mobile, and desktop apps.
- **Rust (2015)** — Solved memory safety without garbage collection. The most-loved language in Stack Overflow surveys for nine consecutive years.

## About This Project

Programming languages are not just tools — they are crystallized ideas about how humans should communicate with machines. Each language embodies a set of beliefs about correctness, expressiveness, and what programs should look like.

This guide attempts to tell that story: not just *what* each language does, but *why* it was created, *what problem* it was solving, and *how* it influenced the languages that came after it.
