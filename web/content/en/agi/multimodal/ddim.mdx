# DDIM

DDIM (Song et al., 2021) enables 10-50x faster sampling by generalizing the diffusion process.

---

## Markov vs Non-Markov

```python
# DDPM (Markov): q(x_{t-1} | x_t, x_0) is Gaussian
# DDIM: q(x_{t-1} | x_t, x_0) can be ANY distribution

def ddim_step(x_t, t, t_prev, noise_pred, eta=0.0):
    """
    eta controls the stochasticity:
    - eta=0: Deterministic (DDIM)
    - eta=1: Equivalent to DDPM
    """
    alpha = sqrt(alpha_bar[t])
    alpha_prev = sqrt(alpha_bar[t_prev])
    
    # Predicted x_0
    x_0_pred = (x_t - sqrt(1-alpha_bar[t]) * noise_pred) / alpha
    
    # Direction pointing to x_t
    dir_xt = sqrt(1 - alpha_bar[t_prev] - eta * sigma**2) * noise_pred
    
    # Random noise (0 when eta=0)
    sigma = eta * sqrt(1 - alpha_bar[t_prev])
    
    x_{t_prev} = alpha_prev * x_0_pred + dir_xt + sigma * torch.randn_like(x_t)
    
    return x_{t_prev}
```

## eta Parameter

| eta | Sampling | Diversity |
|-----|----------|-----------|
| 0.0 | Deterministic | Lower |
| 0.5 | Semi-stochastic | Balanced |
| 1.0 | Full stochastic (DDPM) | Higher |
