# ViT Vision Transformer

ViT (Dosovitskiy et al., 2021) applies the transformer architecture directly to image patches.

---

## Architecture

```python
class ViT(nn.Module):
    def __init__(self, image_size=224, patch_size=16, dim=768, num_heads=12, num_layers=12):
        super().__init__()
        self.patch_size = patch_size
        
        # Patch embedding
        num_patches = (image_size // patch_size) ** 2
        self.patch_embed = nn.Conv2d(3, dim, patch_size, patch_size)
        
        # CLS token
        self.cls_token = nn.Parameter(torch.randn(1, 1, dim))
        
        # Positional embedding
        self.pos_embed = nn.Parameter(torch.randn(1, num_patches + 1, dim))
        
        # Transformer blocks
        self.blocks = nn.ModuleList([
            TransformerBlock(dim, num_heads) for _ in range(num_layers)
        ])
        
        # Classification head
        self.head = nn.Linear(dim, num_classes)
    
    def forward(self, x):
        # Patchify: (B, 3, 224, 224) -> (B, 196, 768)
        x = self.patch_embed(x).flatten(2).transpose(1, 2)
        
        # Add CLS token
        cls_tokens = self.cls_token.expand(x.size(0), -1, -1)
        x = torch.cat([cls_tokens, x], dim=1)
        
        # Add positional embeddings
        x = x + self.pos_embed
        
        # Transformer blocks
        for block in self.blocks:
            x = block(x)
        
        # Classify from CLS token
        return self.head(x[:, 0])
```

## Patch Embedding

```
Image (224x224, RGB)
        │
        ▼
┌───┬───┬───┬───┐
│ P │ P │ P │ P │  16x16 patches
├───┼───┼───┼───┤  → Flatten → 196 patches of 768-dim
│ P │ P │ P │ P │
├───┼───┼───┼───┤
│ P │ P │ P │ P │
└───┴───┴───┴───┘
```

## vs CNNs

| Aspect | ViT | CNN |
|--------|-----|-----|
| **Inductive bias** | Low (patch-level) | High (locality) |
| **Data needed** | More | Less |
| **Compute** | Higher for small models | Lower |
| **Fine-tuning** | Flexible | May need redesign |
| **Attention** | Global from start | Progressive |
