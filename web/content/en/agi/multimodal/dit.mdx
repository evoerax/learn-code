# DeiT Data-efficient Image Transformer

DeiT (Touvron et al., 2021) improves ViT training efficiency through knowledge distillation.

---

## Distillation Token

```python
class DeiT(nn.Module):
    def __init__(self, ...):
        super().__init__()
        # Standard ViT components
        self.patch_embed = nn.Conv2d(3, dim, patch_size, patch_size)
        self.cls_token = nn.Parameter(torch.randn(1, 1, dim))
        self.pos_embed = nn.Parameter(torch.randn(1, num_patches + 1, dim))
        
        # Distillation token (learnable)
        self.dist_token = nn.Parameter(torch.randn(1, 1, dim))
        
        self.blocks = nn.ModuleList([TransformerBlock(dim, num_heads) ...])
        self.head = nn.Linear(dim, num_classes)
        self.dist_head = nn.Linear(dim, num_classes)  #蒸馏头
    
    def forward(self, x):
        x = self.patch_embed(x).flatten(2).transpose(1, 2)
        
        cls_tokens = self.cls_token.expand(x.size(0), -1, -1)
        dist_tokens = self.dist_token.expand(x.size(0), -1, -1)
        x = torch.cat([cls_tokens, dist_tokens, x], dim=1)
        
        x = x + self.pos_embed
        
        for block in self.blocks:
            x = block(x)
        
        # Two classification heads
        return self.head(x[:, 0]), self.dist_head(x[:, 1])
```

## vs ViT

| Aspect | ViT | DeiT |
|--------|-----|------|
| **Training data** | ImageNet-21k or JFT-300M | ImageNet-1k |
| **Teacher** | None | CNN (e.g., RegNet) |
| **Distillation** | None | Soft/hard |
| **Training time** | High | Reduced 3x |

---

## Hard vs Soft Distillation

```
Soft: KL(teacher_prob || student_prob)
Hard: CrossEntropy(teacher_label || student_logit)
```

DeiT uses hard distillation with a CNN teacher, simpler and often more effective.
