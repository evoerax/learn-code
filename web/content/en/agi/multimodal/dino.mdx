# DINO Self-distillation

DINO (Caron et al., 2021) is a self-supervised method using momentum teacher and centering.

---

## Architecture

```python
class DINO(nn.Module):
    def __init__(self, student, teacher, dim=2048, K=65536):
        self.student = student  # online network
        self.teacher = teacher  # momentum network
        
        # Teacher: stop-gradient + momentum update
        self.teacher.momentum_update()
        
        # Center and temperature for teacher output
        self.register_buffer('center', torch.zeros(1, dim))
        self.K = K  # queue size for distillation
    
    def forward(self, student_output, teacher_output):
        # Student: standard softmax
        student_prob = F.softmax(student_output / T_s, dim=-1)
        
        # Teacher: softmax after centering
        teacher_prob = F.softmax((teacher_output - self.center) / T_t, dim=-1)
        
        # Cross-entropy distillation loss
        loss = -torch.sum(teacher_prob * torch.log(student_prob), dim=-1)
        return loss.mean()
```

## Key Components

### Momentum Teacher

```python
# Exponential moving average update
def momentum_update(self, m=0.999):
    for ps, pt in zip(self.student.parameters(), self.teacher.parameters()):
        pt.data = m * pt.data + (1 - m) * ps.data
```

### Centering

```python
# Running center prevents collapse
self.center = m * self.center + (1 - m) * teacher_output.mean(0)
```

## vs MoCo

| Aspect | MoCo | DINO |
|--------|------|------|
| **Teacher** | Key encoder (momentum) | Student+center |
| **Loss** | Contrastive (InfoNCE) | Cross-entropy |
| **Queue** | Dynamic dictionary | None (online) |
