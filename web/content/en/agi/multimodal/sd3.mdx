# SD3 and FLUX

The latest generation of high-quality diffusion models.

## Stable Diffusion 3 (SD3)

| Feature | Value |
|---------|-------|

```python
class MMDiT(nn.Module):
    def __init__(self):
        # Separate transformers for text and image
        self.text_transformer = Transformer()
        self.image_transformer = Transformer()
        
        # Cross-attention between modalities
        # ...
    
    def forward(self, x_t, t, text_emb):
        # Process in parallel, combine at attention
        h_text = self.text_transformer(text_emb, t)
        h_image = self.image_transformer(x_t, t)
        
        # Joint attention
        h = joint_attention(h_text, h_image)
        
        return self.final_layer(h)
```

## FLUX

| Feature | Value |
|---------|-------|
