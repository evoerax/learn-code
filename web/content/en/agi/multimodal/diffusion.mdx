# Diffusion Models

Diffusion models generate data by gradually denoising a random signal.

---

## Forward Process (Noising)

```python
def forward_process(x_0, t, beta_schedule):
    """
    Add noise to image over t timesteps
    x_0: clean image
    t: timestep (0 = clean, T = max noise)
    """
    alpha_bar = (1 - beta_schedule[:t]).prod()
    noise = torch.randn_like(x_0)
    
    x_t = sqrt(alpha_bar) * x_0 + sqrt(1 - alpha_bar) * noise
    return x_t, noise
```

## Reverse Process (Denoising)

```python
class UNet(nn.Module):
    def forward(self, x_t, t):
        # t is timestep embedding

```

## DDPM vs Score Matching

| Aspect | DDPM | Score Matching |
|--------|------|---------------|
| **Objective** | Predict noise ε | Predict score ∇log p(x) |
| **Loss** | MSE(ε - εθ) | MSE(||∇log p - sθ||²) |
| **Sampling** | Many steps | Fewer with SDE |
