# PPO

PPO (Schulman et al., 2017) constrains policy updates to prevent destructively large changes.

---

## Clipped Surrogate Objective

```python
def ppo_loss(old_log_probs, new_log_probs, advantages, clip_eps=0.2):
    """
    PPO's key innovation: clip the probability ratio
    """
    ratio = torch.exp(new_log_probs - old_log_probs)
    
    # Clipped objective
    clipped = torch.clamp(ratio, 1 - clip_eps, 1 + clip_eps)
    
    # Take minimum to prevent large updates
    loss = -torch.min(
        ratio * advantages,
        clipped * advantages
    )
    
    return loss.mean()
```

## Why It Works

```
Without clipping:      With clipping:
                      
L(θ) = r(θ) * A       L(θ) = min(r(θ), clip(r(θ), 1-ε, 1+ε)) * A

r(θ) = π_new/π_old

If r(θ) becomes too large (>1+ε), gradient → 0
This prevents destructive updates!
```
