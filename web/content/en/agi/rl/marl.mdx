# Multi-Agent RL

MARL involves multiple agents learning simultaneously.

---

## Types

| Type | Description |
|------|-------------|
| **Cooperative** | Agents maximize shared reward |
| **Competitive** | Zero-sum games |
| **Mixed** | Both cooperation and competition |

---

## Centralized vs Decentralized

| Approach | Training | Execution |
|----------|----------|-----------|
| **CTDE** | Centralized critic | Decentralized actors |
| **Fully centralized** | All info available | Centralized only |
| **Fully decentralized** | Local info only | Decentralized |
