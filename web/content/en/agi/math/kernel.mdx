# Kernel Methods and SVM

## Kernel Trick

```python
# Instead of computing Φ(x) explicitly:
K(x, z) = Φ(x) · Φ(z)

# Common kernels:
def linear_kernel(x, z):
    return x @ z

def polynomial_kernel(x, z, d=3):
    return (x @ z) ** d

def rbf_kernel(x, z, sigma=1.0):
    return exp(-||x - z||² / (2 * sigma²))
```

## Support Vector Regression

```python
def svr_loss(predictions, targets, alphas, X, epsilon=0.1):
    # ε-insensitive loss: only penalize errors > ε
    errors = (predictions - targets).abs()
    loss = torch.clamp(errors - epsilon, min=0)
    return loss.sum()
```
