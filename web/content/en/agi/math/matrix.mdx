# Matrix Exponential

expm(A) = I + A + A²/2! + A³/3! + ...

---

## Applications

| Application | Use |
|------------|-----|
| **Lie groups** | SO(3), SE(3) for robotics |
| **Neural networks** | Curve networks |
| **Graph networks** | Skip connections |

---

## Computation

```python
# PyTorch
from torch.linalg import matrix_exp

# Compute expm
A_exp = matrix_exp(A)

# For small matrices, series works:
def expm_series(A, n=20):
    result = torch.eye(A.size(0))
    A_pow = A
    for k in range(1, n):
        result = result + A_pow / math.factorial(k)
        A_pow = A_pow @ A
    return result
```
