# YOLO

YOLO (You Only Look Once) enables real-time object detection.

---

## YOLOv1 Architecture

```python
class YOLOv1(nn.Module):
    def __init__(self, num_classes=20, num_boxes=2, grid_size=7):
        super().__init__()
        self.backbone = nn.Sequential(
            nn.Conv2d(3, 64, 7, stride=2, padding=3),
            nn.BatchNorm2d(64), nn.ReLU(),
            nn.MaxPool2d(2, 2),
            # ... more layers
        )
        
        # Detection head
        self.head = nn.Sequential(
            nn.Conv2d(1024, 1024, 3, padding=1),
            nn.Conv2d(1024, grid_size * grid_size * (5*num_boxes + num_classes), 1)
        )
    
    def forward(self, x):
        features = self.backbone(x)
        predictions = self.head(features)
        # predictions: (B, S*S*(5*B + C), H, W)
        return predictions
```
