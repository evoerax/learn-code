# Batch Normalization

BatchNorm normalizes activations to reduce internal covariate shift.

---

## Algorithm

```python
class BatchNorm2d(nn.Module):
    def __init__(self, num_features, eps=1e-5, momentum=0.1):
        super().__init__()
        self.eps = eps
        self.momentum = momentum
        self.weight = nn.Parameter(torch.ones(num_features))
        self.bias = nn.Parameter(torch.zeros(num_features))
        self.register_buffer('running_mean', torch.zeros(num_features))
        self.register_buffer('running_var', torch.ones(num_features))
    
    def forward(self, x):
        if self.training:
            mean = x.mean(dim=(0, 2, 3))
            var = x.var(dim=(0, 2, 3))
            
            # Update running stats
            self.running_mean = (1 - self.momentum) * self.running_mean + self.momentum * mean
            self.running_var = (1 - self.momentum) * self.running_var + self.momentum * var
        else:
            mean = self.running_mean
            var = self.running_var
        
        # Normalize
        x_norm = (x - mean.view(1, -1, 1, 1)) / sqrt(var.view(1, -1, 1, 1) + self.eps)
        
        # Scale and shift
        return self.weight.view(1, -1, 1, 1) * x_norm + self.bias.view(1, -1, 1, 1)
```

---

## Benefits

| Benefit | Description |
|---------|-------------|
| **Faster convergence** | Reduced internal covariate shift |
| **Gradient flow** | Normalized inputs improve gradient propagation |
| **Regularization** | Mini-batch statistics add noise |
| **Higher lr** | Can use higher learning rates |

---

## Issues

| Issue | Solution |
|-------|----------|
| **Small batch** | BatchNorm degrades with small batches |
| **NLP** | LayerNorm instead |
| **Inference** | Running statistics may not match |
