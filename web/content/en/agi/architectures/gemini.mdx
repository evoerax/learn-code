# Gemini Series

Google's native multimodal models.

---

## Evolution

| Model | Year | Capabilities |
|-------|------|-------------|
| **Gemini 1.0** | 2023 | Text, image, audio, video |
| **Gemini 1.5** | 2024 | 1M token context |
| **Gemini 2.0** | 2024 | Agentic, native tool use |
| **Gemini 2.5** | 2025 | Enhanced reasoning |

---

## Architecture

- **Native multimodality**: Single model processes all modalities
- **Long context**: Up to 2M tokens (1.5)
- **Tool use**: Native code execution, Google Search
