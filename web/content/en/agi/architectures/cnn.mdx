# CNN Architecture

---

## Basic Convolution

```python
nn.Conv2d(in_channels, out_channels, kernel_size, stride=1, padding=0)
```

---

## Key Components

| Component | Purpose |
|-----------|---------|
| **Conv layer** | Feature extraction |
| **Pooling** | Spatial reduction |
| **BatchNorm** | Stabilize training |
| **Activation** | Non-linearity |

---

## Classic Architectures

| Network | Year | Key Innovation |
|---------|------|----------------|
| **LeNet** | 1998 | First CNN |
| **AlexNet** | 2012 | ReLU, GPU training |
| **VGG** | 2014 | Deep, simple 3x3 |
| **GoogLeNet** | 2014 | Inception modules |
| **ResNet** | 2015 | Skip connections |
