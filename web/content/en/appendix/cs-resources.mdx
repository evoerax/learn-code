# CS learning resources (sites & repos)

A **practical bookmark list** leaning **free / open** materials. Policies and availability change—verify on each site. Pairs well with the site’s **[programming languages timeline](/timeline)**.

---

## Courses & guided paths

| Resource | Notes |
| :--- | :--- |
| [Teach Yourself Computer Science](https://teachyourselfcs.com/) | Topic-by-topic self-study map (algorithms, systems, OS, …) |
| [OSSU Computer Science](https://github.com/ossu/computer-science) | Community **BS-level** curriculum in open courses + checklist |
| [MIT OpenCourseWare](https://ocw.mit.edu/) | Lecture notes, assignments, recordings (e.g. **6.006**, **6.004**) |
| [Stanford Engineering Everywhere](https://see.stanford.edu/) | Archived Stanford courses (inventory varies) |
| [CS50](https://cs50.harvard.edu/) | Broad intro: C, Python, web, SQL |
| [UC Berkeley CS61A](https://cs61a.org/) | Large intro with a **SICP-flavored** track (check current term site) |
| [Crash Course Computer Science](https://www.youtube.com/playlist?list=PL8dPuuaLjXtNlUrzyH5r6jN9ulIgZBZb_) | Short YouTube overview series |
| [PKUFlyingPig/cs-self-learning](https://github.com/PKUFlyingPig/cs-self-learning) | **Chinese** guide linking global courses |
| [CS DIY Wiki](https://csdiy.wiki/) | Chinese community navigation for courses & books (check freshness) |

## How to use this page

- Finish **one** course’s labs before chasing new links.  
- Expect **English** for cutting-edge materials; use Chinese guides as bridges.  
- Star counts and README links drift—prefer the **official university** page when in doubt.
