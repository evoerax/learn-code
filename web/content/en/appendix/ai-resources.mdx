# AI learning resources (sites & repos)

Entry points for **ML / deep learning / LLMs**: courses, documentation, papers-with-code, and curated GitHub lists. **Licensing and ToS** belong to each project.

---

## Courses & tutorials

| Resource | Notes |
| :--- | :--- |
| [fast.ai](https://www.fast.ai/) | Practical deep learning (PyTorch), free courses + blog |
| [DeepLearning.AI](https://www.deeplearning.ai/) | Short specializations (often on Coursera; check free tiers) |
| [Stanford CS229](https://cs229.stanford.edu/) | Classic machine learning (materials vary by term) |
| [Stanford CS231n](https://cs231n.stanford.edu/) | Deep learning for computer vision |
| [Stanford CS224n](https://web.stanford.edu/class/cs224n/) | NLP with deep learning |
| [Hugging Face NLP Course](https://huggingface.co/learn/nlp-course/chapter1/1) | **Transformers**-centric HF course |
| [Hugging Face Learn (all tracks)](https://huggingface.co/learn) | Audio, vision, RL, etc. |
| [fast.ai course portal](https://course.fast.ai/) | Practical deep learning for coders |
| [3Blue1Brown — neural networks](https://www.3blue1brown.com/topics/neural-networks) | Intuition-first animations |
| [Hung-yi Lee ML lectures (NTU)](https://speech.ee.ntu.edu.tw/~hylee/ml/2023-spring.php) | Popular **Chinese**-language DL/ML lectures (check current term page) |
