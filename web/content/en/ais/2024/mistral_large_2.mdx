# Mistral Large 2

> Mistral AI **Large 2（≈123B 叙事）**：在 **多语种、代码与代理型任务** 上刷新自家旗舰表现，同时保持 **欧洲 AI 独立叙事与开放权重/商用 API 组合拳**。

## 速览

| | |
|---|---|
| **发布** | 2024 年 7 月（公开材料） |
| **竞争** | 对标 Llama-3.1-405B 的部分子任务 |
| **许可** | 以 Mistral 当时 License 为准 |

## 概述

Large 2 体现 Mistral **「中大参数稠密 + 强工程化推理栈」**策略：不以总参数最大为目标，而在 **上下文、函数调用可靠性、工具使用率** 上的服务质量取胜，并嵌入 **Le Chat** 与**企业私有化部署**。
