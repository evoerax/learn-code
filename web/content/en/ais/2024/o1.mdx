# o1 (Code Name: Strawberry)

> _Thinking before answering_

## At a Glance

| | |

## Overview

o1 was OpenAI's first model designed specifically for reasoning. Unlike previous models that generated answers quickly, o1 "thinks" through problems step-by-step, spending more computation time on difficult tasks.

<Callout type="success" emoji="🧠">
o1 scored 83% on International Mathematical Olympiad (IMO) problems, compared to GPT-4's 13%.
</Callout>

## Key Innovation: Test-Time Compute

Previous LLMs computed answers in a single pass. o1 introduced **test-time compute scaling** — the model can spend more tokens (thinking time) on harder problems.

### How It Works

1. **Chain-of-thought**: Generate detailed reasoning steps
2. **Evaluate**: Check if reasoning leads to correct answer
3. **Iterate**: Try alternative approaches if needed
4. **Respond**: Provide final answer with reasoning
