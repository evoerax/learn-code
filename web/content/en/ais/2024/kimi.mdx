# Kimi / Kimi+

> **月之暗面 Moonshot** 的 **Kimi** 助手：**超长上下文**（公开路线曾强调 200K 级别窗口叙事）+ **联网搜索/插件**，专注 **中文长文档**科研与办公。

## 速览

| | |
|---|---|
| **场景** | 论文综述、合同比对、日志分析 |
| **技术** | 公开细节有限，侧重无损压缩与检索增强 |
| **竞争** | 与字节豆包、腾讯混元助手等国产长文本产品对标 |

## 概述

Kimi 把 **「能读一本书」** 做成差异化卖点，在中文互联网引发 **长上下文军备赛**；其迭代也推动 **MoE 稀疏化** 与 **KV 缓存优化** 工程化，是应用层 prompt 与基础设施共同进化的案例。
