# GPT-4o mini

> OpenAI **GPT-4o mini**：在 **显著降低单价** 的前提下提供 **文本 + 图像** 多模态能力，面向 **高并发、成本敏感** 的商业集成。

## 速览

| | |
|---|---|
| **定位** | 替代部分 GPT-3.5 场景，逼近更强多模态 |
| **要点** | 延迟与吞吐优化，适合边缘网关、客服机器人 |
| **关系** | 与 GPT-4o 同属 omni 路线 |

## 概述

4o mini 让开发者以 **更低预算** 获得 **视觉理解**（如 OCR、UI 截图解析），推动「**一切助理默认多模态**」；也与开源小模型（Llama-3.1-8B、Qwen2.5 等）形成 **云 API vs 私有部署** 的成本对照。
