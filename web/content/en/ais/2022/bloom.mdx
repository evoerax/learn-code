# BLOOM

> **BigScience** 社区协作训练的 **~1760 亿参数** 多语模型，覆盖 **46 种语言与方言**，强调 **数据治理、环境影响与开放权重**。

## 速览

| | |
|---|---|
| **组织** | BigScience（Hugging Face 等发起） |
| **算力** | 法国 JZ 超算等公开资源 |
| **价值** | 多语公平、低资源语言代表、开放方法论文 |

## 概述

BLOOM 证明超大规模训练可以 **以跨国志愿者协作**完成：从数据清洗准则到责任发布均可审计；为全球监管讨论提供「**开放大模型**」参照物，与 OPT、LLaMA 形成开源三角。
