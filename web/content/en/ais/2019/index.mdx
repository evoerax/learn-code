# 2019: Zero-Shot Learning Goes Mainstream

## Overview

**2019** saw GPT-2 demonstrate that language models could perform tasks without any task-specific training — the "zero-shot" paradigm. OpenAI's staged release (starting with smaller models due to misuse concerns) became a story in itself.

## Key Milestone

### GPT-2 (OpenAI)
- **Parameters**: ~1.5 billion
- **Architecture**: 12-layer Transformer decoder (scaled from GPT-1)
- **Training Data**: WebText (8 million web pages)
- **Key Innovation**: Zero-shot task transfer via prompts
- **Release Strategy**: Staged release (first small models, then 1.5B)

<Callout type="warning" emoji="⚠️">
OpenAI initially withheld the full 1.5B model citing misuse concerns — the first major "staged release" in AI history.
</Callout>

## Timeline Context

