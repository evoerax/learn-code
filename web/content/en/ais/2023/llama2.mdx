# Llama 2

> Meta **Llama 2**：**7B / 13B / 70B** 开源权重，并附 **商用许可**（在可接受用途内），配套 **对话微调与安全红队** 数据。

## 速览

| | |
|---|---|
| **相对 Llama 1** | 更多 tokens、更强对齐、可服务托管 |
| **意义** | 企业可在私有云微调并合法对外提供服务 |
| **生态** | Llama-2-Chat 成为无数垂直助手基座 |

## 概述

Llama 2 改进了 **预训练语料过滤、上下文长度与分组查询注意力（GQA）** 等；其 **Responsible Use License** 虽有限制，但较 Llama 1「研究许可」显著拓宽商业场景，是 **开源大模型商业化**关键拐点。
