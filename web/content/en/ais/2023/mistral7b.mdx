## Origin Story

Mistral 7B proved that **small models + good architecture** could match models twice their size. Its efficient attention mechanisms made it a favorite for local deployment.

<Callout type="default" emoji="⚡">
Mistral 7B used two key innovations: Grouped-query Attention (GQA) for faster inference and Sliding Window Attention for handling longer contexts efficiently.
</Callout>

## Key Innovations

### Grouped-Query Attention (GQA)
- Shares key/value heads across query groups
- Faster inference with minimal quality loss

### Sliding Window Attention
- Each token attends to a fixed-size window
- Enables efficient handling of long sequences
