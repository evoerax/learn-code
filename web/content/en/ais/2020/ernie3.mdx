# ERNIE 3.0

> 约 **100 亿参数**，**知识增强** + 大规模中文预训练，在**理解**与**生成**任务上均衡优化。

## 速览

| | |
|---|---|
| **机构** | 百度 |
| **特点** | 统一框架兼顾 NLU 与 NLG，引入多任务与知识建模 |
| **谱系** | 与 ERNIE 1.0/2.0 一脉相承，为后续 Titan、4.x 铺路 |

## 概述

ERNIE 3.0 在中文语料与知识源上继续强化 **事实性与语义边界的联合建模**；相比同期以英文为主的 GPT-3，更强调 **中文场景下的检索增强、问答与内容生成**，是国内大模型工业化的重要节点。
