# GPT-3

> _"Language Models are Few-Shot Learners"_

## At a Glance

| | |
|---|---|
| **Year** | 2020 |
| **Organization** | OpenAI |
| **Parameters** | 175 billion |
| **Architecture** | 96-layer Transformer, 96 attention heads |
| **Training Data** | CommonCrawl, WebText, Wikipedia, Books |
| **Context Length** | 2,048 tokens |
| **Key Innovation** | Few-shot learning |

## Origin Story

GPT-3 was a quantum leap — **175 billion parameters**, roughly 10x larger than any previous non-sparse language model. The key insight: at this scale, models could perform new tasks with just a few examples in context (few-shot learning).

<Callout type="success" emoji="🚀">
GPT-3 demonstrated that scale alone could enable in-context learning — no gradient updates needed, just examples in the prompt.
</Callout>

## Few-Shot vs Zero-Shot vs Fine-tuning

| Setting | Description | Example |
|---------|-------------|---------|
| **Fine-tuning** | Update weights on task data | Train on 1000 translation pairs |
| **Zero-shot** | No examples, just instruction | "Translate to French: Hello" |
| **Few-shot** | 2-3 examples in context | "Hello → Bonjour, World → Monde, Cat → ?" |
