# Megatron-Turing NLG（MT-NLG）

> 约 **5300 亿参数**规模（公开披露），**NVIDIA + Microsoft** 联合展示，验证 **万亿参数稀疏/大规模分布式** 训练可行性，为后来 MoE 与更大生成模型铺路。

## 速览

| | |
|---|---|
| **机构** | NVIDIA & Microsoft |
| **技术栈** | Megatron-LM 分布式并行 + DeepSpeed 等 |
| **意义** | 「工程可训练」比单点算法更关键，推动 GPU 集群与内存优化标准 |

## 概述

MT-NLG 把 **模型并行、流水线并行、ZeRO 优化器** 等组合推到新高度，使单模型参数量进入五千亿量级仍可收敛；它预告了 **规模定律** 时代：算力、数据与系统软件与模型算法同等重要。
