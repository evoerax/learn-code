# GLM-130B

> 约 **1300 亿参数** 中英双语 **通用语言模型**（智谱 AI / 清华等），以 **开源权重 + 训练细节** 推动中文社区复现。

## 速览

| | |
|---|---|
| **机构** | 智谱 AI、清华大学等 |
| **定位** | GLM（General Language Model）双向与自回归统一框架的早期大规模实现 |
| **意义** | 中文开源大模型「可下载、可微调、可评测」的重要里程碑 |

## 概述

GLM-130B 在算力有限条件下系统展示 **混合精度、并行策略与稳定收敛** 的工程方案，使中文研究组能在单集群上训练千亿级稠密模型；后续智谱模型族（ChatGLM、GLM-4）均延续此路线。
