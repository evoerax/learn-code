# LaMDA

> 约 **1370 亿参数**（Google），面向 **开放域对话**，引入安全性、事实性、趣味性等多维指标与评价框架。

## 速览

| | |
|---|---|
| **机构** | Google |
| **特点** | 专攻多轮对话、检索增强与对抗性过滤流程 |
| **影响** | 推动「对话模型」与搜索、助手产品融合 |

## 概述

LaMDA 将 **对话** 视为核心产品形态：强调有趣、贴切、且尽量减少有害与虚假陈述；其工程化经验为 Bard / Gemini 对话产品线奠定方法论，也是「对齐」概念在消费级对话中早期大规模实践之一。
