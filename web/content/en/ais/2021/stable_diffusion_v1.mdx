# Stable Diffusion v1

> **Stability AI / CompVis / Runway** 等推动的 **潜在扩散（Latent Diffusion）** 模型，**开源权重**让个人与中小企业本地文生图成为现实。

## 速览

| | |
|---|---|
| **核心** | 在低维潜空间做扩散，降低像素空间计算量 |
| **生态** | Automatic1111、ComfyUI 等工具链爆发 |
| **意义** | 「开源图像生成」对抗封闭 API 的标志性事件 |

## 概述

Stable Diffusion v1 将 U-Net 噪声预测器与 VAE 潜空间结合，并以 CLIP 文本编码为条件；在消费级 GPU 可推理的前提下维持可接受画质，直接重塑设计、游戏、营销等内容生产流程。
