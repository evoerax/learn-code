# PaLM（Pathways Language Model）

> 约 **5400 亿参数**（Google **Pathways** 系统训练），在 **推理、代码生成与多语言** 任务上展示极强规模化能力。

## 速览

| | |
|---|---|
| **机构** | Google |
| **训练** | Pathways 软件栈 + TPU Pod |
| **备注** | 业界常将公开节点归入 2021–2022；本站 **2022** 另有扩展页 |

## 概述

PaLM 验证 **稠密 Transformer 继续随参数与数据scaling**，在少样本、思维链提示（早期形式）、数学与代码上带来质变；与专用芯片/集群协同，标志着「超大单体 LM」在硅谷头部实验室进入常态化。
