# 2017: Architectural shift

## Overview

**2017** brought the **Transformer**: an encoder–decoder stack built purely from attention, friendly to parallel training and displacing RNN/CNN stacks as the default backbone for large LMs.

## Landmarks

### Transformer (Google Brain)
- **Paper**: *Attention Is All You Need*
- **Ideas**: self-attention, multi-head attention, positional encodings, residuals + layer norms
- **Impact**: template for BERT, GPT, and essentially all scaled LMs that followed
