# NFT Token Standards

Token standards define the interface and behavior of NFTs on Ethereum and EVM-compatible chains. Each standard evolved to solve specific limitations.

---

## ERC-721 — The Original NFT Standard

The first standard for non-fungible tokens, created by CryptoKitties team in 2017:

```solidity
interface IERC721 {
    event Transfer(address indexed from, address indexed to, uint256 indexed tokenId);
    event Approval(address indexed owner, address indexed approved, uint256 indexed tokenId);
    event ApprovalForAll(address indexed owner, address indexed operator, bool approved);
    
    function balanceOf(address owner) external view returns (uint256);
    function ownerOf(uint256 tokenId) external view returns (address);
    
    function safeTransferFrom(address from, address to, uint256 tokenId, bytes memory data) external;
    function transferFrom(address from, address to, uint256 tokenId) external;
    function approve(address to, uint256 tokenId) external;
    function getApproved(uint256 tokenId) external view returns (address);
    function setApprovalForAll(address operator, bool approved) external;
    function isApprovedForAll(address owner, address operator) external view returns (bool);
    
    function safeTransferFrom(address from, address to, uint256 tokenId) external;
}
```

### Key concepts

| Function | Purpose |
|----------|---------|
| `ownerOf` | Who owns this specific token |
| `balanceOf` | How many tokens this address owns |
| `approve` | Grant another address permission to transfer |
| `setApprovalForAll` | Grant operator full control of all tokens |
| `safeTransferFrom` | Transfer with receiver callback (prevents lost tokens) |

## ERC-1155 — Multi-Token Standard

A single contract can manage multiple token types (fungible and non-fungible):

```solidity
interface IERC1155 {
    event TransferSingle(address indexed operator, address indexed from, address indexed to, uint256 id, uint256 value);
    event TransferBatch(address indexed operator, address indexed from, address indexed to, uint256[] ids, uint256[] values);
    
    function balanceOf(address account, uint256 id) external view returns (uint256);
    function balanceOfBatch(address[] calldata accounts, uint256[] calldata ids) external view returns (uint256[] memory);
    
    function setApprovalForAll(address operator, bool approved) external;
    function isApprovedForAll(address account, address operator) external view returns (bool);
    
    function safeTransferFrom(address from, address to, uint256 id, uint256 value, bytes calldata data) external;
    function safeBatchTransferFrom(address from, address to, uint256[] calldata ids, uint256[] calldata values, bytes calldata data) external;
}
```

### Advantages over ERC-721

| Aspect | ERC-721 | ERC-1155 |
|--------|---------|----------|
| Contract storage | One per token | One for all token types |
| Batch transfers | Multiple txs | Single tx |
| Gas efficiency | Higher | ~50% cheaper minting |
| Use case | 1/1 art, collectibles | Gaming items, fungible assets |

## ERC-4907 — Rental NFT Standard

Adds time-limited usage rights without transferring ownership:

```solidity
interface IERC4907 {
    struct UserInfo {
        address user;      // Who can use the NFT
        uint64 expires;    // When usage rights expire
    }
    
    function setUser(uint256 tokenId, address user, uint64 expires) external payable;
    function userOf(uint256 tokenId) external view returns (address);
    function userExpires(uint256 tokenId) external view returns (uint256);
    
    event UpdateUser(uint256 indexed tokenId, address indexed user, uint64 expires);
}
```

**Use cases:**
- Game item rentals
- Real estate usage rights
- Subscription-based access

## ERC-721A — Gas-Optimized Minting

OpenSea's implementation optimized for batch minting:

```solidity
// ERC-721A: batch minting at nearly constant gas
function _mint(address to, uint256 quantity) internal {
    for (uint256 i = 0; i < quantity; i++) {
        _addressData[to].balance++;
        _owners[totalSupply()] = to;
        emit Transfer(address(0), to, totalSupply());
        totalSupply++;
    }
}
```

**Gas comparison (mint 10 tokens):**
| Standard | Gas Cost |
|----------|----------|
| ERC-721 | ~2,500,000 |
| ERC-721A | ~560,000 |
