# 1990s — The Internet Changes Everything

The 1990s transformed computing from a business and scientific tool into a global network accessible to everyone. The World Wide Web created demand for new languages overnight. Java promised portable code across any device. Python and Ruby made programming approachable. JavaScript brought interactivity to web pages. By 2000, there were more programmers writing for the internet than for any other platform.

## Key Themes

**Java's platform promise.** Sun's "Write Once, Run Anywhere" was more than a slogan — it was a solution to a real problem. Enterprise software had to run on Windows, Unix, and mainframes, often simultaneously. The JVM abstracted over hardware differences. Servlets and later Spring made Java the backbone of online banking, e-commerce, and enterprise applications.

**Scripting languages rise.** Python, Ruby, PHP, and Perl dominated web development — not because they were faster than C++, but because they let developers iterate quickly. Dynamic typing, interactive shells, and garbage collection freed developers from memory management. The trade-off (slower runtime) was acceptable when the bottleneck was human time, not CPU time.

**Functional programming edges toward practice.** Haskell formalized lazy evaluation, type classes, and monadic I/O. It remained largely academic but influenced every language that came after — Scala, F#, Swift, Kotlin, Rust all incorporated Haskell ideas. OCaml bridged theory and practice, becoming the language of choice for compilers and formal verification tools.

**Open source changes everything.** Linux (C), Apache (C), Python, Ruby, PHP — the most important software of the 1990s was free and open. The commercial software model faced an existential challenge. Languages themselves became community projects, not corporate products.

## Looking Forward

The 1990s ended with the dot-com bubble at its peak. The 2000s would see a crash, a recovery, and a new wave of languages designed for large-scale distributed systems, functional programming in production, and the emerging mobile platform.
