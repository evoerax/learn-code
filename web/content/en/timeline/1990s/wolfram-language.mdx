# Wolfram Language

| | |
|---|---|
| **诞生时间** | 1988 |
| **创始人** | Stephen Wolfram |
| **范式** | 多范式（符号计算、函数式、规则驱动） |
| **当前版本** | 14.0 (2024) |

## 诞生背景

Wolfram Language 由 Stephen Wolfram 于 1988 年推出，作为 Mathematica 平台的核心理论。2013 年随着 Wolfram Alpha 公共 API 发布后独立命名。

Stephen Wolfram 是理论物理学家，17 岁进入牛津学习，20 岁获得加州理工博士学位。他的目标是用计算重新定义知识表示。

## 设计初衷

- 将世界上所有的知识组织成可计算的形式
- 用规则和模式替代传统算法思维
- "让最好的计算方法自然流露"——Stephen Wolfram

## 代码示例

```mathematica
(* Wolfram Language: 自然语言与符号计算的结合 *)
 Integrate[x^2, x]
(* 输出: x^3/3 *)

(* 世界上最聪明的答案引擎 *)
 WolframAlpha["population of France"]
 (* 输出结构化数据 *)

(* 规则驱动的符号操作 *)
 expr /. {x -> 5, y -> 3}
 (* 模式替换规则 *)
```
