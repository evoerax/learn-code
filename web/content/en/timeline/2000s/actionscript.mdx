# ActionScript

> _The language that brought interactivity to the web before JavaScript conquered it._

## At a Glance

| | |

## Origin Story

In 1996, Macromedia acquired FutureWave and its vector graphics software Future Splash. To add interactivity to animations, they developed ActionScript as a scripting language. Initially a simple command language, it evolved rapidly.

By 2000, ActionScript 1.0 brought more structured programming with syntax inspired by JavaScript. When Macromedia released Flash MX in 2002 and later ActionScript 2.0 in 2003, the language gained true object-oriented capabilities with class syntax and type checking.

The pivotal moment came in 2006 when Adobe (which had acquired Macromedia) released ActionScript 3.0—a complete rewrite with a virtual machine (AVM2) that brought performance gains of up to 10x. Flash applications could finally rival desktop software in complexity.

## Design Philosophy

ActionScript was designed to be accessible to designers while powerful enough for developers. It embraced the ECMAScript standard, maintaining compatibility with JavaScript while adding Flash-specific libraries for graphics, animation, and media.

The language evolved from a purely procedural model to a fully object-oriented one. ActionScript 3.0 introduced packages, interfaces, and compile-time type checking, making it suitable for large-scale application development.

## Code Example

```actionscript filename="Flash Game Sample"
// ActionScript 3.0 class-based approach
package {
    import flash.display.Sprite;
    import flash.events.Event;

    public class Ball extends Sprite {
        private var vx:Number = 5;
        private var vy:Number = 5;

        public function Ball() {
            graphics.beginFill(0xFF0000);
            graphics.drawCircle(0, 0, 20);
            graphics.endFill();
            addEventListener(Event.ENTER_FRAME, update);
        }

        private function update(e:Event):void {
            x += vx;
            y += vy;
            if (x > stage.stageWidth || x < 0) vx *= -1;
            if (y > stage.stageHeight || y < 0) vy *= -1;
        }
    }
}
```
