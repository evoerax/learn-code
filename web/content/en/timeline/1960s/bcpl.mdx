## Origin Story

BCPL (Basic Combined Programming Language) was designed by Martin Richards at Cambridge University in 1967. It emerged from his work on a compiler writing project, and quickly became the language of choice for systems programming on early minicomputers.

The language was notably used to write the early versions of the Xerox Alto operating system and other research operating systems at Cambridge and MIT. Most famously, Ken Thompson used BCPL as the inspiration and starting point when he created the B language at Bell Labs, which eventually evolved into C.

## Design Philosophy

<Callout type="default" emoji="💡">
  "BCPL was designed to be a lean, expressive systems programming language that could compile quickly."
</Callout>

Key design features:

- **Curly brace delimiters**: The `{` and `}` characters that C inherited from BCPL became ubiquitous in programming syntax
- **Single type system**: BCPL used a single type (the machine word), with type checking delegated to runtime or programmer discipline
- **Expression-based**: Almost everything in BCPL is an expression, returning values
- **Minimal syntax**: No semicolons required—statements are separated by whitespace and context
- **Pointers as integers**: Memory addresses were treated as integers, giving programmers low-level control

## Code Example

```bcpl
GET "LIBHDR"

LET START() = VALOF

LET FACTORIAL(N) = VALOF
{
    IF N <= 1 THEN RESULTIS 1
    RESULTIS N * FACTORIAL(N - 1)
}
```

## Influence & Legacy

**Inspired by:** CPL (Combined Programming Language, 1963)

**Influenced:** B, C, JavaScript, Java, and virtually all curly-brace languages

BCPL's most enduring contribution to computing history is its syntax. The curly braces `{` and `}` that C popularized—and that virtually every modern language supports—originated in BCPL. When Ken Thompson created B in 1969, he stripped BCPL down to its essentials, and Dennis Ritchie later built C on B's foundation.

Without BCPL, the C family of languages would have a very different syntax, and JavaScript, Java, C++, and countless others would not exist in their current form.
