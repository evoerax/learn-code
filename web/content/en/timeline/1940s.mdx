# 1940s — The Dawn of Computing

The 1940s marks the birth of programmable computing. Before high-level languages existed, programmers worked directly with machine instructions, toggle switches, and punched paper tape. The decade produced the world's first computers — and the first attempts to make them easier to program.

## Context

| | |
|---|---|
| **Era** | Pre-language, machine-code era |
| **Key hardware** | ENIAC (1945), Manchester Baby (1948), EDVAC (1949) |
| **Programming method** | Wiring patch panels, binary codes, assembly mnemonics |
| **Key challenge** | Making machines do *anything* useful |

<Callout type="info">
  Programming in the 1940s was a physical act. On the ENIAC, "programming" meant rewiring 6,000 switches and 3,000 cables. Each new problem required hardware reconfiguration.
</Callout>

## Languages of the 1940s

| Language | Year | Creator | Significance |
|----------|------|---------|--------------|
| Plankalkül | 1943 | Konrad Zuse | First high-level language design (not implemented until 1998) |
| ENIAC Coding System | 1945 | Eckert & Mauchly team | First operational program notation |
| Short Code | 1949 | John Mauchly | First interpreted high-level language on a digital computer |

## Key Themes

**Hardware and software were one.** The ENIAC had no stored program — instructions and data lived in different systems. The stored-program concept (von Neumann architecture), introduced in 1945, changed everything: programs could be data, and machines could modify their own instructions.

**Mathematics drove design.** Konrad Zuse's Plankalkül was designed to express mathematical algorithms and chess-playing logic. Though never implemented in the 1940s, it anticipated many concepts that wouldn't become mainstream until decades later: assignments, subroutines, array operations, even a type system.

**Women programmed the first computers.** The six original ENIAC programmers — Jean Jennings Bartik, Frances Bilas Spence, Betty Holberton, Marlyn Wescoff Meltzer, and others — were female mathematicians who turned mathematical equations into machine instructions without any manuals or languages to guide them.

## Looking Forward

The seeds planted in the 1940s would flower rapidly in the next decade. The stored-program model enabled Fortran, LISP, and COBOL — languages where humans could write something resembling mathematics and English, letting the machine handle translation to binary.
