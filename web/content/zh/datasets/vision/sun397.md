# SUN397 (Scene UNderstanding)

## 数据集简介

SUN397 是由 MIT 视觉团队于 2014 年发布的大规模场景分类数据集，是 Scene UNderstanding 数据集系列的精选版本。该数据集包含 397 个场景类别，共约 108,754 张图像，每张图像来自 ImageNet 的 SUN 场景层次结构中精心筛选的高质量样本。SUN397 的场景类别涵盖室内（办公室、卧室、厨房）、室外（海滩、山脉、城市街道）等多种环境，是评估模型场景感知能力的重要基准。与 Places365 相比，SUN397 更强调语义丰富性和场景多样性，类别间的视觉差异更细微，对细粒度场景理解提出了更高挑战。

## 相关论文

**SUN397: A Large-scale Scene Classification Database**

- **作者**: Jianxiong Xiao, James Hays, Krista A. Ehinger, Antonio Torralba
- **年份**: 2014
- **摘要**: 本文介绍了 SUN397 数据集，一个用于场景分类的大规模图像数据库。该数据集包含 397 个场景类别，每类至少包含 100 张图像，共约 108,754 张图像。SUN397 的类别来自对 ImageNet 场景层次的精心筛选和重新组织，确保类别间的语义差异明显且具有实际意义。与物体分类数据集不同，场景分类要求模型理解空间布局、上下文关系和整体氛围等高层语义。实验评估了多种场景分类方法，发现即使是当时最先进的模型在 SUN397 上的准确率也远低于人类水平（约 90%），表明场景理解仍是计算机视觉的重大挑战。

## 数据示例

**数据格式说明**: SUN397 数据集图像存储为 JPEG 格式，类别名称采用空格分隔的描述性命名（如 "baseball_field"、"waiting_room"）。数据集提供官方划分的训练/测试集，以及类别列表文件。

**数据目录结构**:
```bash
SUN397/
├── class_names.txt
├── image_class_labels.txt
├── train_test_split.txt
├── Images/
│   ├── a/abbey/xxxx_abbey.jpg
│   ├── b/baseball_field/xxxx_baseball.jpg
│   └── ...
└── README.txt
```

**类别列表样本**:
```

```

**真实样本示例**:

```bash
# 样本1
图像路径: Images/b/baseball_field/sun_b baseball_field_0001.jpg
类别: baseball_field
描述: 室外运动场地，包含绿色草坪土丘和观赛区
场景特征: 开放空间、运动场地、日间光照

# 样本2
图像路径: Images/w/waiting_room/sun_w waiting_room_0005.jpg
类别: waiting_room
描述: 室内等候区域，包含座椅、窗户和走廊
场景特征: 封闭空间、室内照明、家具布局

# 样本3
图像路径: Images/k/kitchen/sun_k kitchen_0012.jpg
类别: kitchen
描述: 家庭厨房，包含橱柜、灶台和台面
场景特征: 室内、厨房设备、食物储存
```

**数据集统计**:
```
总类别数: 397
总图像数: 108,754
每类最少图像: 100
图像分辨率: 可变（通常 500×375 左右）
训练/测试划分: 约 50/50 每类
评估指标: Top-1 准确率, Top-5 准确率
人类表现: ~90% Top-1 准确率
```
