# CMRC2018 数据集

## 数据集简介

CMRC2018（Chinese Machine Reading Comprehension 2018）是由中国中文信息学会（CIPS）举办的第三届"讯飞杯"中文机器阅读理解评测数据，旨在促进中文NLP研究发展。该数据集包含约300,000个问题-答案-上下文三元组，来源于中文维基百科文章。问题类型包括"不可回答"问题（需要模型判断问题是否可以从给定段落回答）、"立场选择"问题（判断问题、选项和答案之间的一致性关系）以及"抽取型"问题（从段落中抽取答案span）。CMRC2018是中国首个大规模中文阅读理解评测数据，对于推动中文语言模型发展具有重要意义，被广泛应用于评估BERT等预训练模型在中文阅读理解任务上的性能。

## 相关论文

**CMRC 2018: A Large-scale Chinese Machine Reading Comprehension Dataset**  
Cui Y, Liu T, Che W, et al.  
EMNLP 2019  

本文介绍了CMRC2018数据集的构建过程和评测任务设计。数据集包含30万+三元组，来自维基百科的复杂篇章结构。论文分析了不同模型在该数据集上的表现，发现当前模型在处理需要推理和综合理解的问题时仍存在明显不足。该论文推动了中文阅读理解研究的快速发展，成为中文NLP领域的重要评测基准。

## 数据示例

**数据格式说明**

CMRC2018使用JSON格式存储，结构如下：
```json
{
  "data": [
    {
      "context_id": "DEV_0001",
      "context": "中国的首都是北京，位于华北平原的北部...",
      "question": "中国的首都是哪里？",
      "answers": [
        {"text": "北京", "start_position": 9},
        {"text": "北京", "start_position": 9}
      ],
      "is_impossible": false
    }
  ]
}
```

**真实样本**

1. 上下文: "北京是中华人民共和国的首都，位于华北平原北部，面积约1.6万平方公里。作为中国的政治、文化中心，北京拥有故宫、天安门广场等著名景点。"  
   问题: "中国的首都是哪里？"  
   答案: `{"text": "北京", "start_position": 0}`  
   是否可回答: false

2. 上下文: "人工智能是计算机科学的一个分支，它企图了解智能的实质，并生产出一种新的能以人类智能相似的方式做出反应的智能机器。"  
   问题: "人工智能属于哪个学科？"  
   答案: `{"text": "计算机科学", "start_position": 12}`  
   是否可回答: false

3. 上下文: "水的化学式是H2O，在标准大气压下，水的沸点是100摄氏度，冰点是0摄氏度。"  
   问题: "水在标准大气压下的沸点是多少度？"  
   答案: `{"text": "100摄氏度", "start_position": 25}`  
   是否可回答: false
