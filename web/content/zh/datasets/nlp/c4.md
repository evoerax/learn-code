# C4 (Colossal Clean Crawled Corpus)

## 数据集简介

C4 是 Google 于 2019 年发布的大规模清洗文本语料库，是训练 T5 (Text-to-Text Transfer Transformer) 模型的核心数据来源。该数据集从 Common Crawl 的网页数据中经过多轮严格清洗得到，包括去除重复文档、移除非英文内容、过滤短句子、以及清除包含不当内容的文本等。清洗后的语料库规模约为 750GB，涵盖约 3500 亿个 token。C4 的设计目的是为预训练语言模型提供高质量、大规模的文本数据，其清洗策略成为后续大规模预训练数据集处理的重要参考标准。

## 相关论文

**Exploring the Limits of Transfer Learning with a Unified Text-to-Text Transformer**

- 作者: Colin Raffel, Noam Shazeer, Adam Roberts et al.
- 年份: 2019
- 摘要: 本文提出 T5 模型，将所有 NLP 任务统一建模为文本到文本的转换问题。作者系统性地比较了多种预训练目标、模型架构和数据集清洗策略，发现使用去噪目标在 C4 语料上预训练的模型在多项基准测试上取得最优表现。该研究为大规模预训练语言模型提供了统一框架和详尽的消融实验分析。

## 数据示例

**数据格式说明**: JSON Lines 格式，每行包含单个文档的原始文本内容

**样本 1**:
```json
{
  "text": "The history of natural language processing (NLP) dates back to the 1950s when Alan Turing proposed the famous Turing Test as a measure of machine intelligence. Early NLP systems relied on rule-based approaches before statistical methods revolutionized the field in the 1980s. Today, deep learning techniques enable unprecedented progress in machine translation, sentiment analysis, and question answering."
}
```

**样本 2**:
```json
{
  "text": "Climate change represents one of the most pressing challenges facing humanity in the 21st century. Rising global temperatures have led to melting ice caps, rising sea levels, and increasingly severe weather events. International cooperation efforts like the Paris Agreement aim to coordinate responses to mitigate these impacts."
}
```

**样本 3**:
```json
{
  "text": "Machine learning algorithms can broadly be categorized into supervised, unsupervised, and reinforcement learning paradigms. Supervised learning uses labeled data to train models, while unsupervised learning discovers patterns in unlabeled data. Reinforcement learning trains agents through trial and error interactions with an environment."
}
```
