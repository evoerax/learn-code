# 2017: 架构革命

## 概述

**2017 年** Google 提出 **Transformer**：纯注意力堆叠的编码器–解码器，并行计算友好，取代 RNN/CNN 成为此后大模型的事实标准骨架。

## 关键里程碑

### Transformer（Google Brain）
- **论文**: *Attention Is All You Need*
- **核心**: 自注意力、多头注意力、位置编码、残差与层归一化
- **意义**: 为 BERT、GPT 及后续所有大规模语言模型奠定计算图与归纳偏置基础

</Callout>
