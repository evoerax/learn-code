# Turing-NLG

> 约 **170 亿参数**（公开披露规模），微软早期 **超大规模生成式语言模型** 代表，与 **Microsoft 365** 等生产力场景探索结合。

## 速览

| | |
|---|---|
| **机构** | Microsoft |
| **时代背景** | GPT-3 前后，业界验证「更大解码器 + 更多算力」路线 |
| **意义** | 展示企业级团队在内部基础设施上训练与对齐百亿级模型的能力 |

## 概述

Turing-NLG 与 Megatron 等分布式训练栈配合，在**生成式建模**上达到当时公开披露的最大规模之一；虽声量不及 GPT-3 面向开发者生态的全面开放，但推动微软在 **Office、搜索、对话** 产品线中布局大生成模型能力。
