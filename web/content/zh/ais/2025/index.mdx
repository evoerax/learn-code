# 2025: 推理与多模态之年

## 概述

**2025年**见证了推理模型（OpenAI o3）、多模态全能模型（GPT-4o, Gemini 2.0）的崛起，以及中国 MoE 的主导地位（Qwen3/Qwen3.5 系列和 ERNIE 4.5）。趋势从"更大"转向"更有能力"和"更高效"。

## 关键里程碑

### GPT-4.1 / o3 (OpenAI)
- **参数**: 未披露
- **GPT-4.1**: GPT-4 的后继者，新一代主力
- **o3 / o3-mini**: 专用推理模型；复杂推理、数学、编程表现出色

### Qwen3 / Qwen3.5 (阿里)
- **Qwen3-235B-A22B**: 2350亿总参数 / 220亿激活（MoE）
- **Qwen3-Coder-480B-A35B**: 4800亿总参数 / 350亿激活（编程专用）
- **Qwen3.5-397B-A17B**: 3970亿总参数 / 170亿激活
- **披露**: 完全披露

### ERNIE 4.5 (百度)
- **参数**: 4240亿总参数 / 约470亿激活（MoE）
- **意义**: 百度旗舰，中文理解强
- **披露**: 技术报告完全披露
