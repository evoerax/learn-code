# 2026: AI现状

## 概述

到2026年，AI格局已经形成两个明显不同的范式：**美国闭源实验室**（OpenAI、Google、Anthropic、xAI）保密参数，和**中国开源权重实验室**（阿里、百度、DeepSeek、腾讯）公布详细规格。

## 关键洞察

<Callout type="success" emoji="🎯">
**美国闭源**: 参数常未披露 → 关注能力和 API 定价
**中国开源权重**: 参数完全披露 → 赋能研究和高效部署
</Callout>

## 快速对比

| 方面 | 美国实验室 | 中国实验室 |
|------|----------|----------|
| **典型披露** | 未披露 | 完全披露 |
| **架构** | MoE（估计） | MoE（有文档） |
| **代表性规模** | 1000亿–1万亿+（估计） | 1000亿–7000亿（有文档） |
| **示例模型** | GPT-4, Claude 4, Gemini 3 | DeepSeek-V3, Qwen3.5, ERNIE 4.5 |
