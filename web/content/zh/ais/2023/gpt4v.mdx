# GPT-4V（GPT-4 with Vision）

> OpenAI 为 **GPT-4** 增加 **原生图像输入**：支持照片、截图、手绘草图、论文插图与 UI **分析与推理**。

## 速览

| | |
|---|---|
| **发布** | 2023 年秋（与 ChatGPT 视觉能力逐步整合） |
| **要点** | 单模型同时处理文本 + 多图上下文 |
| **意义** | 引爆「多模态助手」产品形态 |

## 概述

GPT-4V 把 **视觉 token 与语言 token 联合自回归**，实现 OCR、物体关系、简易几何、图表读数、梗图理解等；为后续 GPT-4o **全模态一体化**铺路，也迫使所有竞品发布视觉 API。
