# Qwen1.5 / Qwen2

> 阿里通义 **Qwen** 家族：**0.5B–72B** 多规格开源，针对 **中文、代码、数学、工具调用与长上下文** 持续迭代。

## 速览

| | |
|---|---|

## 概述

Qwen2 在 **数据清洗、对齐与 Chat 模板** 上贴近国际最佳实践，提供 **dense 与 MoE** 多路线；因其 **中文综合能力** 与 **HuggingFace 生态兼容性**，成为亚洲开发者的首选开源基座之一。
