# ERNIE 4.0

> 百度 **ERNIE 4.0**：在 **知识图谱、搜索实时信号与文库** 加持下强化 **中文事实性与行业任务**，并与 **文心一言** 等产品深度整合。

## 速览

| | |
|---|---|
| **亮点** | 多模态理解 + 插件 / 工具调用 |
| **对比Open系** | 强调与国内互联网内容生态合规对接 |
| **延续** | ERNIE 4.5 见 2025 专页 |

## 概述

ERNIE 4.0 将 **「可验证知识」嵌入预训练与对齐**：利用搜索抓取、百科结构化字段减少胡编；面向 **政务、金融、医疗** 等垂直行业提供可控版本，是中国 **产业大模型** 路线代表。
