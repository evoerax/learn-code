# 🗓️ 大模型发展时间线（2017–2026 完整版）

> 按年整理代表性工作与产品线；参数量与能力描述以公开论文、技术报告与官方博客为准，跨年度项目按首次广泛可用年份归类。

若更关心 **架构族与能力支柱**（Transformer、MoE、SSM/混合块、推理时计算、世界模型与具身等），见 **[AGI 技术栈](/zh/agi/overview)**。若更关心 **训练框架、推理服务、智能体编排与评测工具链**，见 **[软件层](/zh/software/overview)**。

正文 **2026 → 2017 倒序**；每年下列出 **模型名 + 一句话总结**。

---

## 2026（前瞻）

> 基于公开路线图与行业预期，实际以各厂商发布为准。

**GPT-6 / GPT-5.5（OpenAI，预期）**

多模态原生与自主任务规划加深，系统 2 式推理与工具链融合，面向通用任务代理。

**Gemini 3.0（Google，预期）**

长上下文、多任务与端云一体推理并进，探索世界模型与具身智能接口。

**Claude 5.0（Anthropic，预期）**

安全对齐与可审计推理路径强化，宪法式 AI 与人类偏好动态建模延续升级。

**DeepSeek-R3 / V4（深度求索，预期）**

MoE、推理链与动态稀疏激活优化，单 token 成本与开源策略持续受关注。

**Qwen4 / Qwen-Omni（阿里，预期）**

在参数效率、轻量化部署与许可组合上平衡，推进文本/图像/音频/视频等统一建模。

**Llama 5（Meta，预期）**

开源多模态家族扩展，覆盖 3D 理解、具身交互与跨设备协同等方向。

**Mistral 5（Mistral AI，预期）**

多语种旗舰与中小企业工具链整合，面向边缘 AI 与隐私计算场景。

**Open Source AGI Initiative（社区协作，预期）**

多机构协作开源超大规模模型，强调透明训练、可复现评估与治理框架。

**ERNIE 4.5（百度，6 月）**

约 4240 亿总参 / 470 亿激活，强调完全开源与中文场景深度优化。

**Mistral Large 2（Mistral AI，7 月）**

约 123B 级别旗舰，多语种与编码能力增强。

**GPT-4o（OpenAI，5 月）**

**GPT-4o mini（OpenAI，7 月）**

**Claude 3.5 Sonnet（Anthropic，6 月）**

**o1-preview / o1-mini（OpenAI，9 月）**

**GLM-4 / GLM-4V（智谱 AI）**

## 2019

约 15 亿参数展示零/少样本续写，分阶段开源应对滥用风险。

约 110 亿 Large，Text-to-Text 统一分类、生成与翻译等任务。

**ALBERT（Google）**

约 2.35 亿 xxlarge，共享参数与因子化嵌入，用更少参数对标 BERT-Large。

**DistilBERT（Hugging Face）**

约 6600 万参数蒸馏 BERT，保留约 97% 性能，推动端侧部署。

**ERNIE 1.0（百度）**

约 1.1 亿参数知识增强预训练，融合实体关系，中文理解领先。

## 2018

**GPT-1（OpenAI）**

约 1.17 亿参数，首个基于 Transformer 解码器的生成式预训练，验证预训练+微调范式。

**BERT（Google）**

约 3.4 亿 Large，双向 Transformer 与 MLM，刷新多项 NLP 任务记录。

**XLNet（Google / CMU）**

约 3.4 亿 Large，排列语言建模与双流注意力，多项理解任务超越 BERT。

**ELMo（Allen AI）**

约 9400 万参数 BiLSTM 动态词向量，推动上下文嵌入时代。

**RoBERTa（Meta）**

约 3.55 亿参数，动态掩码、更大批量与数据，稳健性显著强于原版 BERT 训练。

## 🔑 技术演进主线总结

| 阶段 | 时间 | 核心突破 | 代表模型 |
|------|------|----------|----------|
| 架构奠基 | 2017 | Transformer + 自注意力 | Transformer |
| 预训练范式 | 2018–2019 | MLM/NSP + 微调 | BERT, GPT-2, T5 |
| 规模扩展 | 2020–2021 | 缩放定律 + In-Context Learning | GPT-3, PaLM, GLM-130B |
| 对齐优化 | 2022 | RLHF + Constitutional AI | ChatGPT, InstructGPT, Claude |
| 多模态融合 | 2023–2024 | 原生多模态 + 长上下文 | GPT-4V, Gemini, Qwen-VL |
| 效率与推理 | 2024–2025 | MoE + Test-time Compute | DeepSeek-V3, o1, Mixtral |
| 开源深化 | 2023–2026 | 开放权重 + 轻量化部署 | Llama 系列, Qwen, Mistral |
| 通用智能探索 | 2026+ | 自主 Agent + 世界模型 | （前瞻） |

> **关键趋势**：参数规模竞赛 → 效率与推理优化 → 多模态原生 → 开源生态繁荣 → Agent 与通用任务整合。
