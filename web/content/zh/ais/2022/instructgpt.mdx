# InstructGPT

> 约 **1750 亿参数**（OpenAI），**监督微调（SFT）+ RLHF** 对齐人类意图与安全偏好，为 **ChatGPT** 奠定直接技术基础。

## 速览

| | |
|---|---|
| **方法** | 人类示范、排序偏好、PPO 式策略优化 |
| **产出** | 更小偏好模型打分 + 主模型对齐 |
| **意义** | 「有用、真实、无害」三维对齐成为产品默认模板 |

## 概述

InstructGPT 证明：在基座 LM 规模已很大时，**对齐阶段**能显著改善用户体感——减少有害回答、提高指令跟随；自此 RLHF / RLAIF / DPO 等变体成为所有对话产品的标配环节。
