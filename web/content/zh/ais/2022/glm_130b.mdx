# GLM-130B

> _"基于自回归填空的一般语言模型预训练"_

## 概览

| | |
|---|---|
| **年份** | 2022 |
| **机构** | 清华大学 / 智谱AI |
| **参数** | 1300 亿 |
| **架构** | GLM（通用语言模型） |
| **训练** | 双语（英文 + 中文） |
| **关键创新** | 自回归填空，开源 |

## 起源

GLM-130B 是首批高质量开源双语（中英文）LLM 之一。与标准从左到右的模型不同，GLM 使用独特的**填空**预训练目标。

<Callout type="info" emoji="🇨🇳">
GLM-130B 确立了清华大学和智谱AI在开源 LLM 领域的重要地位，影响了后续中国模型的设计。
</Callout>

## 架构

- **1300 亿参数**（稠密）
- **GLM 架构**: 使用自回归填空而非标准掩码语言建模
- 双语训练：英文 + 中文
- 开源权重
