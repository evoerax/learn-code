# MiniMax / Abab 系列

> **MiniMax（稀宇科技）** 的 **abab** 大模型系列：**多模态对话、语音合成与实时交互**，面向 **C 端娱乐、社交与 B 端垂直 Agent**。

## 速览

| | |
|---|---|
| **特点** | 强调「端到端实时语音大模型」体验 |
| **合规** | 中国大陆生成式服务备案产品之一 |
| **生态** | 与游戏、短剧、虚拟人产业结合紧密 |

## 概述

MiniMax 将 **生成式语音 + LLM** 深度整合，探索 **低延迟情绪对话、角色扮演与内容生产**；其 MoE/长上下文模型常与 **火山引擎、腾讯云** 等渠道集成，代表「**模型公司 + 应用矩阵**」的上海路线。
