# Sora

> 从文本指令生成连续影像与简单镜头语言

## 速览

| | |
|---|---|
| **首次公开** | 2024 年 2 月（演示与技术博客） |
| **机构** | OpenAI |
| **形态** | 文生视频为主，支持扩展与图像动画等能力（以产品说明为准） |
| **开放节奏** | 分阶段邀请 / 订阅（各地法规与产品线不同） |

## 概述

Sora 将扩散式生成从静态图像推进到 **时间维度的视频片段**，强调场景一致性、角色与镜头的可控性；与 DALL·E（图）、GPT（文）并列，代表 OpenAI 在 **多模态生成** 上的延伸。

<Callout type="info" emoji="🎬">
公开演示侧重 **数十秒级** 短片、多主体与简单运镜；实际可及分辨率、时长与版权策略以官方为准。
</Callout>

## 能力要点

- **文生视频**：自然语言描述驱动画面与时间轴  
- **时序一致性**：在片段内保持主体与环境相对稳定  
- **风格与镜头**：写实 / 动画等风格与基础运镜控制  
- **扩展能力**：延长片段、从静态图生成动效（以版本为准）
