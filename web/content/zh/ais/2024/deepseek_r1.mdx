# DeepSeek-R1

> 深度求索 **R1**：在 **V3 基座**上强化 **数学、代码与逻辑推理**，采用 **GRPO** 等算法与 **可验证奖励**，以 **开源权重** 形式发布，震撼全球开发者社区。

## 速览

| | |
|---|---|
| **特点** | 展示可读的「思考」轨迹（视产品封装而定） |

## 概述

R1 将 **强化学习** 与 **规则校验**结合：对可自动判题的数学/代码任务给稠密奖励，对开放式任务用模型评判或人类偏好；其 **蒸馏小模型** 让高校在单卡上复现强推理。
