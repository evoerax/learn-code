# GLM-4 / GLM-4V

> 智谱 **GLM-4** 家族：**中英双语大模型 + 多模态 GLM-4V**，对齐 **长上下文、工具调用与 Agent 框架（AutoGLM 等）**，**开源 + 商用 API** 并行。

## 速览

| | |
|---|---|
| **路线** | 延续 ChatGLM 产品矩阵 |
| **特点** | 专注中文办公、编程助手与行业解决方案 |
| **对比** | 与 Qwen、DeepSeek 构成国内开源三极 |

## 概述

GLM-4 强调 **「模型 + 框架 + 工具链」一体化**：让企业可在私有云部署具备 **检索、执行代码、调用业务 API** 的智能体；4V 系列补齐 **图像理解与 OCR pipeline**，服务金融研报、票据处理等场景。
