# GPT-2

> _"更好的语言模型及其影响"_

## 概览

| | |
|---|---|
| **年份** | 2019 |
| **机构** | OpenAI |
| **参数** | 约 15 亿（最大版本） |
| **架构** | 12 层 Transformer 解码器 |
| **训练数据** | WebText（800 万网页） |
| **关键创新** | 零样本任务迁移 |

## 起源

GPT-2 建立在一个简单但强大的洞察上：**更大的语言模型可以在没有任何任务特定训练的情况下执行任务**——只需通过自然语言阅读指令。

<Callout type="warning" emoji="⚠️">
OpenAI 最初以滥用担忧为由扣留了完整的 15亿模型，采用"分阶段发布"策略。这是 AI 历史上首次重要的分阶段发布。
</Callout>

## 版本

| 模型 | 参数 |
|------|------|
| Small | 1.17亿 |
| Medium | 3.45亿 |
| Large | 7.62亿 |
| XL | 15亿 |

## 关键创新：零样本学习

与需要微调每个任务的 GPT-1 不同，GPT-2 展示了**零样本**能力：

- 阅读自然语言指令
- 无需任务特定训练即可执行翻译、摘要、问答
- 仅从上下文学习
