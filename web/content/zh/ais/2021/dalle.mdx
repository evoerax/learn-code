# DALL·E

> 约 **120 亿参数**（OpenAI），**文本到图像**生成模型，把 **CLIP 式图文对齐** 与 **自回归/VQ 视觉 token** 结合，打开「说人话出图」的产品化路径。

## 速览

| | |
|---|---|
| **机构** | OpenAI |
| **特点** | 统一文本与图像 token 序列上的自回归建模 |
| **延续** | DALL·E 2（扩散） |

## 概述

DALL·E 证明大规模多模态预训练能让模型 **遵循复杂自然语言描述** 合成新颖图像，催生创意工具链；其方法学与后来的 **扩散模型 + CLIP** 路线形成互补，共同塑造 AIGC 产业。
