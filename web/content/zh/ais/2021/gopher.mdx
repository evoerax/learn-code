# Gopher

> 约 **2800 亿参数**（DeepMind），用系统实验评估 **缩放定律、数据质量与危害评测**，定义「大规模 LM 该怎么评」。

## 速览

| | |
|---|---|
| **机构** | DeepMind |
| **重点** | Retro 等改进检索、对话安全性与事实性基准 |
| **意义** | 与 Chinchilla 等工作共同推动「算力换数据/最优 token 预算」讨论 |

## 概述

Gopher 不仅堆规模，还系统报告 **偏见、毒性、编造** 与下游任务曲线，强调大模型研发必须与 **评测与社会影响** 同步；其技术叙事深刻影响后续工业界训练报告形态。
