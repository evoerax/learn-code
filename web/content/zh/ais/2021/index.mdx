# 2021: 中国LLM进入竞争

## 概述

**2021年**中国AI实验室推出大规模模型。百度的 ERNIE 3.0 Titan 展示了知识增强预训练，以 2600 亿参数成为当时最大的中文稠密模型之一。

## 关键里程碑

### ERNIE 3.0 Titan (百度)
- **参数**: 2600 亿
- **架构**: 知识增强型稠密 Transformer
- **训练**: 大规模中文 + 多语言语料库，知识整合
- **意义**: 当时最大的中文稠密模型之一，中文理解能力强

<Callout type="info" emoji="🇨🇳">
ERNIE 3.0 Titan 在中国开创了知识增强预训练，将结构化知识整合到预训练过程中。
</Callout>

## 时间线

```
2020: GPT-3 — 1750亿（美国）
2021: ERNIE 3.0 Titan — 2600亿（中国）
2022: PaLM — 5400亿（美国），GLM-130B — 1300亿（中国）
```
