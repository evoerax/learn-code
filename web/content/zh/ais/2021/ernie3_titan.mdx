# ERNIE 3.0 Titan

> _"探索更大规模的知识增强预训练"_

## 概览

| | |
|---|---|
| **年份** | 2021 |
| **机构** | 百度 |
| **参数** | 2600 亿 |
| **架构** | 知识增强型稠密 Transformer |
| **语言** | 中文 + 多语言 |
| **关键创新** | 知识增强预训练 |

## 起源

2021年12月，百度发布 ERNIE 3.0 Titan——以 2600 亿参数成为当时最大的中文稠密模型之一。关键创新是将**结构化知识**直接整合到预训练过程中。

<Callout type="info" emoji="🇨🇳">
ERNIE 3.0 Titan 是中国 AI 的重要里程碑，证明中国可以在 GPT-3 规模上进行训练，并具有额外的知识整合能力。
</Callout>

## 架构

- **2600 亿参数**（稠密 Transformer）
- 知识增强预训练与结构化知识图谱整合
- 多任务学习统一理解和生成
