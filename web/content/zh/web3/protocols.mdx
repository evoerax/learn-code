## Layer 1 格局 (2024)

| 链 | TVL | TPS (最大) | 共识 | 原生代币 |
|------|-----|-----------|--------|----------|
| **Ethereum** | ~$60B | ~15-30 | PoS | ETH |
| **Bitcoin** | ~$200B | ~7 | PoW | BTC |
| **Solana** | ~$8B | ~65,000 | PoH + PoS | SOL |
| **BNB Chain** | ~$5B | ~3,000 | PoSA | BNB |

## L1 对比

| 特性 | Ethereum | Solana | BNB |
|------|----------|--------|-----|
| **智能合约** | 是 | 是 | 是 |
| **EVM 兼容** | 原生 | 否 | 是 |
| **Rollup 中心** | 是 | 否 | 否 |
| **去中心化** | 高 | 增长中 | 中等 |
| **吞吐量** | 低 (L1) | 高 | 中 |
