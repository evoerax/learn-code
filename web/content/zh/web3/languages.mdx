# 智能合约语言

智能合约是部署在区块链上的程序，在条件满足时自动执行。编写它们需要专门的编程语言：

- **确定性** — 相同输入总是产生相同输出
- **Gas 优化** — 每次计算都消耗费用
- **安全性** — 漏洞可能导致资金永久损失
- **形式化验证** — 数学证明正确性

## 语言对比

| 语言 | 链 | 范式 | 风格 |
|------|-----|------|------|
| **Solidity** | Ethereum + EVM | 面向对象 | JavaScript 类 |

| **Rust** | Solana, Near, Polkadot | 系统 | C 类 |
| **Move** | Aptos, Sui | 资源导向 | Rust 类 |
