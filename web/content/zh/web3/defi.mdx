# DeFi — 去中心化金融

去中心化金融（DeFi）使用智能合约在公共区块链上重建传统金融工具——交易所、借贷、稳定币、衍生品。用户保留资产完全托管权，同时通过可组合协议访问金融服务。

## DeFi 堆栈

| 层级 | 描述 |
|------|------|
| **Layer 1** | Ethereum / Solana 等 |
| **协议** | DEX、借贷、稳定币 |
| **聚合层** | 收益优化器、组合器 |

---

## 关键指标

| 指标 | 描述 |
|------|------|
| **TVL** | 总锁仓量 — 存入的总资产 |
| **Volume** | 周期交易或借贷量 |
| **APY** | 年化收益 — 有效回报 |
| **利用率** | 已借出存款的百分比 |

---

## DeFi vs 传统金融

| 方面 | DeFi | 传统金融 |
|------|------|----------|
| **托管** | 自托管（你拥有密钥） | 第三方托管 |
| **访问** | 任何有互联网的人 | 需要 KYC，地理限制 |
| **时间** | 24/7/365 | 仅工作时间 |
| **结算** | 分钟或秒 | 数天 |
| **中介** | 无（或最少） | 银行、经纪人、清算所 |
| **可组合性** | 无需许可的混合 | 孤立系统 |
