# NFT 游戏与元宇宙

游戏中的 NFT 实现游戏资产的真正所有权、玩家驱动的经济体和新商业模式。

---

## GameFi 经济

### 传统 vs NFT 游戏

| 方面 | 传统 | NFT 游戏 |
|------|------|----------|
| **所有权** | 开发者拥有资产 | 玩家拥有资产 |
| **转让** | 不能 | 可以（市场） |
| **二次收入** | 无 | 版税给开发者 |
| **经济体** | 封闭 | 开放，玩家驱动 |

## P2E 问题

2021-2022 年 P2E 热潮因以下原因崩溃：

| 问题 | 描述 |
|------|------|
| **庞氏经济学** | 新玩家资金支付老玩家 |
| **无有机需求** | 资产仅因收益有价值 |
| **恶性通货膨胀** | 代币供应超过需求 |
| **机器人** | 自动farm贬值努力 |
