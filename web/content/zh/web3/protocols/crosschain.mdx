# 跨链基础设施

随着多个区块链获得采用，在链之间移动资产和数据变得至关重要。跨链基础设施实现互联的 Web3 生态系统。

---

## 为什么跨链重要

| 原因 | 描述 |
|------|------|
| **流动性碎片化** | 资本锁定在单一链 |
| **不同用例** | 每条链专长不同 |
| **用户体验** | 用户在多条链持有资产 |
| **套利** | 链间价格差异 |

---

## 桥接类型

### 锁定和铸造

1. 在源链锁定资产
2. 在目标链铸造打包资产

```
Chain A: 锁定 1 ETH → Chain B: 铸造 1 WETH
```

### 燃烧和铸造

1. 在源链燃烧资产
2. 在目标链铸造等量资产

```
Chain A: 燃烧 1 SOL → Chain B: 铸造 1 SOL
```

## 多链未来

| 愿景 | 描述 |
|------|------|
| **链抽象** | 用户不知道/不在意哪条链 |
