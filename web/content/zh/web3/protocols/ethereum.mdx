# Ethereum

Ethereum 是智能合约、DeFi 和 NFT 的基础平台。由 Vitalik Buterin 于 2013 年创建，首创可编程区块链。

---

## 演进

| 年份 | 里程碑 | 意义 |
|------|--------|------|
| **2015** | 主网上线 | 首个通用智能合约平台 |
| **2020** | EIP-1559 | 费用燃烧，基础费用机制 |
| **2022** | 合并 | PoS 过渡，减少 99.95% 能源 |
| **2023** | EIP-4844 (proto-danksharding) | 更便宜的 L2 数据 blobs |
| **2024+** | 完全 danksharding | 完全数据可用性采样 |

## Gas 和费用 (EIP-1559)

```solidity
// 优先费用（给验证者的小费）
PriorityFee = min(transaction.maxPriorityFee, transaction.maxFee - BlockBaseFee)

// 有效 gas 价格
EffectiveGasPrice = min(transaction.maxFee, BlockBaseFee + PriorityFee)

// 总费用
TotalFee = GasUsed * EffectiveGasPrice
```
