# 零知识证明

零知识证明（ZKP）允许一方证明给另一方某个陈述是真实的，而不透露除陈述有效性之外的任何信息。这种密码学原语是区块链隐私和扩容的基础。

## ZK 证明的三个属性

| 属性 | 描述 |
|------|------|
| **完备性** | 如果陈述为真，诚实验证者会被诚实证明者说服 |
| **可靠性** | 如果陈述为假，欺骗证明者无法说服诚实验证者 |
| **零知识** | 验证者除了知道陈述为真外，什么都不知道 |

## 在区块链中的应用

| 应用 | ZK 如何使用 |
|------|-------------|
| **隐私币** | Zcash 屏蔽交易金额和参与方 |
| **扩容** | ZK-Rollups 链下证明有效性，链上验证 |
| **桥接安全** | 证明状态转换而不信任中继 |
| **身份** | 证明公民身份而不透露身份证号 |
