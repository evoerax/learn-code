# ZK-SNARKs

ZK-SNARKs（零知识简洁非交互式知识论证）是区块链中部署最广泛的零知识证明系统，为 Zcash、zkSync、StarkEx 和大多数 ZK-Rollups 提供支持。

---

## SNARKs 背后的数学

### 多项式承诺

SNARKs 依赖于多项式承诺：

```
给定: P(x) 是 d 次多项式

承诺: C = g^P(s) 其中 g 是生成元，s 是秘密点

证明: 证明 P(s) = y 而不透露 s
```

---

## 信任设置仪式

ZK-SNARKs 需要"信任设置"——生成初始参数的过程。

### 为什么需要信任设置？

```
必须丢弃随机值（有毒废物）：
    τ = random()
    α = random()
    
如果 τ 未被销毁，攻击者可以创建假证明。
```

## Groth16 vs PLONK vs Halo2

| 协议 | 证明大小 | 验证 gas | 信任设置 |
|------|----------|----------|----------|
| **Groth16** | ~200 字节 | ~300K gas | 每电路（有毒） |
| **PLONK** | ~400 字节 | ~300K gas | 通用（可更新） |
| **Halo2** | ~800 字节 | ~200K gas | 无（递归） |
