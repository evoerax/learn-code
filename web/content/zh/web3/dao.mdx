# DAO — 去中心化自治组织

DAO 是由代码和集体决策治理的互联网原生组织。它们用透明规则、链上投票和社区控制的国库取代传统公司结构。

## DAO 类型

| 类型 | 代表 | 治理 |
|------|------|------|
| **协议 DAO** | Uniswap, Compound, Aave | 代币持有者治理协议 |
| **社交 DAO** | Friends with Benefits | 代币门控访问 |
| **投资 DAO** | BitDAO, The LAO | 池资本，投资项目 |
| **媒体 DAO** | Bankless, Decrypt | 去中心化内容 |
| **资助 DAO** | Gitcoin | 资助公共产品 |

## 治理代币经济学

```solidity
contract GovernanceToken is ERC20 {
    function delegate(address delegatee) external {
        _moveDelegation(delegates[msg.sender], delegatee, balanceOf(msg.sender));
        delegates[msg.sender] = delegatee;
    }
}
```
