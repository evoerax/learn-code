# 借贷与借贷

DeFi 借贷协议使用户能够赚取存款利息或以抵押品借入资产——所有这些都无需银行。

---

## 健康因子

健康因子决定头寸是否可以被清算：

```
健康因子 = (抵押品 × 清算阈值) / 借入价值
```

| 健康因子 | 状态 |
|----------|------|
| > 1.0 | 安全 |
| = 1.0 | 清算阈值 |
| < 1.0 | 可清算 |

## 利率模型

### 变量利率

利率根据利用率连续调整：

| 参数 | 描述 |
|------|------|
| **U_OPTIMAL** | 目标利用率（如 80%） |
| **SLOPE_1** | 低于最优时的增长率 |
| **SLOPE_2** | 高于最优时的增长率 |

## 闪电贷

**用例:** 套利、清算、抵押品交换
