# 工具链与评测

**范围：** 团队如何 **证明系统可用**：评测集、基准脚手架、回归测试、追踪、漂移监控，以及提示词/配置的发布门禁。

---

## 常见产物

- **黄金集**：体量小、信号密度高的评测切片。
- **追踪**：单次请求在路由、检索、工具、模型之间的 span。
- **报告**：按模型版本与提示词哈希汇总的分数趋势。
