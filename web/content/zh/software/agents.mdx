# 智能体与编排

**范围：** 让模型与外部世界 **协同工作** 的软件：工具 schema、检索、状态机、人在回路，以及多智能体拓扑。

---

## 常见设计取舍

- **确定性 vs 探索**：何时用有向无环的流程，何时允许重规划。
- **上下文预算**：哪些进 prompt，哪些进外存。
- **安全与审计**：工具入出参记录、策略过滤、审批流。
