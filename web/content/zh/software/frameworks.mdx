# 框架与库

**范围：** 用于 **训练与研究** 模型的软件：张量运行时、自动微分、优化器，以及数据并行 / 张量并行 / 流水线并行、ZeRO 类分片等 **分布式训练** 拼装方式。

---

## 概念分层

1. **算子 / 编译层**：算子融合、自定义 CUDA/ROCm、图捕获、以及在框架内暴露的编译路径（如类 XLA / Inductor）。
2. **核心库**：动态图与静态图、混合精度、断点与恢复。
3. **训练上层**：Trainer、调度器、与数据管线、实验追踪系统的对接。
