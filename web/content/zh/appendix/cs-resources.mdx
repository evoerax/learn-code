# CS 学习资源（网站与仓库）

以下链接为**自用向**备忘，偏向**免费开放**或**源开放**内容；机构政策与版权以各站点为准。与本站 [编程语言时间线](/zh/timeline) 可对照阅读。

## 公开课与系统路线

| 资源 | 说明 |
| :--- | :--- |
| [Teach Yourself Computer Science](https://teachyourselfcs.com/) | 英文：按主题（算法、体系结构、操作系统等）给自学路线图与教材建议 |
| [OSSU Computer Science](https://github.com/ossu/computer-science) | GitHub：社区维护的 **CS 本科水平** 免费课程清单与进度表 |
| [MIT OpenCourseWare](https://ocw.mit.edu/) | 麻省理工公开课：讲义、作业、录像（如 **6.006 算法**、**6.004 计算结构** 等） |
| [Stanford Engineering Everywhere (SEE)](https://see.stanford.edu/) | 斯坦福部分课程归档（如 CS106A/B 等，以页面现有课程为准） |
| [CS50](https://cs50.harvard.edu/) | 哈佛 CS50：从零基础到 C、Python、Web、SQL 等广度入门 |
| [UC Berkeley CS61A](https://cs61a.org/) | **SICP 风格** Python/Scheme 大规模入门课（学期页面随年份更新） |
| [Crash Course Computer Science](https://www.youtube.com/playlist?list=PL8dPuuaLjXtNlUrzyH5r6jN9ulIgZBZb_) | YouTube 短片系列：概念扫盲友好 |
| [cs-self-learning（中文）](https://github.com/PKUFlyingPig/cs-self-learning) | GitHub：**中文**计算机自学指南，含国内外课程链接 |
| [CS Wiki / 鹏城实验室等汇总（中文）](https://csdiy.wiki/) | **CS DIY**：中文社区常见的选课、电子书与工具导航（维护状况以站点为准） |

## 使用建议

- **路线 > 收藏**：优先跟一门课做完实验，再扩展横向链接。  
- **英语主厨**：前沿 CS 资料仍以英文为主，可与中文导读（如 cs-self-learning）穿插。  
- **仓库星标会过期**：若某课程迁域名或闭源，以官方系主页为准。
