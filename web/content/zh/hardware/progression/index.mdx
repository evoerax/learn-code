# 进阶层导读

**进阶层**放在 **基础层** 与 **整机脉络** 之间：基础层把电、算、动、感、控、联打牢后，整机章节专注无人机 / 汽车 / 机器人三类集成；进阶层则帮你把**不够放进单章、但又迟早要啃**的块对齐到同一张地图里——尤其是数理根基（见 [编程语言与 timeline](/zh/timeline)）、机械结构、智能算法（见 [软件总览](/zh/software/overview) 与 [AI 模型](/zh/ais/overview)）、工程素养，以及一站式的**完全精通路线（可勾选进度）**。
