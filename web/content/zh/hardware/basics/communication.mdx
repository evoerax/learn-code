# 总线与通信

## 板级与机内

- **UART / SPI / I2C**：MCU 与传感器、驱动芯片。  
- **CAN**：车规与工业机器人里多节点、仲裁与错误帧粗识。  
- **以太网 / USB**：高带宽传感器、调试与上位机。  

## 链路层（航模/小车）

- **遥控接收**：协议与安全（失控保护、看门狗）。  
- **数传 / 图传**：带宽、时延与法规（频段、功率）。

## 设计要点

带宽、时延、抖动；**协议与线序文档化**，避免样机「只有一个人能飞/能开」。
