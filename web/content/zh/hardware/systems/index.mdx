# 整机脉络导读

**一级「整机脉络」**下有三个**二级入口**（侧栏可展开），每个入口里还有**三级专题**，便于按子系统补课。

---

## 二级：三条整机路线

| 维度 | 无人机 | 汽车 | 机器人 |
|------|--------|------|--------|
| **动力架构** | 高转速 BLDC + ESC + 螺旋桨（升力来自桨叶推力） | 牵引电机（PMSM / BLDC）+ 减速器 + 车轮（力矩直接作用于车轮） | 关节电机（BLDC/FOC）+ 谐波减速器 / 行星减速（力矩在关节输出端） |
| **安全关键标准** | 无强制认证（ hobby 除外）；功能安全依赖飞控固件 | ISO 26262 ASIL B/C/D（乘用车）；强制型式认证 | ISO 10218-1/2 + ISO/TS 15066（协作臂）；风险评估文档化 |
| **典型控制带宽** | 内环 1–8 kHz（电机转速），姿态环 100–500 Hz，位置环 1–5 Hz | 动力域 1–2 kHz（电机电流），底盘安全域 50–100 Hz | 关节电流环 500–2 kHz，速度环 100–300 Hz，位置环 10–50 Hz |
| **实时通信** | RC（SBUS/CRSF）+ MAVLink 遥测；数传链路 | CAN/CAN FD + Automotive Ethernet + LIN；OBD-II 诊断 | EtherCAT / PROFINET（工业）；ROS 2 / DDS（研究） |
| **传感器冗余** | IMU 典型单源 + 气压计 + GPS（户外）；LIDAR 可选 | 传感器冗余（毫米波雷达 + 摄像头 + 超声波）+ EKF 融合 | 环境感知激光雷达 + 视觉；关节双编码器（粗精） |
| **法规/认证** | 各国民航无人机法规（重量分级 / VLOS / BVLOS）；CE / FCC | 车辆型式认证（排放 / 安全 / EMC）；UN ECE 法规体系 | CE 机械指令（工业机器人）；ISO/TS 15066（协作安全） |
| **典型功耗范围** | 50–5,000 W（多旋翼悬停 ~200–2,000 W） | 10–150 kW（乘用车驱动）；12V/48V 辅助电网 | 100 W–20 kW（工业臂 6–20 kg 负载典型 1–5 kW） |
| **入口** | [无人机](/zh/hardware/systems/drones) | [汽车](/zh/hardware/systems/automotive) | [机器人](/zh/hardware/systems/robotics) |
| **三级专题侧重** | 机架动力、飞控回路、链路安全 | 车载网络、线控、辅助驾驶传感器 | 机械臂、移动底盘、系统集成与安全 |
