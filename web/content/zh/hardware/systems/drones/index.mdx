# 无人机（二级总览）

侧栏 **三级**：[机架与动力](/zh/hardware/systems/drones/airframe) · [飞控与回路](/zh/hardware/systems/drones/flight_control) · [链路与安全](/zh/hardware/systems/drones/link_safety)

## 子系统链（双向路径）

无人机子系统链分为两条路径，方向相反但相互依赖：

**机械链（需求传导）：** 机架 → 无刷电机 → 电调（ESC）→ 螺旋桨 → 电池（BMS）→ 连接器与电源分配板（PDB）

**控制链（信号闭环）：** 飞控 → IMU → ESC → 电机 → 螺旋桨推力 → 机架姿态 → IMU（闭环）

机械链决定各部件的极限参数（电机 KV 值 × 桨直径决定最大转速，电池 C 率 × 容量决定最大放电电流）；控制链决定各环的时间响应（内环 1–8 kHz / 中环 100–500 Hz / 外环 10–100 Hz）。两条链在 ESC 和电池处交汇，ESC 接收飞控的 PWM/SBUS 信号并调制电池输出功率，三级页各有详细展开：[机架与动力](/zh/hardware/systems/drones/airframe) 覆盖电机 KV/桨距/C 率/连接器选型；[飞控与回路](/zh/hardware/systems/drones/flight_control) 覆盖 IMU 安装/嵌套环带宽/日志分析；[链路与安全](/zh/hardware/systems/drones/link_safety) 覆盖遥控协议/失控保护状态机/空域合规。

## 控制层次

- **内环**（速率 / 姿态）：飞控以内环几百 Hz 的频率维持角速率稳定，内环带宽最高、延迟最低，直接决定飞行手感。
- **外环**（位置 / 航迹）：外环依赖良好的状态估计（GPS 或视觉），需要比内环低一个数量级的带宽（10–100 Hz），主要任务是抗风与轨迹跟踪。

内外环带宽的比值（通常 10:1）决定了两者的调参优先级：先调内环（姿态环）至稳定，再调外环（位置环），顺序不可颠倒。

## 安全与法规

**解锁逻辑：** 飞控上电后需满足多项安全条件（IMU 就绪、GPS 锁定的条件解锁、遥控信号有效）方可解除电机锁定，防止意外启动。

**失控保护（Failsafe）：** 遥控信号丢失时触发预设动作——RTL（返回发射点）、Land（原地降落）或 Motor Cut（切断电机）。各模式的前提条件不同：RTL 需要 GPS 可用且 home 点已记录，Motor Cut 适用于室内无 GPS 场景。

**各国法规要点：** 重量分级（美国 FAA 250 g 以上需注册）、视距内飞行（VLOS）要求、禁飞区（机场周边/人口密集区）、BVLOS 需要额外申请。

**量产认证：** CE / FCC 电磁兼容认证之外，还需通过民航局型号合格审定（STC），涉及振动测试、温度循环、功能安全分析等。

## 飞行参数估算（三维权衡快速参考）

| 参数 | 影响要素 | 典型范围 |
|------|---------|---------|
| 悬停时间 | 电池容量（mAh）× C 率 / 悬停电流 | 15–30 分钟（5 Ah / 4S / 250 g 机型） |
| 最大推重比 | 电机功率 / 全机重量 | 2:1 为最低安全余量，3:1 以上才有机动性 |
| 最高速度 | 推重比 / 空气阻力 | 50–80 km/h（四旋翼典型） |
| 通信距离 | 发射功率 / 频率 / 天线增益 | 1–5 km（典型 2.4 GHz 遥控） |

实际飞行时间由悬停电流和电池可用容量共同决定，不可单凭 C 率估算。

## 学习路径建议

套件焊接调试 → 地面站日志分析（识别振荡来源）→ 换桨/配平理解机架动力学 → 自定义 PID 参数 → 考虑自研飞控软件。在未充分理解现有固件为何会失败之前，不要贸然重写飞控——这是最常见的新手陷阱。
