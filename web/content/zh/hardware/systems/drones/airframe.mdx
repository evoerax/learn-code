# 机架与动力（三级）

## 机架与桨

**机架尺寸与惯性矩**：轴距（wheelbase）决定整机惯性——5 寸穿越机 inertia ~0.001 kg·m²，10 寸农业机 inertia ~0.05 kg·m²。惯性越小，响应越快但抗风能力越弱。

**桨叶直径/螺距与 KV 匹配原则**：KV 值 × 电池电压 = 空载电机转速（rpm）。4S（14.8V）+ 2306 1700KV = 25160 rpm，配 5 寸桨（直径 5"~127mm）在悬停约 60% 油门时效率最高。加大桨径可降低转速需求（提升效率但增加机械尺寸），加螺距可增大静压（提升推力但降低效率）。

**共振对 IMU 的影响与缓解**：机架固有频率若与电机 PWM 载波（典型 24 kHz/32 kHz）重合，会导致 IMU 加速度数据出现拍频噪声，姿态估计发散。缓解措施：加软质减震球（硬度 30~60 Shore A）、将 IMU 板与机架间悬置（使用硅胶垫圈）、使用较高 PWM 频率（> 32 kHz）避免与常见机架模态重叠。

## 电调与电机

**三相接线与换相逻辑**：无刷电机三根相线（U/V/W）任意两根对调会逆转方向。电调固件中的 FOC（场定向控制）或六步换相（six-step commutation）决定何时给哪相通电——六步换相每 60° 电角度切换一次通电状态，FOC 则连续跟踪转子位置，效率更高但算力需求大。电调 firmware 中的 governor 模式可在负载变化时自动补偿转速（常见于固定翼，穿越机多用手拧）。

**同步启动 vs 异步启动**：同步启动指电机转速随油门线性跟踪，无转速滞后；异步启动（异步 commutation）多见于低 KV 大电机，转子加速慢于磁场旋转，可能出现短暂抖动。堵转保护（blocked-rotor protection）通过检测电流峰值（> 额定 3~5 倍）在 ms 级切断输出，防止线圈烧毁。

**电池 C 率与瞬态电流**：电池最大持续放电倍率（C）决定安全供电电流。2200 mAh 45C 电池持续放电 99A，瞬态（10 秒以内）可达 2C（198A）。电调持续电流规格应 ≥ 电池最大持续电流；导线截面积（AWG）按 10A/mm² 选择（20AWG≈0.5mm²载流5A，AWG 越小线越粗）。

## 电池与配电

**锂聚合物存储与运输法规**：LiPo 电池长时间存储应保持在 3.7~3.85V/cell（约 40~60% SOC），避免过充（高于 4.2V/cell）或过放（低于 3.0V/cell）导致容量不可逆衰减。运输时必须放入防火袋或阻燃箱内；航空公司对 >100Wh 电池有限制（国航标准每人不超过 160Wh，且备用电池必须单独防护）。

**常见插头规格对照**：XT60（Dean-T 兼容，额定 60A，穿越机最常用）、XT90（90A，大载重机）、AS150（防反插，额定 150A，航模老标准）。EC5/Hilo（分电板端接，额定 60A）。插头规格选型时务必使额定电流 ≥ 电调峰值电流的 1.5 倍，留有余量。

**分电板（PDB）设计要点**：分电板将电池 DC 输入分配给各电调，典型设计使用 PCB 铜厚 2 oz（载流约 30A/inch）及布线宽度计算。高级 PDB 集成 bulk 电容（1000~2200 µF 6S），吸收电机换相电流尖峰（di/dt 可达 10A/µs），降低电源纹波。

**低压报警与降落策略**：
- 3.6~3.7V/cell（软报警）：提示返航，冗余电机余量开始不足
- 3.5~3.6V/cell（硬报警）：立即返航，推力余量不足额定 50%
- 3.3~3.5V/cell（严重报警）：触发 Land 或 RTL，飞机可能无法维持高度
- 3.3V/cell 以下（强制降落）：电调切断输出以保护电池不被过放

建议在地面站设置每 Cell 电压阈值，结合飞行时间估算，提前规划返航窗口（剩余电量 30% 时必须返航是安全惯例）。
