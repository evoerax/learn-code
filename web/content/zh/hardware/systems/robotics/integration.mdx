# 系统集成与安全

本页面覆盖从「机器人独立运行」到「机器人在生产环境中安全可靠运行」之间所需的硬件和系统级问题：安全架构、协作空间设计、 commissioning 清单。这些是概念验证（PoC）演示与工程化部署之间的本质差距。

---

## 工业场景

工业机器人部署的首要条件不仅是性能指标——安全架构是一类硬件需求。

**安全 PLC 与互锁**：安全可编程逻辑控制器（Safety PLC）负责监控急停电路、安全区域侵入和模式切换。它与运行生产逻辑的标准 PLC 不同——Safety PLC 通过 IEC 62061 认证为 SIL 2 或 SIL 3，运行冗余逻辑，且主控制器无法覆盖。每个机器人工作站至少包含：接入 Safety PLC 的 E-stop 链、区域光幕或区域扫描仪、门互锁开关（工人进入工作站时强制安全停机）。

**风险评估与残余风险文档**：ISO 12100 要求每个机器人安装必须进行正式风险评估——识别夹点、挤压区和摆动范围。输出残余风险文档，且须定期维护审查。这不是软件任务，涉及物理防护、安全距离（ISO 13855 根据接近速度计算到危险的最小距离）和功能限制文档。

**典型安全架构组件对照表**：

| 组件 | 功能 | 标准 |
|---|---|---|
| Safety PLC | 冗余逻辑，SIL 2/3 | IEC 62061，ISO 13849 |
| E-stop 链 | 硬接线安全停止，故障安全 | ISO 13850 |
| 光幕 | 区域检测，即时停止 | IEC 61496 |
| 安全地垫 / 区域扫描仪 | 区域存在检测 | IEC 61496 |
| 门互锁开关 | 防护门位置监控 | ISO 14119 |
| 安全继电器 | 监控并执行停止信号 | IEC 62061 |

## 协作机器人

协作机器人（cobot）设计为与人类共享工作空间，无需物理隔离——但「协作」不等于「无限力」。力限制是关节执行器的硬件属性，而非软件设置。

**力与扭矩限制**：ISO/TS 15066 定义了各人体部位的协作操作功率和力限值。这些限制由关节扭矩传感器、碰撞检测（电流 sensing）和安全限速共同强制执行。当接触事件超过阈值时，机器人执行受控退回，而非失控坠落。

**碰撞检测**：最简单形式是基于电流的检测——电机控制器监测定子电流，在检测到异常时在几毫秒内触发停止。更先进的 cobot 在每个轴上使用关节扭矩传感器以获得更高分辨率。碰撞检测须根据机器人声明的工作空间和负载进行测试——5 kg 负载与 0.5 kg 负载产生的惯性完全不同。

**示教模式与手动引导**：在示教模式下，机器人在降低的速度下运行且无负载，操作员通过引导末端执行器遍历示教点。这些示教点被记录后可回放。示教模式需要一个物理使能装置（deadman switch），操作员必须按住——释放即触发立即安全停止。这与纯位置控制的本质区别在于：操作员释放时机器人必须主动抵抗重力。

**协作臂 vs. 传统工业臂**：传统 6 轴工业臂（如 Fanuc、Kuka、ABB）速度更快、负载更高，但需要安全围栏，不适合人类接近。协作臂（如 Universal Robots、Franka、Techman）本质上通过设计实现力限制，支持共享工作空间。选型是工程权衡：吞吐量和负载 vs. 工作空间灵活性和人机交互。

## 样机清单

从「机器人能演示一个任务」到「机器人能在现场完成 commission」需要工程化强化。以下清单捕获了在实验室环境中容易被忽视的硬件级问题。

**上电顺序**：机器人工作站应遵循确定的上电顺序：Safety PLC 优先，然后是驱动器，最后是主控制器。跳过此顺序可能导致 Safety PLC 处于不确定状态。关机顺序同样重要——断电前执行受控停止（STO）可防止电机抱闸磨损和控制器文件损坏。

**E-stop 可达性**：每个机器人工作站必须在所有正常操作位置（包括任何维护通道）都能触及 E-stop 按钮。ISO 13850 规定了最大触及时间和放置标准。E-stop 按钮必须为蘑菇头式、黄红配色、物理锁定——软件停止按钮不满足标准要求。

**线缆应力与维护口**：机器人线缆包（从底座到工具的电缆和软管束）在运行过程中持续弯曲。线缆必须根据预期的循环次数选择规格；在工业线缆包中使用消费级线缆会导致过早失效。维护口应布置在线缆更换时无需拆卸整个工作站的位置。

**Commissioning 差距**：工作台上的演示不等于已 commission 的工作站。工程化差距包括：接地与 EMC 合规（CE 标志）、断电时紧急回原点（STO 不定义「原点」是什么）、极端环境温度下的节拍时间验证，以及安全功能测试日志文档化。

## 与其他页面的关系

系统集成与安全处于机器人整机系统的交叉点：

- **执行器**：协作臂关节的力限制能力是关节执行器的硬件属性——见 [机械臂与关节](/zh/hardware/systems/robotics/manipulators)
- **移动底盘**：AGV 或 AMR 工作站增加了 fleet manager 安全（VDA 5050 急停传播）到标准机器人工作站清单——见 [移动底盘与 AGV](/zh/hardware/systems/robotics/mobile_base)
- **控制**：Safety PLC 和 E-stop 链回归到底层伺服控制环——见 [控制与估计基础](/zh/hardware/basics/control)
- **通信**：EtherCAT 或 PROFINET 是将机器人控制器与 Safety PLC 集成的常见现场总线——见 [通信基础](/zh/hardware/basics/communication)
