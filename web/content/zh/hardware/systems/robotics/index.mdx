# 机器人（二级总览）

侧栏 **三级**：[机械臂与关节](/zh/hardware/systems/robotics/manipulators) · [移动底盘与 AGV](/zh/hardware/systems/robotics/mobile_base) · [系统集成与安全](/zh/hardware/systems/robotics/integration)

## 构型

串联机械臂、SCARA、并联、移动底盘（差动/阿克曼/全向）与复合型 AGV。

## 硬件要点

关节减速器（谐波/行星）、扭矩传感器（可选）、末端 IO 与**安全围栏/光幕**（工业场景）。

## 控制与软件

关节伺服、轨迹规划、与上位调度/MES 对接属**系统问题**；嵌入式层仍回到 [控制](/zh/hardware/basics/control) 与 [通信](/zh/hardware/basics/communication)。

## 个人路径

桌面机械臂套件 → 单轴辨识与 PID → 多轴插补 → 再选移动平台或协作臂标准接口。
