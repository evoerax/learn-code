# 移动底盘与 AGV

移动底盘是自主移动机器人（AMR）或自动导引车（AGV）的运动层。与固定基座的机械臂不同，移动底盘必须同时解决"我在哪里"和"我要去哪里"两个问题，同时管理牵引力、地面条件和（工业场景下）与车队管理器的集成。轮子的机械构型决定了运动学约束，传感器套件决定了机器人估计自身位姿的精度。

---

## 构型选择

轮型选择是移动底盘最根本的设计决策。

**差速驱动** 是室内 AMR 最简单、最常见的配置。两个独立驱动的轮子布置在同一车轴上，通过差速速度实现前进/后退和原地转向，转弯半径为零，无需阿克曼转向几何，控制律简单：`（左速，右速）→（线速度，角速度）`。缺点是在不平地面上可能出现随动轮偏摆，导致里程计漂移。

**阿克曼转向** 复制汽车几何：两个后轮驱动，两个前轮装在转向主销上，带有前束角，拐弯时轮胎零滑移。这是室外配送机器人、农业漫游车以及在平坦地面高速移动平台的标配配置。阿克曼无法原地转向，最小转弯半径由轴距决定，高速平台还需要悬架来保持轮胎与地面接触。

**全向轮**（轴向与轮轴呈 45° 或 90° 的自由滚轮，或带倾斜滚轮的麦克纳姆轮）可实现真正的全向运动：平台可以同时侧向移动和旋转，无需差速驱动的复杂走位。在工厂车间等狭窄空间里，机器人需要侧向滑入货架时非常有用。代价是每个轮的承载力下降，麦克纳姆逆运动学复杂，且对地面杂物敏感——碎屑会卡住倾斜的滚轮。

**履带式（滑移转向）** 使用两条无限履带独立驱动，类似推土机。机器人通过差速履带速度产生滑移转向。履带在软质或不平地面（室外、泥土、地毯）上牵引力出色，但拐弯时滑移明显，降低里程计精度并在光滑地面上加速履带磨损。

| 构型 | 原地转向 | 最小转弯半径 | 侧向移动 | 典型用途 |
|---|---|---|---|---|
| 差速驱动 | 可以 | 零 | 不可以 | 室内 AMR、医院、仓库 |
| 阿克曼 | 不可以 | 取决于轴距 | 不可以 | 室外配送、农业 |
| 麦克纳姆/全向 | 可以 | 零 | 可以 | 紧凑室内空间、工厂 |
| 履带式（滑移） | 可以 | 近似零 | 不可以 | 室外、崎岖地形 |

## 驱动系统与电机选型

选定运动学构型后，需要为驱动系统选型。三个关键参数是**爬坡能力**、**额定负载**和**最高速度**。

轮端电机扭矩由负载的反射惯性决定。机器人必须克服滚动阻力、克服坡度，并在要求时间内加速到最高速度。对于配有两个驱动轮和两个随动脚轮的差速驱动 AMR，驱动轮通常分担 30%–60% 的负载（其余由脚轮承担）。电机几乎都是无刷直流电机或齿轮减速直流电机，减速箱以行星轮最常见。对于协作机器人底座，电机、编码器、驱动、制动器四合一的集成关节模组正越来越普遍。

制动器选型对安全至关重要：在坡道上停稳的 AMR 必须保持位置。失电时制动的弹簧制动器（fail-safe）是标准配置，尤其用于天花板悬挂式或倾斜轨道 AGV。

## 里程计与定位传感器

移动底盘需要知道自身相对于环境的位置。这通过分层方法解决：

**轮式里程计** 是基础传感器。光学或磁编码器测量每个驱动轮的转动量，通过运动学模型（差速或阿克曼）将轮转增量积分为估计位姿 `(x, y, θ)`。里程计漂移随距离累积，因为它无法修正轮子打滑、地面不平整或脚轮偏摆。典型室内 AMR 里程计误差为行驶距离的 1%–5%。

**惯性测量单元（IMU）** 在编码器刻度之间补充数据。6 轴或 9 轴 IMU（加速度计 + 陀螺仪 ± 磁力计）提供航向角速率和线加速度。扩展卡尔曼滤波器（EKF）将轮式里程计和 IMU 数据融合为更平滑且不易受高频轮子打滑影响的位姿估计。IMU 不能修正长期漂移——那需要外部参考。

**2D 激光雷达** 是室内 AMR 定位的主导外部传感器。旋转激光雷达以 10–25 Hz 扫描水平面，产生 2D 点云。SLAM（同步定位与地图构建）算法将当前扫描与预存地图匹配，修正里程计漂移。有效量程（工业激光雷达通常为 0.1–30 m）和角分辨率（0.25°–0.5°）决定系统在高特征稀疏环境中检测特征的能力。室外底盘通常加装 GNSS RTK 以获得绝对定位，解决室内系统无法修正的长期漂移。

**自然与人工路标** 提供额外修正：安装于墙壁的 AprilTag、天花板标记、激光雷达反光板或 RFID 地面标记。在可接受永久地面标记的工厂环境中，磁条或线导引（接地感应回路）提供无漂移的路径跟踪，代价是基础设施投入。

## 导航、调度与车队集成

移动底盘的导航栈分层运作：

- **路径规划**：给定起始位姿和目标位姿，全局规划器（A*、RRT 或动态窗口法）通过已知地图计算无碰撞路径。
- **运动控制**：局部规划器或纯追踪控制器追踪全局路径，同时对激光雷达检测到的临时障碍做出反应。
- **车队管理**：在多 AMR 仓库中，车队管理器（通常兼容 VDA 5050）分配任务、管理交叉冲突并优化路径调度。

**VDA 5050** 是 AMR 车队通信的新兴德国汽车行业标准，定义 fleet manager 与各 AGV 之间的 JSON 消息格式，实现状态、订单、订单更新和故障处理的标准化通信。采纳 VDA 5050 可在同一车队中混用不同厂商的 AMR，对大型部署有重大实际意义。

**AGV 与 AMR 的区别** 至关重要：AGV 依赖固定基础设施（磁带、线导引或画线），无法自主绕开障碍物重新规划路线；而 AMR 使用 SLAM 动态导航，无需改变物理基础设施即可接收新目标。现代室内物流机器人大多数是 AMR。

## 与其他页面的关系

- **机械臂与关节**（manipulators）：复合机器人 = 移动底盘 + 机械臂。移动底盘负责运动，臂负责操作。集成点是共享电源总线和底座与臂控制器之间的通信桥接（通常为 EtherCAT 或 ROS 2）。
- **执行器与电源**（basics/actuation）：移动底盘的驱动电机是无刷直流电机或齿轮减速直流电机。电机选型原则（扭矩、速度、电流）在此处同样适用，与无人机电子调速器和汽车牵引电机一致。
- **控制与估计**（basics/control）：轮式里程计与 IMU 的传感器融合是基础控制章节中卡尔曼滤波器概念的实践应用。
- **系统集成与安全**（integration）：工业移动底盘必须实现紧急停车回路、安全等级受控停止（STO，符合 IEC 62061），以及通过安全激光扫描仪（如 SICK S300 或同类产品）碰撞检测，其定义保护区域和告警区域。

## 实践注意事项

移动底盘选型时三个最常见的错误是：低估地面表面对里程计精度的影响；为目标环境狭窄通道选用的激光雷达角分辨率不足；以及漏选在加速时会产生侧向滑动的随动脚轮（合适的 360° 旋转脚轮配合充足轮径可减少此问题）。初次搭建时，采用前随动脚轮的差速驱动平台，配合安装在踝部高度（约距地面 200 mm）的 2D 激光雷达，提供了最直接的 SLAM 和导航开发路径。
