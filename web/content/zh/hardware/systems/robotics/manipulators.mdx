# 机械臂与关节

工业机械臂是现代自动化的核心——从汽车装配到半导体包装。与 [基础 · 执行器](/zh/hardware/basics/actuation) 中介绍的通用执行器不同，机械臂必须解决完整的运动学链：将末端执行器的笛卡尔目标转化为关节空间轨迹，同时遵守扭矩限制、关节限位和奇异位形约束。本页涵盖硬件基础与运动学入口。

---

## 关节硬件

### 谐波减速器

**谐波减速器**（strain wave gear）由三个组件构成：刚轮（固定外环，内齿）、柔轮（薄壁柔性外环，齿数略少）和波发生器（使柔轮变形的椭圆轴承）。任意时刻只有约 30% 的齿参与啮合，使得在紧凑体积内实现高减速比（30:1 到 320:1），且回差接近零。

关键特性：
- **减速比**：工业机械臂单级减速通常 30:1 到 160:1，多级可达 320:1。
- **回差**：基本为零——对取放和焊接的精密定位至关重要。
- **扭矩密度**：相对于体积较高，是六轴串联机械臂的主流减速器。
- **寿命**：通常 10,000 到 20,000 小时，受柔轮疲劳限制。
- **预紧**：必须预紧以消除死区；会增加启动力矩，但对精度是必要的。

### 行星齿轮箱

**行星齿轮箱**通过太阳轮、行星轮和内齿圈实现减速。相比谐波减速器，它提供更高扭矩容量和更简单的结构，但存在一定回差（精密行星减速器通常 1–5 弧分）。常见于 SCARA 机械臂和大负载机械臂的基座关节（在那些回差要求不高的场景）。

关键特性：
- **减速比**：多级行星通常每级提供 3:1 到 100:1。
- **回差**：精密行星 1–5 弧分 vs 谐波接近零回差。
- **成本**：同等扭矩下低于谐波——在成本敏感应用中首选。
- **效率**：每级 90%–97%，略高于谐波减速器。

### 集成关节模组

现代协作机器人越来越多地采用**集成关节模组**（也称关节执行器或关节套件）：将电机、驱动器、减速器、编码器和制动器集成在单一 package 中。集成关节模组简化了装配和布线，但使现场维修更困难。

典型集成关节模组包含：
- **无刷直流电机**（BLDC），KV 值与减速器输出匹配
- **绝对编码器**（磁式或光式），提供关节位置反馈
- **驱动器**：CAN 或 EtherCAT 接口；电流环在模组内部运行
- **谐波或行星减速器**，预集成
- **保持制动器**，断电时防止重力导致的手臂下落

### 回差与刚度

**回差**（啮合齿之间的间隙）在换向时产生定位误差——是慢速精密运动中"抖动"或振荡的已知来源。缓解策略：
- 预紧式谐波减速器以更高启动力矩为代价消除回差。
- 双驱动系统（两个电机作用于同一关节）可通过差分控制补偿回差。
- 在控制器中测量并补偿回差偏移的标定程序。

**刚度**（柔度的倒数）决定关节在负载下的偏转量。高刚度是精密任务（焊接、加工）的必要条件；低刚度（更柔顺的臂）在协作机器人中更可取，因为必须限制与人的接触力。

## 运动学入口

### 正运动学

**正运动学（FK）** 根据给定关节角度计算末端执行器的位姿（位置和姿态）。在六轴串联机械臂中，这需要通过每个关节链锁四个×四个齐次变换矩阵。Denavit-Hartenberg（DH）约定是给每个关节分配坐标系的标准化方法。

对每个关节 i：
1. 连杆长度 a_i：沿前一关节轴到新关节轴的距离。
2. 连杆偏距 d_i：沿新关节轴从前连杆到当前连杆的距离。
3. 关节角 theta_i：绕新关节轴的旋转角度。
4. 扭曲角 alpha_i：两关节轴之间的夹角。

最终变换是所有单独连杆变换的乘积。

### 逆运动学

**逆运动学（IK）** 求取实现期望末端执行器位姿所需的关节角度。对于具有球形手腕（后三个轴相交）的六轴机械臂，IK 可分解为位置 IK（前三个关节）和姿态 IK（后三个关节），使实时计算变得可行。

关键挑战：
- **多解性**：大多数六轴机械臂有多达八个有效 IK 解（肘上/肘下、肩左/肩右、手腕翻转）。
- **奇异位形**：某些关节配置使 IK 病态（见下文）。
- **工作空间边界**：解仅在臂的可到达工作空间内存在。

### 工作空间

**可到达工作空间**是末端执行器至少以一种姿态可到达的所有位置的集合。**灵巧工作空间**（可到达工作空间的子集）是所有姿态均可实现的位置集合。工作空间分析对单元格布局至关重要——如果目标位置落在工作空间之外，机器人可能无法到达某些夹具位置。

典型六轴工业臂（500 mm 臂展）：
- 可到达球面半径等于最大臂展。
- 灵巧工作空间通常是可到达体积的 30%–50%，集中在机器人基座附近。

### 奇异位形

**奇异位形**发生在机器人雅可比矩阵失去秩的时候——臂相对于末端执行器运动获得或失去一个自由度。在奇异位形处，某些笛卡尔方向变得不可控（或需要无限关节速度）。

三类常见奇异位形：
- **腕部奇异**：当手腕中心位于经过肩部轴的直线上时——手腕姿态变得模糊。
- **肩部奇异**：当肩部轴与肘部轴共线时——肘部配置变得不确定。
- **工作空间边界奇异**：接近可到达工作空间边缘时，小的笛卡尔运动需要大的关节运动。

工业控制器处理奇异位形的方式：
- **关节速度限制**：在奇异位形附近防止物理上不可能的命令。
- **任务框架重定向**：移动使问题方向被规避。
- **完全停止**：如果奇异位形被过于接近地逼近。

## 与控制的衔接

### 三环分层

机械臂控制遵循与其他精密系统相同的三环架构，只是针对机械负载做了调整：

1. **电流环**（最内层，最快）：使用 PWM 和 Park/Clarke 变换控制电机相电流。典型带宽 500–2000 Hz。电流环通常完全在关节驱动器内部。
2. **速度环**（中层）：使用编码器闭环电机轴角速度。典型带宽 100–300 Hz。提供阻尼并处理摩擦。
3. **位置环**（最外层，最慢）：根据期望与测量末端执行器或关节位置之间的误差命令速度。典型带宽 10–50 Hz。

### 重力补偿

重力持续作用于每个连杆。没有补偿时，关节在静止时可能无法保持位置——PID 中的积分项会持续积分试图对抗重力。**重力补偿**计算每个关节在当前姿态下抵消各连杆重量所需的扭矩，并将其作为前馈项加入：

```
tau_g = J^T * F_gravity
```

其中 J 是雅可比转置，F_gravity 是作用在末端执行器上的重力向量。有了重力补偿，位置环只需处理惯性和摩擦负载，显著减少积分饱和。

### 前馈与摩擦补偿

除重力外，**摩擦**（库仑摩擦和粘滞摩擦）在低速时导致稳态误差。简单的前馈摩擦项可提高跟踪精度。更高级的摩擦模型（Stribeck 模型）用于高性能运动控制。

对于轨迹跟踪，还可加入基于期望加速度的前馈项：

```
tau_ff = M(q) * q_ddot + C(q, q_dot) + G(q)
```

其中 M 是惯性矩阵，C 是科里奥利/离心力项，G 是重力项。这是计算力矩控制律。
