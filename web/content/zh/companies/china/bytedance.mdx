# 字节跳动（ByteDance）

字节跳动有限公司（ByteDance Ltd.）是一家中国互联网科技公司，总部位于北京市海淀区，2012年3月由**张一鸣**和**梁汝波**联合创立。公司运营着全球下载量最大的社交媒体应用——国际版**TikTok**和中国版**抖音**，并拥有新闻聚合平台**今日头条**和短视频剪辑工具**剪映（CapCut）**等产品组合。截至2024年底，字节跳动估值约**3000亿美元**，是全球最有价值的私营公司之一。

字节跳动采用在开曼群岛注册的"可变利益实体"（VIE）架构引入外资，但运营实体北京字节跳动科技有限公司受中国监管，并设有1%的政府**"黄金股"**（通过中国文化产业投资基金持有）。

字节跳动核心竞争力是**AI驱动的内容推荐算法**——实时学习用户偏好并推送个性化视频流。该算法最初用于今日头条的新闻聚合，后被应用于短视频，成为抖音和TikTok病毒式增长的技术引擎。

## 发展时间轴

### 2012–2015年：创立与今日头条崛起

- **2012年3月**：字节跳动在北京创立，由**张一鸣**（CEO兼董事长）和**梁汝波**（现任CEO）与来自前项目"九九房"的联合创始人共同创办。张一鸣的愿景："打造信息创造和分发的基础设施"，通过AI推荐而非社交图谱或人工编辑来分发内容。
- **2012年8月**：上线**今日头条**——一款AI驱动的新闻与内容聚合应用，通过算法学习用户偏好，推送个性化新闻流。与传统新闻应用不同，今日头条没有编辑——算法决定用户看到什么。3年内，今日头条月活用户突破1亿。
- **2012年**：上线**内涵段子**——幽默内容平台，包含笑话、表情包和短视频。（后来改名并逐步整合到其他字节跳动产品中。）
- **2014年**：字节跳动推出**BuzzHandler**和**TopBuzz**——面向美国和日本市场的国际新闻聚合应用。
- **2015年**：今日头条月活用户达到**5.3亿**——成为中国使用量最大的新闻应用。
- **2015年**：字节跳动开始国际化扩张，在日本、东南亚和美国设立办事处。

### 2016年：抖音上线与AI Lab

- **2016年9月**：在中国上线**抖音**——一款带有音乐、滤镜和AI推荐功能的短视频平台。抖音比阿里巴巴的优酷和腾讯的微视仅晚六个月上线，但凭借更优的AI算法和病毒式"挑战赛"机制实现了爆发式用户增长。
- **2016年**：成立**字节跳动AI Lab**——内部研发部门，专注于机器学习，自然语言处理和计算机视觉，目标是在所有产品中推进推荐算法能力。

### 2017年：国际化扩张与Musical.ly收购

- **2017年**：字节跳动在国际市场上线**TikTok**（初期面向中国市场以外），功能模式参照抖音但作为独立的国际产品。
- **2017年2月**：收购**Flipagram**——一款美国短视频应用，拥有大量美国用户基础，交易金额约5000万美元。这次收购让字节跳动在美国市场建立了桥头堡。
- **2017年11月**：以约**10亿美元**收购**Musical.ly**——一款在欧美青少年中非常流行的中国短视频应用，拥有美国和欧洲6000万用户。这次收购让字节跳动直接获得了西方市场庞大的青少年用户群。

### 2018年：TikTok与Musical.ly合并

- **2018年8月2日**：字节跳动将**Musical.ly合并入TikTok**，关闭Musical.ly品牌并将所有用户迁移到TikTok。合并后的实体继承了Musical.ly的青少年用户基础和TikTok的全球野心。这是一个关键时刻——TikTok迅速成为全球下载量最大的应用之一。
- **2017年11月–2018年**：收购**News Republic**——全球新闻聚合平台，交易金额约1亿美元，将其编辑团队整合到TikTok内容运营中。
- **2018年**：字节跳动在软银愿景基金（SoftBank Vision Fund）领投的一轮融资中获得30亿美元，估值达到**750亿美元**，成为当时全球估值最高的独角兽公司。

### 2019–2020年：快速增长与印度禁令

- **2019年**：上线**Lark**（后更名为**飞书/Feishu**）——企业协作与办公套件，与阿里巴巴的钉钉和腾讯的企业微信竞争。飞书在后新冠时期因远程办公需求激增而快速发展。
- **2020年**：上线**剪映（CapCut）**——免费AI驱动视频剪辑应用，成为全球最受欢迎的视频剪辑工具之一（尤其对于TikTok内容创作者）。
- **2020年6月29日**：**印度永久封禁TikTok和CapCut**及其他数十款中国应用，以国家安全为由——这是对字节跳动最大的监管打击之一。彼时印度是TikTok最大的用户市场（2亿+用户）。字节跳动后来在2023年将印度业务出售给Glance。
- **2020年**：上线**TikTok Shop**——TikTok内置电商功能，最初在印尼上线，随后扩展至英国、美国等市场。

### 2021–2022年：CEO交接与游戏布局

- **2021年3月**：**张一鸣卸任CEO**，转任董事长；**梁汝波**（联合创始人）接任CEO。张一鸣表示需要"新的管理层"来管理字节跳动这一在全球60多个国家和地区运营的跨国组织。
- **2021年**：收购**沐瞳科技（Moonton）**——东南亚热门移动MOBA游戏**《无尽对决》（Mobile Legends: Bang Bang）**的开发商，交易金额约40亿美元。（该收购后来面临监管审查。）
- **2021年8月**：收购**Pico**——中国领先的VR（虚拟现实）初创企业，交易金额约7亿至10亿美元。此举让字节跳动进入元宇宙/VR领域，与Meta的战略相似。
- **2021年**：创建**BytePlus**事业部——向企业客户提供字节跳动AI技术（包括推荐算法）作为B2B SaaS产品的新业务部门。

### 2023年：AI大模型与豆包

- **2023年**：上线**豆包**——字节跳动的AI聊天机器人和虚拟助手，与百度文心一言等中国大语言模型竞争。豆包基于字节跳动的**Seed AI**模型系列开发。
- **2023年**：成立**Seed AI**团队——字节跳动的大型语言模型（LLM）研究部门。该团队开发了豆包聊天机器人和底层语言模型，字节跳动开始将其整合到整个产品矩阵中。
- **2023年**：将Pico出售给TikTok VR部门，进行内部重组。向西方市场推广**小红书（Lemon8）**（生活方式图片分享应用），定位为"Pinterest的替代品"。
- **2023年**：收购**Music.com**域名——显示在音乐流媒体领域的布局意向。

### 2024–2025年：美国立法与OmniHuman

- **2024年**：字节跳动面临美国新的立法压力，要求其**出售TikTok**或面临封禁。2024年4月拜登总统签署的**《保护美国人免受外国对手控制应用程序侵害法案》（Protecting Americans from Foreign Adversary Controlled Applications Act）**给了字节跳动至2025年1月的最后期限来出售TikTok美国业务，否则面临封禁。该法律在法院受到挑战，截止日期被延长。
- **2025年**：展示了**OmniHuman-1**——一款可以从单张图片生成全身人类动作视频的AI系统，代表了字节跳动在生成式AI视频领域的持续投入。
- **2025年**：截至2025年，美国要求字节跳动剥离TikTok的立法仍是一个悬而未决的法律和政治问题，多起法庭挑战和立法尝试不断出现。TikTok在美国的运营（拥有约1.7亿美国用户）仍处于不确定状态。
- **2025年**：有报道称字节跳动正在探索非TikTok业务（包括抖音、今日头条和企业产品）在美国上市的可能性，以应对TikTok在美面临的压力。

## 核心产品与服务

### 社交媒体
- **TikTok**——国际版短视频平台（中国除外）；全球15亿+用户
- **抖音**——中国国内短视频平台；集成电商、直播、本地生活服务；月活约7亿+
- **TikTok Shop**——应用内电商（美国、英国、印尼等）

### 内容平台
- **今日头条**——AI驱动新闻聚合；中国日活1.2亿+
- **TopBuzz / BuzzVideo**——国际内容视频平台（部分市场已停止运营）
- **News Republic**——授权新闻内容（现已大部分整合）

### 生产力/企业
- **飞书/Lark**——企业协作套件（聊天、文档、日历、视频会议）；与钉钉、企业微信竞争
- **飞书**——Lark的中国国内品牌名

### AI与工具
- **剪映（CapCut）**——AI驱动视频编辑器；月活2亿+；TikTok创作者的必备工具
- **豆包**——基于Seed AI大语言模型的AI聊天机器人和虚拟助手
- **小红书（Lemon8）**——生活方式图片分享应用，2023年扩展至西方市场
- **TikTok AI**——TikTok内置的生成式AI功能（AI虚拟形象、配音等）

### 云与B2B
- **火山引擎（Volcano Engine）**——云服务器和数据智能平台，2021年上线；向企业客户提供字节跳动的AI和推荐技术
- **BytePlus**——B2B SaaS平台，向第三方开发者提供推荐算法和计算机视觉工具

## 所有权与投资者

截至2024年底，字节跳动主要投资者包括：
- **杰夫·雅斯（Jeff Yass）的萨班路国际集团（SIG）**——通过多只基金持有约40%股份
- **红杉资本（Sequoia Capital）**——早期投资者
- **软银集团（SoftBank Group）**——2018年以750亿美元估值投资30亿美元
- **KKR**——基础设施和增长型股权
- **泛大西洋投资（General Atlantic）**
- **高瓴资本（Hillhouse Capital）**
- **春华资本（Primavera Capital）**
- **中国政府**——通过中国文化产业投资基金持有北京字节跳动科技有限公司1%的"黄金股"

张一鸣是最大个人股东。公司注册在**开曼群岛**，以VIE架构持有北京字节跳动科技有限公司。
