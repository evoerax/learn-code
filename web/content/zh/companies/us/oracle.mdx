# 甲骨文（Oracle）

**甲骨文公司**（NYSE: ORCL）是全球最大的企业软件公司，专注于关系型数据库管理系统（RDBMS）、云基础设施和企业应用软件。公司由**拉里·埃里森**（Larry Ellison）、**鲍勃·米纳**（Bob Miner）和**埃德·奥茨**（Ed Oates）于**1977年**创立——三人离开一家咨询公司，决心构建商业化关系型数据库。甲骨文是**Oracle Database**的发明者，这是全球使用最广泛的企业数据库系统，运行着世界上绝大多数商业数据。公司总部位于德克萨斯州奥斯汀（在2020年从加州红木海岸迁出），业务已从数据库扩展为与微软Azure、亚马逊AWS和谷歌云竞争的全面云计算提供商。公司服务180个国家的400,000+客户，包括全部财富100强。2025财年营收为**574亿美元**，**净利润124.4亿美元**，约**164,000名员工**。

## 发展时间轴

### 创立与关系型数据库愿景（1977–1983）

**拉里·埃里森**、**鲍勃·米纳**和**埃德·奥茨**于**1977年**离开Ampex公司，在加州圣克拉拉创立了**软件开发实验室（SDL）**。公司名称"Oracle"来自他们为CIA工作的一个项目名称。

创始团队阅读了IBM研究员**埃德加·科德**（Edgar F. Codd）**1976年的论文**，该论文描述了**关系型数据库管理系统（RDBMS）**的理论。科德提出数据应存储在具有行和列的**表格（关系）**中，查询应基于数学集合论而非导航式记录访问——这与当时占主导地位的分层和网络数据库系统形成鲜明对比。

埃里森立即意识到科德的关系模型是数据库的未来。SDL的第一款产品是基于科德论文的**Oracle RDBMS**——最早的商品化关系型数据库实现之一。IBM用了三年时间才推出自己的RDBMS（DB2于1983年发布），此时甲骨文已建立了强大的客户群。

公司经历多次更名：

- **1977–1979**：软件开发实验室（SDL）
- **1979–1983**：关系软件公司（RSI）
- **1983–1995**：Oracle系统公司
- **1995至今**：甲骨文公司

**1986年**，甲骨文在NASDAQ上市，融资5200万美元，市值约2.7亿美元，首日股价上涨300%。

### 增长与PeopleSoft恶意收购（1984–2004）

1990年代甲骨文快速增长，但**1990年**面临严重危机——快速扩张超过了财务控制能力，被迫重报盈利，股价一天内下跌50%。这导致公司引入专业管理和重组。

**2003年**，甲骨文对PeopleSoft发起恶意收购要约，出价51亿美元。经18个月的法律和代理权争夺，最终以**103亿美元**于**2004年12月**完成收购——这是甲骨文当时最大的收购，将甲骨文从纯数据库公司转型为企业应用公司。

### CEO交接与萨芙拉·卡茨时代（2005–2019）

**2005年**，埃里森卸任CEO，由**萨芙拉·卡茨**（总裁）和**马克·赫德**（CEO）接任。两人都来自惠普。赫德以严格的运营效率著称，专注于每股收益。**2019年10月马克·赫德突然去世**，埃里森重新更加活跃地参与公司事务。

### Sun Microsystems收购：改变甲骨文（2010）

**2010年1月**，甲骨文以**74亿美元**收购**Sun Microsystems**，这是甲骨文历史上最变革性的收购。Sun的产品包括：

- **SPARC处理器**和**Solaris** Unix操作系统
- **Java**（1995年Sun收购）——全球使用最广泛的编程语言
- **MySQL**——全球最流行的开源数据库（Sun于2008年以10亿美元收购）
- **存储系统**（Sun StorageTek）

收购后甲骨文对Sun多个产品进行调整或停产，MySQL社区分叉出**MariaDB**，成为与Oracle MySQL持续竞争的分支。

### 云转型与AI基础设施（2012–2025）

甲骨文在云服务领域落后于AWS、Azure和谷歌云。**2016年**推出真正的云产品**Oracle Cloud**，以自有的**Oracle Database**和应用程序在甲骨文云上原生运行作为差异化优势。

关键云产品：

- **Oracle Cloud Infrastructure（OCI）**：IaaS——虚拟机、存储、网络
- **Oracle Autonomous Database**：全球首个自动驾驶、自修复数据库
- **OCI Supercluster**：用于AI训练的大规模GPU集群

**2024年**，甲骨文宣布部署** Nvidia GB200 NVL72机架**——由72个Blackwell GPU通过NVLink互联的大规模GPU集群，用于AI训练。与**xAI**（马斯克的AI公司）、Cohesity等公司签署了在其OCI上运行AI工作负载的协议。

### Cerner收购与医疗健康（2022）

**2022年6月**，甲骨文以**283亿美元**完成对**Cerner Corporation**的收购——这是医疗技术史上最大的收购之一。Cerner是美国第二大电子健康记录（EHR）公司（仅次于Epic Systems），系统用于全球35,000+医疗机构。

**2023年**，甲骨文对Cerner进行约**8.85亿美元**的减值处理，原因是整合挑战和云迁移慢于预期。新合并的公司更名为**Oracle Health**。

## 产品与业务板块

### Oracle数据库与技术

甲骨文的核心产品：

- **Oracle Database**（21c, 23ai）：全球最先进的RDBMS，集成JSON/XML支持、内存处理、区块链表和ML能力
- **MySQL**：全球最流行的开源RDBMS
- **Oracle Exadata**：为Oracle数据库性能优化的软硬件集成系统
- **Oracle Autonomous Database**：自动驾驶、自修复的云数据库服务

### 云基础设施（OCI）

- **OCI Compute**：裸金属服务器、VM、GPU实例（A100、H100、GB200）
- **OCI Superclusters**：用于AI训练的大规模GPU集群

### 企业应用（Fusion Cloud SaaS）

- **Oracle Fusion Cloud ERP**：财务、采购、项目管理、供应链管理
- **Oracle Fusion Cloud HCM**：人力资源管理、薪资、人才管理
- **Oracle Fusion Cloud CX**：销售、营销、服务、商务
- **Oracle Fusion Cloud SCM**：供应链计划、制造、物流

### Oracle Health（Cerner）

- **Oracle Health EHR**：面向医院和诊所的电子健康记录系统
- **Oracle Health Data Platform**：AI就绪的健康数据平台

## 主要管理层

**拉里·埃里森**（联合创始人，2025年80岁）自2005年卸任CEO以来一直担任董事长兼首席技术官。他持有甲骨文约**40%的股份**，是世界最富有人士之一。

**萨芙拉·卡茨**（总裁）自2019年以来实际上担任首席运营官。**大卫·埃里森**（执行副总裁、产品开发）是埃里森的儿子，被广泛认为是未来潜在的接班人。
