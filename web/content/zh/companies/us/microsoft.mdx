# Microsoft（微软公司）

Microsoft Corporation（NASDAQ: MSFT）是一家美国科技公司，1975年创立，总部位于华盛顿州雷德蒙德。是全球收入最大的软件公司，市值约3万亿美元（2025年）。由**比尔·盖茨**（Bill Gates）和**保罗·艾伦**（Paul Allen）创立，通过Windows操作系统和Microsoft Office办公套件称霸个人电脑时代。在CEO**萨蒂亚·纳德拉**（Satya Nadella，2014年起）的领导下，公司完成了从软件授权商到云计算巨头的历史性转型，**Microsoft Azure**成为全球第二大云平台（仅次于AWS）。2023年以**687亿美元**收购**动视暴雪**成为游戏史上最大收购。主要业务板块包括：智能云（Azure、服务器产品）、生产力（Microsoft 365、LinkedIn）、个性化计算（Windows、Xbox、Surface、Bing）。

## 时间线

### 1975–1984年：创立与IBM合同

- **1975年4月4日**：微软由**比尔·盖茨**（19岁，哈佛辍学生）和**保罗·艾伦**（22岁，MIT毕业生）在新墨西哥州阿尔伯克基创立——最初名为"Micro-Soft"（微型软件）。他们的第一个产品是**Altair BASIC**解释器——为Altair 8800（第一台个人电脑）编写，用了仅仅8周时间。
- **1980年8月12日**：微软与**IBM**签署合同，为即将推出的IBM PC提供操作系统——**MS-DOS**（从西雅图电脑产品公司收购QDOS，更名而来）。IBM的PC成为行业标准，MS-DOS成为主导的PC操作系统。
- **1981年4月25日**：微软正式注册为**Microsoft Corporation**。
- **1985年11月20日**：发布首个零售版**Microsoft Windows 1.0**——在MS-DOS上运行的图形操作系统环境。
- **1986年3月13日**：微软在**NASDAQ上市**，发行价每股21美元，融资6100万美元。盖茨保留约45%股权。
- **1986年**：微软迁至华盛顿州雷德蒙德至今。

### 1985–1994年：Windows与Office帝国

- **1990年**：发布**Windows 3.0**——大规模商业成功，使Windows成为主导的PC GUI。同时发布**Microsoft Office**（最初为Macintosh，后扩展至Windows）。
- **1990年**：微软收入首次突破**10亿美元**。
- **1993年**：发布**Windows NT**——面向企业级网络的操作系统，最终与消费级Windows合并（2001年Windows XP）。
- **1994年**：美国**司法部**对微软提起反垄断诉讼——指控IE浏览器与Windows捆绑销售存在反竞争行为。

### 1995–2000年：互联网备忘录与鲍尔默时代

- **1995年5月26日**：比尔·盖茨发布著名的**"互联网海啸"备忘录**——宣告互联网是最高优先级，一夜之间改变了公司战略。
- **1995年8月24日**：发布**Windows 95**——4天内售出100万份，包含开始按钮和任务栏。Windows 95奠定了微软OS主导地位。
- **1997年**：比尔·盖茨提拔**史蒂夫·鲍尔默**（1980年加入）为总裁，成为实际上的一把手。
- **2000年1月13日**：**史蒂夫·鲍尔默**接替比尔·盖茨成为CEO。盖茨保留董事长职位。

### 2001–2007年：Xbox、Windows XP与Vista

- **2001年10月25日**：发布**Windows XP**——微软最长寿命的操作系统之一（主流支持至2009年）。
- **2001年11月15日**：发布首款**Xbox**游戏主机——进入300亿美元游戏市场。Xbox以亏损定价（每台约100美元）销售游戏机，以获得客厅的一席之地。
- **2002年**：发布**Xbox Live**——首个主要游戏机在线游戏网络。
- **2005年**：发布**Xbox 360**——重大升级，售出约8400万台。
- **2007年1月30日**：发布**Windows Vista**——Vista广受批评，被视为微软较不成功的OS版本之一。

### 2008–2014年：云优先、移动失误

- **2008年10月27日**：发布**Windows Azure**（后更名为**Microsoft Azure**）——微软进入云计算领域。Azure最初与2002年已主导市场的亚马逊AWS竞争。
- **2009年10月22日**：发布**Windows 7**——Vista的直接继承者，解决了性能和可用性问题，成为未来十年主导的企业操作系统。
- **2011年**：以约**85亿美元**收购**Skype**——当时微软最大收购，但整合效果不佳。
- **2013年9月3日**：宣布以约**72亿美元**收购诺基亚手机业务——最终大幅减记（2015年约76亿美元减值）——这是微软最不成功的收购之一。

### 2014–2017年：纳德拉时代——云与开源

- **2014年2月4日**：**萨蒂亚·纳德拉**成为CEO——接替史蒂夫·鲍尔默。纳德拉提出三大战略支柱：移动优先/云优先、拥抱开源、建立富有同情心的AI。
- **2014年9月**：以**25亿美元**收购**Mojang**（《我的世界》）。
- **2015年7月29日**：发布**Windows 10**——"一个Windows适配所有设备"，以免费升级策略加速普及。
- **2016年6月13日**：以**262亿美元**收购**LinkedIn**——微软第二大收购，LinkedIn以独立子公司运营。LinkedIn已增长至超10亿会员，年收入约150亿美元。

### 2018–2021年：GitHub、OpenAI与游戏帝国

- **2018年6月4日**：以**75亿美元**收购**GitHub**——全球最大软件开发平台（2500万开发者）。
- **2019年**：向**OpenAI**投资**10亿美元**——获得将OpenAI技术商业化的独家权利。
- **2020年**：发布**Microsoft Teams**——Slack的竞争对手，COVID-19期间爆发式增长，到2023年月活超3亿。
- **2020年9月**：同意以**75亿美元**收购**ZeniMax Media**（贝塞斯达母公司），2021年3月完成。
- **2021年10月5日**：发布**Windows 11**——时隔多年重大OS版本更新。

### 2022–2023年：动视暴雪与AI整合

- **2022年1月18日**：宣布以**687亿美元**收购**动视暴雪**——游戏史上最大收购和微软史上最大收购。
- **2023年1月**：向OpenAI追加投资约**100亿美元**——深化合作伙伴关系，Azure获得运行OpenAI模型的独家云计算权。
- **2023年2月**：发布**Microsoft 365 Copilot**——集成到Word、Excel、PowerPoint、Outlook和Teams的AI助手。
- **2023年10月13日**：完成**动视暴雪**收购——英国CMA批准后完成交易。微软成为全球第三大游戏公司（按收入）。
- **2023年11月**：发布自研**AI芯片**——**Maia 100** AI加速器和**Cobalt 100** CPU。

### 2024–2025年：AI基础设施与Copilot无处不在

- **2024年**：**GitHub Copilot**达到**100万付费用户**。
- **2024年5月**：微软市值短暂超越**3万亿美元**。
- **2025年**：发布**Copilot+ PCs**——配备专用NPU（神经处理单元）的新型Windows PC类别。
- **2025年**：微软承诺到2025年投入**350亿美元**用于AI基础设施。

## 主要产品与业务板块

### 智能云
- **Microsoft Azure**——全球第二大云平台（约24%市场份额），约200个数据中心区域。
- **Azure AI**——包括OpenAI服务、Azure机器学习、认知服务。
- **GitHub Enterprise**、**Visual Studio**——开发者工具。

### 生产力和业务流程
- **Microsoft 365**——全球主导生产力套件（Word、Excel、PowerPoint、Outlook、Teams），2024年商业用户超4亿。
- **LinkedIn**——专业社交网络，超10亿会员，年收入约150亿美元。
- **Dynamics 365**——ERP和CRM套件。

### 个性化计算
- **Windows**——仍占桌面操作系统约72%市场份额。
- **Surface**——微软自有硬件线（Surface Pro、Surface Laptop、Surface Studio）。
- **Xbox**——游戏主机（Xbox Series X/S），Game Pass Ultimate超2500万订阅用户。
- **Bing**——现与AI（Copilot）集成，全球搜索市场份额约3%。

## 领导层

- **CEO**：**萨蒂亚·纳德拉**（Satya Nadella，2014年2月4日起）——此前领导Azure和服务器与工具业务。
- **创始人**：**比尔·盖茨**——1975-2000年担任CEO兼董事长，2000年后担任"首席软件架构师"，2006年退出一线，2020年卸任董事长。
- **CFO**：**艾米·胡德**（Amy Hood，2013年起）——财富100强公司中为数不多的女性CFO。

## 财务摘要

- **2025财年（截至2025年6月）收入**：约2817亿美元
- **2025财年净利润**：约1018亿美元
- **2025财年营业收入**：约1285亿美元
- **总资产**：约6190亿美元（2025年）
- **员工人数**：约22.8万人（2025年）
- **市值**：约3万亿美元（2025年）

## 制造

微软主要是**软件和服务公司**。Xbox游戏机和Surface设备由合作伙伴（Flex、Celestica等）在中国和其他亚洲国家制造。微软的云基础设施使用Dell、HP Enterprise和微软自己的服务器，配备Intel、AMD、NVIDIA处理器以及自研Maia和Cobalt芯片。
