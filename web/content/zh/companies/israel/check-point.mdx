# Check Point（检查点软件公司）

Check Point软件技术有限公司（Check Point Software Technologies Ltd.）是全球网络安全领域的领导者，总部位于以色列特拉维夫，美国总部位于加州圣卡洛斯。Check Point由**Gil Shwed**于1993年创立（当时年仅25岁），开创了现代防火墙和网络安全行业。Shwed在希伯来大学就读期间就开发了Check Point的**状态检测**（Stateful Inspection）技术——该技术成为现代企业网络安全的基石。Shwed截至2024年仍担任CEO（使他成为全球任期最长的科技CEO之一）。Check Point于1996年上市，成长为全球最大的专业网络安全公司之一，2024年营收约24亿美元，在全球拥有超过7,000名员工。公司的产品涵盖防火墙、威胁检测、端点安全、云安全和移动安全——保护着数十万企业免受网络攻击。

## 发展时间轴

### 1993–1999年：创立与防火墙革命

- **1993年**：**Gil Shwed**、Marius Nacht（联合创始人）和Shlomo Kremer（第三位联合创始人，早期离开）在以色列特拉维夫创立了**Check Point软件技术有限公司**。Shwed当时年仅25岁，在耶路撒冷希伯来大学攻读计算机科学期间开发了网络安全概念。公司的首款产品旨在解决前互联网时代的企业网络安全问题。
- **1994年**：Check Point发布**Firewall-1**——首批商业化状态检测防火墙之一。关键创新是Shwed开发的**状态检测**技术——不仅检查单个网络数据包，还检查网络连接的状态——只允许授权流量中属于已建立合法会话的数据包。这比简单的包过滤防火墙有显著改进，成为企业网络安全的行业标准。Firewall-1还是首批提供防火墙管理**图形用户界面（GUI）**的产品之一——这是重大的可用性创新。
- **1994年**：Firewall-1赢得**COMDEX最佳展示奖**——Check Point获得的第一个重大行业认可。
- **1996年**：Check Point在**NASDAQ**上市（股票代码：CHKP）——融资约4,700万美元。IPO时公司估值约1.7亿美元。强劲的公开市场反应验证了网络安全日益增长的重要性。
- **1997年**：Check Point发布**Check Point VPN-1**——将防火墙和虚拟专用网络（VPN）功能结合在单一产品中。随着企业开始接入互联网，VPN-1成为安全远程访问的必备产品。VPN-1是首款将状态检测防火墙与站点间VPN加密相结合的产品。

### 1999–2006年：互联网泡沫、峰值与诺基亚合作

- **1999年**：在网络泡沫高峰期，Check Point股价达到每股**239美元**——使公司市值约**200亿美元**。当时，这使Check Point成为全球最有价值的技术公司之一（按营收计算），说明了投资者对网络安全的重视程度。
- **2000年**：网络泡沫崩溃使Check Point股价从239美元跌至约**20美元**（跌幅90%）。许多网络泡沫时代的公司都经历了类似的崩盘，但Check Point的下跌尤为严重，因为其估值已经过高。
- **2000年**：泡沫破灭后，Check Point重组并重新聚焦盈利能力。公司从纯增长故事转向可持续的商业模式。这一时期塑造了Check Point保守的财务文化——公司历史上一直保持高利润率，避免大额债务。
- **2003年**：Check Point发布**Check Point NG（下一代）**——将应用层网关功能和基于身份的安引入防火墙。这将Check Point的技术从纯网络边界安全扩展到更细粒度的安全控制。
- **2007年**：Check Point与**诺基亚**建立重大**分销合作关系**——这家芬兰电信巨头当时是网络设备的主要厂商。诺基亚的网络安全设备运行Check Point软件，该合作将Check Point的业务扩展到基于硬件的安全解决方案。诺基亚的安全设备业务后来被出售给私募股权公司。
- **2007年**：Check Point发布**ZoneAlarm**——面向Windows PC的消费者防火墙和安全产品。ZoneAlarm成为最受欢迎的消费者安全产品之一，与诺顿/赛门铁克在消费者市场展开竞争。

### 2008–2016年：收购与3D平台战略

- **2009年**：Check Point收购**诺基亚的安全设备业务**——将硬件安全设备纳入内部，并深化与诺基亚的合作关系。
- **2011年**：Check Point收购**FaceTime Communications**（约7,000万美元）——一家为实时通信和社交媒体应用提供安全保护的厂商。这使Check Point获得了对Web 2.0安全威胁（社交媒体、即时通讯、视频会议）的可见性。
- **2012年**：Check Point收购**Simplicity Financial Technologies**（约2,000万美元）——一家专注于金融科技安全的以色列初创公司。这次收购反映了Check Point建立垂直特定安全能力的战略。
- **2013年**：Check Point发布**Check Point R75**——首个整合高级威胁防护和反僵尸网络技术的重大版本，应对日益普遍的APT（高级持续性威胁）攻击。
- **2014年**：Check Point推出**Check Point ThreatCloud**——一个协作式威胁情报网络，从全球传感器汇总数据，实时识别新兴威胁。这是首批协调式威胁情报平台之一。
- **2015年**：Check Point发布**Check Point 15000系列**——为大型数据中心和服务提供商设计的新一代高端安全设备系列。
- **2016年**：Check Point推出**Check Point Infinity架构**——统一保护网络、端点、移动和云的综合安全架构。

### 2017–2020年：收购与SecureAnywhere转型

- **2018年**：Check Point收购**Dome9**（约1.75亿美元）——一家云安全态势管理（CSPM）公司，为多云环境（AWS、Azure、谷歌云）提供安全保障。这次收购使Check Point进入了快速增长的云安全市场。
- **2018年**：Check Point收购**ForceNock**（约3,000万美元）——一家专注于端点检测和响应（EDR）的以色列初创公司。这为Check Point的端点安全组合增添了EDR能力。
- **2019年**：Check Point收购**Cymplify**（约3,000万美元）——一家专注于物联网（IoT）设备和嵌入式系统安全的以色列初创公司。
- **2019年**：Check Point收购**Hadar**（约3,000万美元）——一家专注于云安全分析的初创公司。该时期总收购金额接近2亿美元。
- **2019年**：Check Point发布**CloudGuard**——全面的云安全平台，覆盖IaaS、SaaS和容器环境。CloudGuard是传统网络安 全厂商推出的首批主要云安全平台之一。
- **2020年**：新冠疫情迫使企业迅速采用远程办公。Check Point的**Check Point远程访问VPN**解决方案需求大幅增长，因为各组织需要保护分布式员工队伍。云向的转变也加速了Check Point的云安全业务。

### 2021–至今：Infinity平台与AI安全

- **2021年**：Check Point推出**Check Point Horizon**——结合威胁防护、事件响应和安全培训的安全运营平台。Horizon面向托管安全服务提供商（MSSP）和企业安全运营中心（SOC）。
- **2021年**：Check Point发布**Quantum**——基于新硬件平台（Quantum Titan芯片）和软件架构的新一代防火墙系列。Quantum防火墙旨在处理5G网络和AI驱动应用的海量流量。
- **2022年**：Check Point推出**Check Point Quantum Smart Network**——AI驱动的网络安全平台，使用机器学习进行实时威胁检测和自动策略执行。
- **2023年**：Check Point推出**Infinity平台**——综合AI驱动的安全架构，在单一管理控制台下统一网络、云、端点、物联网和移动安全。Infinity平台使用AI/ML进行威胁检测，并包括自动威胁防护能力。
- **2023年**：Check Point推出**Check Point AI Na**——AI驱动的威胁检测系统，使用在Check Point全球威胁情报数据上训练的机器学习模型，实时识别新型攻击。
- **2024年**：Check Point报告**2024财年营收约24亿美元**，拥有强劲的自由现金流。公司为全球超过10万个组织提供服务，包括大多数财富500强企业。
- **2024年**：Check Point的威胁情报网络（ThreatCloud）每天处理全球传感器超过**30亿次安全事件**。
- **2025年**：Check Point继续投资AI驱动的安全，其AI驱动的威胁防护成为对抗传统竞争对手（Palo Alto Networks、Fortinet）和新型AI原生安全初创公司的关键差异化优势。

## 核心产品与业务板块

### 网络安全
- **Check Point Quantum**——用于边界和数据中心安全的新一代防火墙（NGFW），配备用于5G和AI工作负载的Quantum Titan芯片
- **Check Point Quantum安全网关**——用于企业边界安全的硬件和虚拟设备
- **Check Point ThreatCloud**——协作式AI驱动威胁情报网络，从全球传感器汇总数据

### 云安全
- **Check Point CloudGuard**——覆盖IaaS（AWS、Azure、GCP）、SaaS和容器环境的全面云安全平台
- **CloudGuard姿态管理（CSPM）**——来自Dome9收购；自动化云安全合规和错误配置检测
- **CloudGuard工作负载保护**——容器和Kubernetes环境的安全

### 端点和移动安全
- **Check Point端点安全**——端点保护平台（EPP）和端点检测与响应（EDR）
- **Check Point Harmony**——集成的端点、移动和远程访问安全
- **Harmony Mobile**——面向iOS和Android的移动威胁防御

### 威胁防护
- **Check Point Infinity架构**——跨网络、云、端点、物联网和移动的统一安全
- **反僵尸网络、反勒索软件**——高级威胁防护技术
- **入侵防御系统（IPS）**——基于网络的威胁检测和阻止
