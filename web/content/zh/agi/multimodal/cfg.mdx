# 无分类器引导

在无条件模型上实现条件生成。

---

## 训练

```python
def train():
    # 随机丢弃条件
    if random() < cfg_prob:
        cond = None  # 无条件
    else:
        cond = text_embed
    
    output = model(noise, cond)
```

---

## 推理

```python
def cfg_sample(model, noise, cond_emb, guidance_scale=7.5):
    # 条件和无条件预测
    cond_out = model(noise, cond_emb)
    uncond_out = model(noise, None)
    
    # 引导
    out = uncond_out + guidance_scale * (cond_out - uncond_out)
    return out
```

---

## 典型引导强度

| cfg_scale | 效果 |
|-----------|------|
| 1.0 | 无引导，模糊 |
| 3.0 | 轻度引导 |
| 7.5 | Stable Diffusion 标准 |
| 10-15 | 强引导，可能过饱和 |
| 20+ | 极端，动漫化 |

---

## 原理推导

```
无 CFG:
  noise = f(x_t, t | y) ≈ ε(y) + small_correction

有 CFG (高 scale):
  noise = ε(no_y) + 7.5 * (ε(y) - ε(no_y))
        = ε(no_y) + 7.5*ε(y) - 7.5*ε(no_y)
        = -6.5*ε(no_y) + 7.5*ε(y)

模型被"强制"产生更多条件输出。
```

## 计算量

条件和无条件推理需要两次前向传播。

可以用 classifier guidance 替代计算：
```python
# 单次前向 + 分类器梯度
cond_out + w * classifier_gradient
```
