# ViT Vision Transformer

将 Transformer 直接应用于图像块。

---

## 架构

```python
class ViT(nn.Module):
    def __init__(self, image_size=224, patch_size=16, dim=768, num_heads=12):
        super().__init__()
        self.patch_embed = nn.Conv2d(3, dim, patch_size, patch_size)
        self.cls_token = nn.Parameter(torch.randn(1, 1, dim))
        self.pos_embed = nn.Parameter(torch.randn(1, num_patches + 1, dim))
        self.blocks = nn.ModuleList([TransformerBlock(dim, num_heads) ...])
        self.head = nn.Linear(dim, num_classes)
```

---

## 分块处理

```
图像 (224x224) → 16x16 块 × 196 → 768 维向量
```

## vs CNN

| 特性 | ViT | CNN |
|------|-----|-----|
| 归纳偏置 | 低 | 高 |
| 数据需求 | 大 | 小 |
| 计算量 | 高 | 低 |
