# DINO 自蒸馏

通过动量教师实现无监督 ViT。

---

## 架构

```python
class DINO(nn.Module):
    def __init__(self, student, teacher):
        self.student = student
        self.teacher = teacher
    
    def forward(self, student_out, teacher_out):
        # 学生: 标准 softmax
        student_prob = F.softmax(student_out / T_s, dim=-1)
        # 教师: 带 centering 的 softmax
        teacher_prob = F.softmax((teacher_out - self.center) / T_t, dim=-1)
        # 交叉熵蒸馏
        loss = -torch.sum(teacher_prob * torch.log(student_prob), dim=-1)
        return loss.mean()
```

---

## 动量更新

```python
def momentum_update(self, m=0.999):
    for ps, pt in zip(self.student.parameters(), self.teacher.parameters()):
        pt.data = m * pt.data + (1 - m) * ps.data
```
