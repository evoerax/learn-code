# 修正流

Rectified Flow (Liu et al., 2022) 优化流量以遵循直线轨迹。

---

## 曲线路径问题

```
标准流: noise → data 通过曲线路径
        曲线: 复杂, 难以采样

修正流: 投影到直线
        直线: 直接, 快速
```

---

## 修正算法

```python
def rectify_flow(flow, data_loader, num_steps=1000):
    """
    迭代细化流使其更直
    """
    for iteration in range(num_iterations):
        # 采样配对
        x_0 ~ noise, x_1 ~ data
        
        # 当前轨迹
        t = torch.rand(1)
        x_t = flow.interpolate(x_0, x_1, t)
        v_t = flow.velocity(x_t, t)
        
        # 拟合残差以拉直
        target_v = x_1 - x_0  # 直线速度
        
        loss = mse(v_t, target_v)
        loss.backward()
        optimizer.step()
    
    return flow
```

## 直线采样

```python
def rectified_sample(model, x_T, num_steps=1):
    """
    完全直线流的单步采样
    """
    t = 1.0
    dt = 1.0 / num_steps
    
    x = x_T
    for step in range(num_steps):
        t_current = step * dt
        v = model(x, t_current)
        x = x - v * dt  # 沿直线欧拉积分
    
    return x
```
