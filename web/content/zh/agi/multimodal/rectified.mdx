# 修正流

优化轨迹以加速采样。

---

## 问题

标准 Flow Matching 轨迹可能不是最优：

```python
# 线性插值
x_t = (1-t) * x_0 + t * x_1
# 但中间点可能不合理
```

---

## 修正

class RectifiedFlow(nn.Module):
    def __init__(self, velocity_net):
        self.velocity = velocity_net
    
    def get_velocity(self, x_t, t):
        # 预测速度
        return self.velocity(x_t, t)
    
    def training_loss(self, x_0, x_1):

## 采样

```python
def sample(rf, noise, num_steps=10):
    dt = 1.0 / num_steps
    x = noise
    for i in range(num_steps):
        t = 1.0 - i * dt
        v = rf(x, torch.tensor([t]))
        x = x - v * dt  # 反向积分
    return x
```
