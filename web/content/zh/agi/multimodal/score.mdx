# Score Matching 与 SDE

通过学习 score function 生成样本。

---

## Score Matching

```python
def score_matching_loss(model, x):
    # 学习 log p(x) 的梯度 (score)
    score = model(x)
    
    # Score matching 损失
    # E[||score(x)||^2 + tr(Jac(score)(x))]
    loss = (score ** 2).sum(dim=-1) + jacobian_trace(score, x)
    return loss.mean()
```

---

## SDE 形式

```python
# 连续时间扩散
dx = f(x, t) dt + g(t) dw

# VP SDE (Variance Preserving)
dx = -0.5 * beta(t) * x * dt + sqrt(beta(t)) * dw

# VE SDE (Variance Exploding)
dx = sqrt(d[sigma^2]/dt) * dw
```

---

## 逆向 SDE

```python
# 从噪声恢复数据
dx = [f(x, t) - g(t)^2 * score(x, t)] dt + g(t) * dw
```

---

## 概率流 ODE

```python
# 与 SDE 等价的确定性 ODE
dx = [-0.5 * beta(t) * x - beta(t) * ∇log p(x)] dt

# 使用黑盒 ODE 求解器
# 比随机采样更快
```

## 三者联系

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   DDPM      │────▶│  Score      │────▶│    SDE      │
│  (离散)     │     │  Matching   │     │  (连续)     │
└─────────────┘     └─────────────┘     └─────────────┘
```

- **DDPM**: 离散的噪声预测模型
- **Score Matching**: 学习 score function 的框架
- **SDE**: 连续的随机微分方程 formulation
