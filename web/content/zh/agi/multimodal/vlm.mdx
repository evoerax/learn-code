# VLM 视觉语言模型

连接视觉编码器与语言模型。

---

## 架构

```python
class VLM(nn.Module):
    def __init__(self, vision_encoder, llm, projector):
        self.vision_encoder = vision_encoder
        self.projector = projector  # 线性层或 MLP
        self.llm = llm
    
    def forward(self, images, text):
        # 视觉编码
        vision_embeds = self.vision_encoder(images)
        vision_embeds = self.projector(vision_embeds)
        
        # 文本嵌入
        text_embeds = self.llm.get_input_embeddings()(text)
        
        # 拼接
        combined = torch.cat([vision_embeds, text_embeds], dim=1)
        
        # LLM 生成
        return self.llm(inputs_embeds=combined)
```
