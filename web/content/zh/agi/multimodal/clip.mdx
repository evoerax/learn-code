# CLIP

对比学习连接文本与图像。

---

## 双塔架构

```python
class CLIP(nn.Module):
    def __init__(self, image_encoder, text_encoder):
        self.image_encoder = image_encoder
        self.text_encoder = text_encoder
        self.temperature = nn.Parameter(torch.ones([]) * 0.07)
    
    def forward(self, images, texts):
        image_embeds = self.image_encoder(images)
        text_embeds = self.text_encoder(texts)
        # 对比损失
        logits = (image_embeds @ text_embeds.t()) / self.temperature
        return logits
```

---

## 损失函数

```python
def clip_loss(image_logits, text_logits):
    # 对称交叉熵
    loss_i = cross_entropy(image_logits, torch.arange(len(images)))
    loss_t = cross_entropy(text_logits, torch.arange(len(texts)))
    return (loss_i + loss_t) / 2
```
