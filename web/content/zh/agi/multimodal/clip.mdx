# CLIP

CLIP (Radford et al., 2021) 通过自然语言监督学习视觉概念。

---

## 训练目标

```python
def clip_loss(image_emb, text_emb, temperature):
    """
    image_emb: (B, d) 图像嵌入
    text_emb: (B, d) 文本嵌入
    """
    # 归一化
    image_emb = F.normalize(image_emb, dim=-1)
    text_emb = F.normalize(text_emb, dim=-1)

    # 相似度矩阵
    logits = (image_emb @ text_emb.T) / temperature

    # 对称损失
    labels = torch.arange(len(image_emb)).to(device)
    loss_i = F.cross_entropy(logits, labels)  # 图像 → 文本
    loss_t = F.cross_entropy(logits.T, labels)  # 文本 → 图像

    return (loss_i + loss_t) / 2
```

## 零样本分类

```python
def zero_shot_classify(image, model, class_names):
    # 将类别名编码为提示
    texts = [f"a photo of a {c}" for c in class_names]
    text_emb = model.encode_text(texts)
    image_emb = model.encode_image(image)

    # 余弦相似度
    similarity = (image_emb @ text_emb.T).softmax(dim=-1)

    return similarity
```

---

## 架构

```
图像 → ViT → image_emb → 对比损失 ← text_emb ← 文本 → Transformer
```
