# 扩散模型

扩散模型通过逐步对随机信号去噪来生成数据。

---

## 前向过程（加噪）

```python
def forward_process(x_0, t, beta_schedule):
    """
    在 t 步内对图像添加噪声
    x_0: 干净图像
    t: 时间步（0 = 干净，T = 最大噪声）
    """
    alpha_bar = (1 - beta_schedule[:t]).prod()
    noise = torch.randn_like(x_0)

    x_t = sqrt(alpha_bar) * x_0 + sqrt(1 - alpha_bar) * noise
    return x_t, noise
```

## 反向过程（去噪）

```python
class UNet(nn.Module):
    def forward(self, x_t, t):
        # t 是时间步嵌入
        # 预测噪声或 x_0
        return self.network(x_t, t)

```

## DDPM vs Score Matching

| 方面 | DDPM | Score Matching |
|------|------|----------------|
| **目标** | 预测噪声 ε | 预测分数 ∇log p(x) |
| **损失** | MSE(ε - εθ) | MSE(\|\|∇log p - sθ\|\|²) |
| **采样** | 多步 | SDE 可更少步 |
