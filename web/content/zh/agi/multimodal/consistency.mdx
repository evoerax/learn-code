# 一致性模型

蒸馏扩散模型为单步生成。

---

## 核心思想

学习"一致性函数"，将任意点映射到轨迹终点：

```
F(x) = x_0  对任意 t ∈ [0, T]
```

单步生成: x_T → F(x_T) = x_0

---

## 训练

```python
class ConsistencyModel(nn.Module):
    def __init__(self, unet):
        super().__init__()
        self.net = unet
    
    def forward(self, x, t):
        # 从 (x_t, t) 预测 x_0
        return self.net(x, t)
    
    def consistency_loss(self, x):
        t = torch.randint(0, self.T, (1,))
        noise = torch.randn_like(x)
        
        x_t = self.noise_image(x, t, noise)
        x_0_pred = self(x_t, t)
        
        with torch.no_grad():
            x_0_target = self(x_t, t + 1)  # 来自下一步的目标
        
        return mse(x_0_pred, x_0_target)
```

---

## 蒸馏

```
扩散模型 (多步) → 一致性模型 (单步)
```

---

## 采样

```python
def sample(cm, noise, num_steps=1):
    x = noise
    for i in range(num_steps):
        x = cm(x, torch.tensor([epsilon]))  # epsilon 是起始噪声水平
    return x
```
