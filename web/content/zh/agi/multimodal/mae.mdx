# MAE 掩码自编码器

高比例掩码重建图像。

---

## 掩码策略

```python
def mae_encode(self, x):
    # 随机掩码 75% 的块
    mask = torch.rand(B, num_patches) > 0.75
    x = x * mask.unsqueeze(-1)
    return self.encoder(x)

def mae_decode(self, x):
    return self.decoder(x)
```

---

## 非对称编码器-解码器

```python
class MAE(nn.Module):
    def __init__(self):
        self.encoder = ViT(patch_size=16, dim=768)
        self.decoder = nn.Sequential(
            nn.Linear(768, 512),
            nn.GELU(),
            nn.Linear(512, 256),  # 预测像素
        )
```
