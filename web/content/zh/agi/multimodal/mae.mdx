# MAE 掩码自编码器

MAE (He et al., 2021) 通过重建随机掩码的块来预训练 ViT。

---

## 训练

```python
def mae_loss(model, images, mask_ratio=0.75):
    B, C, H, W = images.shape

    # 分块
    patches = patchify(images)  # (B, N, patch_dim)

    # 随机掩码
    num_masked = int(N * mask_ratio)
    noise = torch.rand(B, N)
    ids_shuffle = torch.argsort(noise, dim=-1)
    ids_restore = torch.argsort(ids_shuffle, dim=-1)
    ids_keep = ids_shuffle[:, :N - num_masked]

    # 保留可见块
    patches_keep = patches.gather(1, ids_keep.unsqueeze(-1).expand_as(patches))

    # 仅编码可见块
    visible_emb = model.encoder(patches_keep)

    # 解码器重建所有块（带掩码 token）
    pred = model.decoder(visible_emb, ids_restore)  # (B, N, patch_dim)

    # 仅在掩码块上计算损失
    loss = F.mse_loss(pred[:, N - num_masked:], patches[:, N - num_masked:])

    return loss
```

---

## 掩码为何有效

| 设计选择 | 原理 |
|----------|------|
| **高掩码率（75%）** | 强制全局理解 |
| **解码器重建像素** | 低级特征，非语义 |
| **微调时不掩码** | 标准监督训练 |
| **ViT-Large 效果好** | 更多容量有帮助 |
