## 神经渲染

```python
class NeuralRenderer(nn.Module):
    def __init__(self):
        self.decoder = nn.Sequential(
            nn.Linear(feature_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 3),
            nn.Sigmoid()
        )
    
    def forward(self, features, coords):
        # 从特征预测颜色
        return self.decoder(features[coords])
```
