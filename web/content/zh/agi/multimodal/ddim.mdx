# DDIM

非马尔可夫加速采样。

---

## vs DDPM

| 特性 | DDPM | DDIM |
|------|------|------|
| 采样步数 | 1000+ | 50-100 |
| 采样时间 | 分钟 | 秒 |
| 轨迹 | 随机 | 确定性 |
| 质量 | 最高 | 相近 |

---

## 马尔可夫 vs 非马尔可夫

```python
# DDPM (马尔可夫): q(x_{t-1} | x_t, x_0) 是高斯分布
# DDIM: q(x_{t-1} | x_t, x_0) 可以是任意分布

def ddim_step(x_t, t, t_prev, noise_pred, eta=0.0):
    """
    eta 控制随机性:
    - eta=0: 确定性 (DDIM)
    - eta=1: 等价于 DDPM
    """
    alpha = sqrt(alpha_bar[t])
    alpha_prev = sqrt(alpha_bar[t_prev])
    
    # 预测 x_0
    x_0_pred = (x_t - sqrt(1-alpha_bar[t]) * noise_pred) / alpha
    
    # 指向 x_t 的方向
    dir_xt = sqrt(1 - alpha_bar[t_prev] - eta * sigma**2) * noise_pred
    
    # 随机噪声 (eta=0 时为 0)
    sigma = eta * sqrt(1 - alpha_bar[t_prev])
    
    x_{t_prev} = alpha_prev * x_0_pred + dir_xt + sigma * torch.randn_like(x_t)
    
    return x_{t_prev}
```

---

## eta 参数

| eta | 采样方式 | 多样性 |
|-----|----------|--------|
| 0.0 | 确定性 | 较低 |
| 0.5 | 半随机 | 平衡 |
| 1.0 | 完全随机 (DDPM) | 较高 |
