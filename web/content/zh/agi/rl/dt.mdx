# 决策Transformer

用 GPT 风格的模型做 RL。

---

## 核心思想

```python
class DecisionTransformer(nn.Module):
    def __init__(self, state_dim, action_dim, hidden=256):
        super().__init__()
        self.model = GPT(
            embed_dim=hidden,
            num_heads=8,
            num_layers=6
        )
        self.state_embed = nn.Linear(state_dim, hidden)
        self.action_embed = nn.Embedding(action_dim, hidden)
        self.return_embed = nn.Linear(1, hidden)
        self.timestep_embed = nn.Embedding(100, hidden)
    
    def forward(self, states, actions, returns_to_go, timesteps):
        # 嵌入
        s_emb = self.state_embed(states)
        a_emb = self.action_embed(actions)
        r_emb = self.return_embed(returns_to_go)
        t_emb = self.timestep_embed(timesteps)
        
        # 序列建模
        x = s_emb + a_emb + r_emb + t_emb
        return self.model(x)
```

## 轨迹建模

```
R_1, s_1, a_1, R_2, s_2, a_2, ... → 预测 a_t
```
