# MuZero

通过学习模型进行规划。

---

## 核心思想

学习一个可用于规划的隐模型：

```python
class MuZero:
    def __init__(self):
        self.representation = R()  # 编码状态
        self.dynamics = D()       # 隐模型
        self.prediction = P()     # 价值/策略头
    
    def forward(self, obs):
        # 编码观察
        s = self.representation(obs)
        
        # MCTS 规划
        action = self.mcts(s)
        return action
    
    def train(self, batch):
        s_0 = self.representation(batch.obs)
        
        # 隐状态转移
        s_1 = self.dynamics(s_0, batch.action)
        
        # 预测
        policy, value = self.prediction(s_1)
        
        # 损失
        loss = policy_loss(policy, batch.policy) + \
               value_loss(value, batch.value) + \
               reconstruction_loss(s_0, batch.obs)
```

## MCTS 规划

```python
def mcts(root_s, num_simulations=50):
    for _ in range(num_simulations):
        s = root_s
        node = tree[root_s]
        
        while not node.is_leaf():
            # 选择
            action = node.uct_child()
            s, node = expand(s, action)
        
        # 评估
        v = evaluate(s)  # 通过预测网络
        
        # 回传
        node.backup(v)
    
    return tree[root_s].best_action()
```
