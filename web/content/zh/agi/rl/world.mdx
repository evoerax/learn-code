# World Models

压缩环境表示用于规划和想象。

---

## 架构

```
感知 → 编码 → 隐藏状态 h
              ↓
         世界模型 (MDN-RNN)
              ↓
         控制器 C → 动作
```

---

## VAE + MDN-RNN

```python
class WorldModel(nn.Module):
    def __init__(self):
        self.encoder = VAE()  # 压缩图像
        self.rnn = MDNGRU()  # 预测下一状态
    
    def forward(self, z, a, hidden):
        # 隐状态 + 动作 → 下一隐状态
        h_next = self.rnn(z, a, hidden)
        return h_next
```

def train_world_model(videos):

## 控制器

```python
class Controller(nn.Module):
    def __init__(self, z_dim, h_dim, action_dim):
        self.net = nn.Sequential(
            nn.Linear(z_dim + h_dim, 64),
            nn.ReLU(),
            nn.Linear(64, action_dim)
        )
    
    def forward(self, z, h):
        return self.net(torch.cat([z, h], dim=-1))
```
