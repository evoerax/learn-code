# 矩阵指数

与微分方程和扩散模型相关。

---

## 定义

```python
# 矩阵指数: exp(A) = I + A + A^2/2! + A^3/3! + ...
def matrix_exp(A, n_terms=20):
    result = torch.eye(A.shape[0])
    A_power = A.clone()
    factorial = 1
    for i in range(1, n_terms):
        factorial *= i
        result = result + A_power / factorial
        A_power = A_power @ A
    return result
```

---

## 与微分方程

```python
# dx/dt = Ax 的解
# x(t) = exp(At) * x(0)

# 离散化
# x_{t+1} = exp(A * dt) * x_t
```

## 在扩散模型中

```python
# 矩阵指数用于某些 SDE 求解器
# dX = f(X,t)dt + g(t)dW
# 解: X(t) = exp(∫f dt) * X(0)
```

---

## 计算技巧

| 方法 | 复杂度 | 精度 |
|------|--------|------|
| 级数展开 | O(n^3 k) | 中 |
| Padé 近似 | O(n^3) | 高 |
| 特征分解 | O(n^3) | 高 |
| Krylov 子空间 | O(n^2 k) | 可控 |

---

## PyTorch 实现

```python
# 近似计算
A_exp = torch.matrix_exp(A)

# 谱分解
eigenvalues, eigenvectors = torch.linalg.eig(A)
A_exp = eigenvectors @ torch.diag(torch.exp(eigenvalues)) @ eigenvectors.inv()
```
