# 核方法与 SVM

核函数将线性分类器扩展到非线性。

---

## 核函数

```python
def linear_kernel(x, y):
    return x @ y.T

def polynomial_kernel(x, y, d=3):
    return (x @ y.T + 1) ** d

def rbf_kernel(x, y, sigma=1.0):
    # 高斯核
    dist = torch.cdist(x, y)
    return torch.exp(-dist ** 2 / (2 * sigma ** 2))
```

## SVM

```python
class SVM:
    def __init__(self, kernel='rbf'):
        self.kernel = kernel
    
    def fit(self, X, y):
        # 求解对偶问题
        # min 0.5 * α^T Q α - sum(α)
        # s.t. y^T α = 0, 0 <= α <= C
        
        n = X.shape[0]
        K = self.kernel_matrix(X)
        Q = torch.outer(y, y) * K
        
        # SMO 算法求解
        alphas = solve_smo(Q, y)
        
        # 支持向量
        sv = alphas > 1e-5
        self.sv_x = X[sv]
        self.sv_y = y[sv]
        self.alphas = alphas[sv]
```
