# AGI 技术栈总览

本领域从 **2026 年技术视角** 出发，系统化梳理从 **Transformer、MoE** 到长序列、推理、多模态与具身智能的关键路线；适合作为「通用人工智能（AGI）」探索的补充阅读，与站点内 **[AI 模型](/zh/ais/overview)** 时间线互为参照。

## 两条主线怎么读

若用一句话概括：**Transformer 给了模型更强的序列「记忆与关联」能力，MoE 在算力约束下把「规模」做上去**；而下面这些方向则在补上 **推理、长效记忆、与物理世界交互** 等能力。技术栈可分为：

1. **演进路线图**：[演进路线图](/zh/agi/roadmap) — 2017–2026 的阶段性叙事。  
2. **四大支柱**：[技术支柱](/zh/agi/pillars) — 长序列、系统 2、多模态、外部知识与长上下文。  
3. **架构全景**：[架构全景](/zh/agi/landscape) — 除 Transformer / MoE 外的 SSM、线性注意力、混合块、DLM、多模态统一与工程技巧。  
4. **清单**：[2026 技术栈清单](/zh/agi/stack) — 算力、架构、数据、推理、安全等维度速查。

## 技术演进对比（速览）

| 技术阶段 | 代表技术 | 解决的核心问题 |
| :--- | :--- | :--- |
| **感知阶段** | Transformer, BERT, ViT | 跨模态特征提取与关联 |
| **规模化阶段** | MoE (Mixture of Experts) | 在算力有限下推高有效参数量与容量 |
| **长序列阶段** | SSM (Mamba 等), FlashAttention, 线性/混合注意力 | 长文档、长视频与 KV 效率 |
| **逻辑阶段** | RLHF / DPO, CoT, MCTS, 推理时计算 | 规划、验证与「慢思考」 |
| **具身阶段** | World Models, JEPA, VLA | 物理常识与与真实世界交互 |

## 2026 视角下别漏掉：世界模型

**World Models（世界模型）** 强调在表征中内化 **因果与物理规律**（例如物体运动、接触与后果），而不仅拟合文本统计。它与视频/交互数据、预测式表征（如 JEPA 类思路）以及机器人 **VLA（视觉-语言-动作）** 路线高度相关，是「从数字到物理」转折上的重要概念锚点。
