# AGI 技术演进路线图（2017–2026）

本节用**叙事化阶段**串起主流工业与研究趋势；具体名词与变体见 [架构全景](/zh/agi/landscape)。

## 1. 奠基：密集型 Transformer（约 2017–2022）

- **核心**：自注意力、Encoder-Decoder / Decoder-only / Encoder-only 等范式；预训练 + 扩展定律（Scaling Laws）推动模型从亿级向更高规模跃迁。  
- **代表方向**：Vanilla Transformer、GPT 系列、BERT、早期大稠密 LM（如 GPT-3、部分稠密基座）。

## 2. 效率与规模：MoE（约 2023–2024）

- **核心**：稀疏激活 —— 通过 **Router** 在每步或每层只激活部分专家（FFN），以相对可控的推理算力承载更大总参数量。  
- **工程要点**：专家并行、负载均衡、通信与显存优化。  
- **产业认知**：MoE 逐渐成为大模型「标配」选项之一，与稠密路线并存。

## 3. 推理与「系统 2」（约 2024–2025）

- **核心**：不仅做「下一个词」，而是在回答前分配 **测试时计算（Test-time / Inference-time Compute）** —— 搜索、反思、纠错与多步验证。  
- **相关技术栈**：RLHF / RLAIF / DPO 等对齐与偏好学习；**显式或可学习的 CoT**；树搜索（如 MCTS 系思路）与自我对弈式筛选；产品质量上的「推理专用」变体。  
- **比喻**：从快速的 System 1「接话」走向可读的 System 2「推演」。

## 4. 当前前沿：物理常识与具身（约 2025–2026）

- **世界模型 / 预测式表征**：通过视频、交互或仿真环境中的 **预测下一状态/表征**，内化空间、时间与因果。  
- **长序列与混合架构**：SSM（如 Mamba 系）与 Transformer 注意力 **交错**，在百万级以上上下文与成本之间找平衡。  
- **具身智能**：多模态大模型作为「脑」，**VLA** 将自然语言与视觉观测映射为低层动作或技能策略；工业与消费级机器人、自主 Agent 共享部分技术栈。

---

**小结**：若搭建个人知识库，建议突出两个转折 —— **从概率预测到可验证推理**，以及 **从纯文本世界到物理与具身世界**。
