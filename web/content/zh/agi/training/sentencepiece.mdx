# SentencePiece

SentencePiece (Kudo & Richardson, 2018) 是 Google 提出的无监督分词器，被众多 LLM 采用。

---

## 核心特性

| 特性 | 优势 |
|------|------|
| **语言无关** | 直接处理原始文本，无需分词 |
| **子词正则化** | 训练时支持多种分词结果 |
| **字节级 BPE** | 支持任意 Unicode 字符 |
| **自定义词表** | 可在领域数据上训练 |

---

## Unigram 语言模型

SentencePiece 使用 Unigram 模型，为每个子词分配概率：

```python
def train_sentencepiece(corpus, vocab_size=32000):
    # 初始化所有可能的子词
    tokens = generate_initial_subwords(corpus)

    # EM 算法优化 token 概率
    for _ in range(10):
        # E-step: 计算子词计数
        for sentence in corpus:
            segmentations = viterbi_segment(sentence, tokens)
            for seg in segmentations:
                for token in seg:
                    token_counts[token] += 1

        # M-step: 更新概率
        total = sum(token_counts.values())
        for token in tokens:
            tokens[token].prob = token_counts[token] / total

    # 裁剪到目标词表大小
    while len(tokens) > vocab_size:
        # 移除最低概率 token，合并到其他分词中
        tokens = prune_and_remerge(tokens)

    return tokens
```

---

## vs BPE

| 方面 | BPE | SentencePiece |
|------|-----|---------------|
| **分词方式** | 确定性 | 多假设 |
| **正则化** | 无 | 有（子词正则化） |
| **Unicode** | 需 UTF-8 | 字节级 |
| **使用模型** | GPT-2, GPT-4 | Llama, T5, mT5 |
