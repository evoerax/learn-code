# 反向传播

神经网络训练的核心算法。

---

## 计算图

```
Loss
  ↑
  | (梯度)
dL/dy
  ↑
  | (反向)
ReLU ← Linear ← ...
```

---

## 链式法则

```python
# dL/dx = dL/dy * dy/dx

# 梯度传播
grad_output = upstream_grad

# Linear 反向
grad_input = weight @ grad_output
grad_weight = grad_output @ input.T
grad_bias = grad_output.sum(dim=0)

# ReLU 反向
grad_input = grad_output * (input > 0)
```

---

## 自动微分

```python
# PyTorch 自动计算
x = torch.randn(10, requires_grad=True)
y = x ** 2 + 2 * x
y.sum().backward()

print(x.grad)  # d(y.sum())/dx = 2x + 2
```

---

## 梯度消失/爆炸

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 梯度消失 | 链式法则连乘 | 残差, LSTM |
| 梯度爆炸 | 链式法则连乘 | 梯度剪切, 归一化 |

---

## 梯度剪切

```python
# 防止梯度爆炸
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
```
