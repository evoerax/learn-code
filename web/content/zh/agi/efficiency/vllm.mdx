# vLLM

PagedAttention 实现的高效推理。

---

## 核心特性

| 特性 | 说明 |
|------|------|
| PagedAttention | KV Cache 分页管理，减少内存碎片 |
| 连续批处理 | 动态批次，最大化 GPU 利用率 |
| 异步抢占 | OOM 时优雅处理 |
| 量化支持 | INT8/FP16/BF16 |
| 张量并行 | 多 GPU 推理 |
| 流式输出 | Token 级输出 |

---

## 使用方式

```python
from vllm import LLM, SamplingParams

llm = LLM(model="meta-llama/Llama-2-70b-hf",
          tensor_parallel_size=4,
          gpu_memory_utilization=0.9)

sampling_params = SamplingParams(
    temperature=0.8,
    top_p=0.95,
    max_tokens=256
)

outputs = llm.generate(prompts, sampling_params)
```

---

## 架构

```python
# vLLM 内部组件

class LLMEp:
    def __init__(self):
        self.scheduler = Scheduler()      # 连续批处理
        self.model_runner = ModelRunner()  # PagedAttention
        self.kv_cache = KVCacheManager()   # 虚拟内存管理

    def forward(self, seq_group):
        # 1. 调度序列
        scheduled = self.scheduler.schedule(seq_group)

        # 2. 使用 PagedAttention 执行模型
        output = self.model_runner(scheduled)

        # 3. 更新 KV cache
        self.kv_cache.update(scheduled)

        return output
```
