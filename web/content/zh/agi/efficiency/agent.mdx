# Agent 框架

构建自主执行任务的 LLM 系统。

---

## Agent 循环

```
观察 → 思考 → 行动 → 观察 → ...
```

```python
class Agent:
    def __init__(self, llm, tools, memory):
        self.llm = llm
        self.tools = tools
        self.memory = memory
    
    def step(self, observation):
        # 思考：决定行动
        plan = self.llm.think(
            prompt=f"Observation: {observation}\nHistory: {self.memory}",
            tools=self.tools
        )
        
        # 执行
        if plan.action == "tool":
            result = self.execute_tool(plan.tool, plan.args)
            self.memory.add(observation, plan, result)
            return result
        else:
            return plan.response
```
