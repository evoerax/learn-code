# 人机交互（HCI）

研究人怎样与计算系统**自然、高效、可信地**协作；常与设计、认知科学、社会科学交叉。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **交互与可用性** | 任务分析、原型、评估方法 |
| **多模态与 XR** | 语音、手势、AR/VR、空间计算 |
| **协同与群体** | CSCW、协同编辑、意识与权限 |
| **可视化** | 信息可视化（与 [图形学](/zh/noaics/graphics) 交界） |
| **可信与无障碍** | 可解释体验、可访问性、公平与伦理交互 |

---

## 与 AI 的交叉

大模型产品化后，大量难题落在**意图、可控性、反馈、协作流程与信任**，常被误归为“模型问题”，本质是 **HCI 问题**。ACM [PACM HCI](https://dl.acm.org/journal/pacmhci) 等持续出版物仍是该方向的主阵地。
