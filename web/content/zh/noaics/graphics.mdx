# 计算机图形学与可视化

研究「看见」与**生成/处理**几何与图像内容；与游戏、影视、CAD、数字孪生、机器人仿真强相关。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **建模与渲染** | 光栅/光线追踪、材质、实时管线 |
| **几何** | 网格、体数据、曲面与拓扑 |
| **动画与模拟** | 物理、角色、流体 |
| **科学 / 信息可视化** | 标量场、图与网络、沉浸式分析 |
| **XR 与空间计算** | 与 [HCI](/zh/noaics/hci)、机器人感知交界 |

---

## 与 AI 的交叉

**神经渲染、生成式 3D** 与模型侧强相关；**经典图形学**（采样、重建、几何算法）仍是独立会议与课程线。存在感常被“生成式内容”盖过，但**工业可视化与数字孪生**需求仍大。
