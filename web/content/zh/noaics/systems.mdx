# 计算机系统

偏底层与性能，回答「机器怎样跑得**更快、更稳、更省**」。与 [NOAICS 总览](/zh/noaics/overview) 中的地图一致，本页展开**系统方向**的常见子题与和 AI 的交界。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **操作系统** | 调度、虚拟内存、文件系统接口、虚拟化、容器与内核交互 |
| **体系结构** | ISA、CPU/GPU/加速器、缓存与内存层次、功耗与可靠性 |
| **编译与运行时** | 优化、JIT/AOT、LLVM 类中间表示、语言实现 |
| **并行与分布式** | 一致性、容错、消息传递与共享内存、调度与资源管理 |
| **存储** | 文件系统、块存储、NVMe、近数据计算 |
| **HPC** | 大规模并行、互连、数值库与性能可移植性 |

---

## 与 AI 的交叉（边界说明）

「AI Infra」里大量工作（训练调度、推理服务、显存与显存碎片、分布式 checkpoint）**本质仍是系统问题**；本站把**模型与发布叙事**放在 [AI 模型](/zh/ais/overview)，**框架与 serving 栈**放在 [AI 软件](/zh/software/overview)，本页则保留**计科传统意义上的系统**表述。
