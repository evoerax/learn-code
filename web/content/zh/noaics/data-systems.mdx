# 数据库与数据管理

研究数据怎么存、怎么查、怎样在规模下保持**正确性、成本与可运维性**可控。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **关系与 SQL** | 查询优化、索引、事务与隔离级别 |
| **NoSQL / NewSQL** | 伸缩模型、CAP 权衡、专用存储引擎 |
| **分布式数据库** | 复制、分片、共识与故障转移 |
| **分析型与 Lakehouse** | 列存、物化视图、元数据与治理 |
| **流式** | Exactly-once、窗口、与 Kafka/Flink 类系统的语义 |
| **检索与特化索引** | 全文、向量（向量常作为**应用特性**与 AI 同屏出现，但引擎与存储仍是数据方向） |

---

## 与 AI 的交叉

**向量检索、特征存储**多与模型应用同屏；**存储引擎、日志结构、查询优化、容灾**仍是经典数据库研究。本站 [数据集](/zh/datasets/overview) 偏**语料与基准**；本页偏**数据系统**本身。
