# 机器人、嵌入式与信息物理系统（CPS）

**独立大方向**：与 AI 重叠（感知、规划、学习控制），但**实时性、安全关键、机电集成**仍是计算机系/自动化系长期主线。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **嵌入式与实时** | 调度可预测性、WCET、RTOS、与硬件协同设计 |
| **传感与估计** | 滤波、融合、状态估计（与 [控制](/zh/hardware/basics/control) 呼应） |
| **机器人系统** | 机构集成、运动规划、多机协同 |
| **无人系统** | 空天地、法规与任务约束 |
| **CPS** | 计算与物理过程耦合、安全与可靠性 |
