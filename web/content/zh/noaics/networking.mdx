# 计算机网络

研究数据怎么传、系统怎么连起来；云计算、物联网、移动互联网都依赖这一层。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **协议与性能** | 可靠传输、拥塞控制、QUIC/TCP、应用层语义 |
| **数据中心网络** | 拓扑、负载均衡、RDMA、低延迟 RPC |
| **云与边缘** | CDN、边缘节点、5G/6G 接入与回传 |
| **无线与移动** | Wi‑Fi、蜂窝、自组网、大规模接入 |
| **测量与优化** | 流量工程、主动/被动测量、可编程网络 |

---

## 与 AI 的交叉

大模型训练与推理对**带宽、延迟、多机互联**极其敏感；**分布式训练网络**常出现在系统与网络交叉会议。叙事上仍区分：**网络作为独立学科**（协议、拓扑、路由）与 **为 ML 服务的网络工程**（后者多被包装进 infra 岗位）。
