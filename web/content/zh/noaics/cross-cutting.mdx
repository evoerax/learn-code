# 交叉与应用计算

用计算方法解决**其它学科或行业**中的问题：偏应用，但**算法、数据、系统与领域知识**缺一不可。

---

## 典型领域（举例）

| 领域 | 关注点 |
| :--- | :--- |
| **计算生物学 / 生物信息** | 序列、结构、进化、高通量数据处理 |
| **计算医学与健康信息学** | 临床数据、影像、隐私与监管 |
| **计算金融** | 高频与风控、市场微观结构（与系统/数学交叉） |
| **智慧城市与交通** | 仿真、优化、多源数据融合 |
| **工业软件与数字孪生** | CAD/CAE 流水线、PLM、与物联网/CPS 衔接 |

---

## 与「纯 AI」的边界

许多应用会用到 **ML**，但**问题定义、数据采集、评测与落地**仍是**领域 + 系统**问题；本版块强调**交叉方向作为独立招生与科研线**，而非「一律归入 AI」。
