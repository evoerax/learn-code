# 网络安全与信息安全

研究如何**防攻击、控权限、保机密与完整性**，并满足合规与审计要求。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **密码学** | 协议设计、后量子、同态与多方计算（与应用场景强相关） |
| **系统安全** | 内核、沙箱、TEE、硬件安全模块 |
| **软件与供应链** | 依赖、SBOM、构建可重现性、漏洞利用与缓解 |
| **Web / 云安全** | 身份、认证授权、多租户隔离、配置错误 |
| **隐私** | 差分隐私、匿名化、法规与工程落地 |

---

## 2026 增量叙事

公开讨论中常出现：**机器身份与非人主体**、**AI Agent 权限与治理**、**IaC 与生成代码的供应链风险**。可对照 [Cloud Security Alliance](https://cloudsecurityalliance.org/) 等来源。安全与 AI **强交叉**，但**安全作为独立学科**仍是大类（学会目录、学位方向）。
