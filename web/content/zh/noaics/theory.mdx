# 理论计算机科学

偏数学与极限，回答「**能算什么**、**要多长时间、多少资源**」；是计科最「学院内核」之一。

---

## 子方向

| 子题 | 典型问题 |
| :--- | :--- |
| **算法设计与分析** | 近似、在线、随机化、并行 |
| **计算复杂性** | P、NP、电路下界、通信复杂性 |
| **离散结构** | 图论、组合、编码 |
| **形式化方法** | 模型检测、定理证明、程序语义 |
| **自动机与逻辑** | 与验证、数据库理论、密码学基础有交叉 |

---

## 与 AI 的交叉

**机器学习理论**（泛化、优化 landscape）常独立成会议线；经典理论 TCS 仍与**密码学、量子计算、验证**等强绑定。本站 [AI 模型](/zh/ais/overview) 偏**发布与能力叙事**；本页偏**传统 CS 理论**表述。
