# T-SQL

| | |
|---|---|

| **当前版本** | SQL Server 2022 |

## 诞生背景

T-SQL（Transact-SQL）由 Microsoft 和 Sybase 于 1989 年联合开发，最初为 SQL Server 提供过程化扩展。Sybase 后来被 SAP 收购，而 Microsoft 继续独立发展 T-SQL，使其成为 SQL Server 的核心语言。

T-SQL 是微软技术栈的数据库基石，从中小企业到大型企业都有广泛应用。

## 设计初衷

- 在数据库层面实现复杂业务逻辑
- 提供完整的错误处理和事务控制
- 与 .NET 技术栈无缝集成

## 代码示例

```sql
-- T-SQL: SQL Server 的过程扩展
CREATE PROCEDURE sp_UpdateInventory
    @ProductID INT,
    @Quantity INT,
    @Result INT OUTPUT
AS
BEGIN
    SET NOCOUNT ON;

    BEGIN TRANSACTION;

    UPDATE Products
    SET Stock = Stock - @Quantity
    WHERE ProductID = @ProductID;

    IF @@ROWCOUNT = 0
    BEGIN
        SET @Result = 0;
        ROLLBACK TRANSACTION;
        RETURN;
    END

    SET @Result = 1;
    COMMIT TRANSACTION;
END;
```
