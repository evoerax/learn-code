# 1990s — 互联网改变一切

1990 年代将计算从商业和科学工具转变为人人可及的全球网络。万维网在一夜之间创造了对新语言的需求。Java 承诺代码可以在任何设备上运行。Python 和 Ruby 让编程变得更易上手。JavaScript 为网页带来了交互性。到 2000 年，为互联网编写代码的程序员已超过任何其他平台。

| | |
|---|---|

| **重要事件** | WWW 公开发布（1991）、Netscape IPO（1995）、Linux 内核 1.0（1994） |
| **里程碑** | Java 的"一次编写，随处运行"（1995） |
| **核心挑战** | 跨平台软件、动态 Web 内容、快速开发 |

## 核心主题

**Java 的平台承诺。** Sun 的"一次编写，随处运行"不只是一句口号——它是一个真实问题的解决方案。企业软件必须在 Windows、Unix 和大型机上运行，往往同时运行。JVM 抽象了硬件差异。Servlet 和后来的 Spring 使 Java 成为网上银行、电子商务和企业应用的骨干。

**脚本语言崛起。** Python、Ruby、PHP 和 Perl 主导了 Web 开发——不是因为它们比 C++ 更快，而是因为它们让开发者能够快速迭代。动态类型、交互式 Shell 和垃圾回收将开发者从内存管理中解放出来。这种权衡（运行时较慢）在瓶颈是人的时间而非 CPU 时间时是可以接受的。

**函数式编程向实践靠近。** Haskell 形式化了惰性求值、类型类和单子 I/O。它在很大程度上仍是学术性的，但影响了此后的每一种语言——Scala、F#、Swift、Kotlin、Rust 都融入了 Haskell 的思想。OCaml 架起了理论与实践之间的桥梁，成为编译器和形式验证工具的首选语言。

**开源改变了一切。** Linux（C）、Apache（C）、Python、Ruby、PHP——1990 年代最重要的软件都是免费开源的。商业软件模式面临存在主义挑战。语言本身也成了社区项目，而非企业产品。

## 展望未来

1990 年代结束时，互联网泡沫达到顶峰。2000 年代将经历崩溃、复苏，以及为大规模分布式系统、生产环境函数式编程和新兴移动平台而设计的新一波语言。
