# BCPL

> _花括号语言的先驱，为C语言的诞生播下了种子。_

## 诞生背景

BCPL（Basic Combined Programming Language，基本组合编程语言）由马丁·理查兹于1967年在剑桥大学设计。它诞生于理查兹的编译器编写项目，并迅速成为早期小型计算机系统编程的首选语言。

该语言被广泛用于施乐 Alto 操作系统早期版本以及剑桥和 MIT 的其他研究操作系统。最著名的是，肯·汤普森在贝尔实验室创建 B 语言时，以 BCPL 为灵感和起点，B 最终演变为 C。

## 设计理念

<Callout type="default" emoji="💡">
  "BCPL 被设计为一门简洁、富有表现力的系统编程语言，能够快速编译。"
</Callout>

核心设计特点：

- **花括号分隔符**：`{` 和 `}` 字符被 C 语言继承，成为编程语法的标志
- **单一类型系统**：BCPL 使用单一类型（机器字），类型检查交由运行时或程序员负责
- **表达式导向**：BCPL 中几乎一切都是表达式，都返回数值
- **极简语法**：不需要分号——语句通过空白符和上下文区分
- **指针即整数**：内存地址被当作整数处理，赋予程序员底层控制能力

## 代码示例

```bcpl
GET "LIBHDR"

LET START() = VALOF
{
    FOR I := 1 TO 10 DO
        WRITEF("你好，世界！*N")
    RESULTIS 0
}

LET FACTORIAL(N) = VALOF
{
    IF N <= 1 THEN RESULTIS 1
    RESULTIS N * FACTORIAL(N - 1)
}
```

## 影响与遗产

**灵感来源：** CPL（组合编程语言，1963）

**影响语言：** B、C、JavaScript、Java 及所有花括号语言

BCPL 对计算机历史最持久的贡献是其语法。1970年代 C 语言推广开来的花括号 `{` 和 `}` ——如今几乎所有现代语言都支持——起源于 BCPL。当肯·汤普森在1969年创建 B 语言时，他将 BCPL 精简到本质，后来丹尼斯·里奇在 B 的基础上创建了 C。

如果没有 BCPL，C 语言家族将有截然不同的语法，JavaScript、Java、C++ 以及无数其他语言也不会以现在的形式存在。
