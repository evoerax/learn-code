# B

> _那个被剥到极致的解释型语言，最终孕育出了强大的C语言。_

## 诞生背景

B 由肯·汤普森于 1969 年在贝尔实验室创建，最初用于新开发的 PDP-11 小型计算机。汤普森之前在 PDP-7 上使用 BCPL，想要在新硬件上拥有一门类似但更简单的语言。他将 BCPL 精简到本质，创造了 B。

名字本身是自然的演进：BCPL 在 B 之前，而 B 是字母表中的第二个字母——虽然有开玩笑说它代表 "Basic"，或者仅仅是 C 之前的字母。B 在贝尔实验室的早期 UNIX 开发中被广泛使用，尤其是用于编写 UNIX 操作系统本身及其工具。

## 设计理念

<Callout type="default" emoji="💡">
  "B 是被剥到极致的 BCPL，设计目的是适应 PDP-11 有限的内存。"
</Callout>

核心特点：

- **极简主义**：汤普森移除了大量特性，使其能适应紧张的内存限制
- **单一整数类型**：与 BCPL 一样，B 对所有数据使用单一类型
- **内存效率**：程序紧凑，适合 24KB 的 PDP-11
- **解释执行**：早期的 B 是解释型而非编译型
- **简洁语法**：继承了 BCPL 的表达式导向、花括号分隔的风格

## 代码示例

```b
/* B 语言 - 阶乘 */
main()
{
    auto i, r;
    i = 1;
    r = 1;
    while(i <= 10) {
        r = r * i;
        i = i + 1;
    }
    putchar(r);
}
```

## 影响与遗产

**灵感来源：** BCPL

**影响语言：** C、早期 UNIX 生态系统

B 的重要性主要在于它通向何方。丹尼斯·里奇在 B 的基础上演化出 C，在 1970 年代早期添加了结构体类型、更好的类型安全和编译执行能力。C 最终成为计算机历史上最重要的语言之一，用于操作系统、嵌入式系统和无数应用。

B 还直接影响了 UNIX 操作系统的开发。早期 UNIX 用 B 和汇编编写，而用 C 编写 UNIX、C 又在 UNIX 上开发的组合创造了一个自举生态系统，塑造了现代计算。
