## 诞生背景

ALGOL（算法语言）诞生于 1958 年瑞士苏黎世的一次计算机科学家会议。目标是创造一种能够在国际范围内描述算法的通用编程语言。

这门语言由委员会设计——美国 ACM 与欧洲德国信息学协会（GI）的合作。这使 ALGOL 成为第一门真正的国际化编程语言。

ALGOL 率先将**块结构**概念引入编程，允许代码以嵌套作用域组织，变量局部化。这一创新深刻影响了所有后续的过程式语言。

## 设计理念

<Callout type="default" emoji="💡">
  "ALGOL 的设计目标是让算法能够以一种独立于任何特定机器架构的方式表达。"
</Callout>

关键创新：
- **块结构**：代码以嵌套作用域组织，变量局部化
- **BNF 表示法**：成为标准的正式语法描述
- **嵌套函数**：函数可以在其他函数内部定义
- **正确的参数传递**：参数的值语义
- **结构化控制流**：if-then-else、while、for 循环

## 代码示例

```algol
BEGIN
  INTEGER PROCEDURE FACTORIAL(N);
    INTEGER N;
    BEGIN
      IF N <= 1 THEN
        FACTORIAL := 1
      ELSE
        FACTORIAL := N * FACTORIAL(N - 1);
    END;

  PRINT(FACTORIAL(10));
END
```

## 影响与遗产

**灵感来源：** FORTRAN、IPL

**影响语言：** ALGOL 60、Pascal、C、Ada、Simula，所有块结构语言

ALGOL 的影响是巨大的。每一种现代结构化编程语言都欠 ALGOL 一些东西：
- C 的语法和块结构
- Pascal 的整体设计
- Ada 的结构化编程方法

为 ALGOL 语法描述而开发的 BNF 表示法，成为描述编程语言语法的通用标准。
