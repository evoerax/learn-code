# 1940s — 计算机的诞生

1940 年代是可编程计算机的起源。在高级语言出现之前，程序员直接操作机器指令、拨动开关和打孔纸带。这个十年诞生了世界上最早的计算机——以及让它们更易于编程的最初尝试。

## 背景

| | |
|---|---|
| **时代特征** | 前编程语言时代，机器码时代 |
| **代表硬件** | ENIAC（1945）、Manchester Baby（1948）、EDVAC（1949） |
| **编程方式** | 接线面板布线、二进制编码、汇编助记符 |
| **核心挑战** | 让机器做任何有用的事情 |

<Callout type="info">
  1940 年代的编程是一项体力劳动。在 ENIAC 上，"编程"意味着重新连接 6000 个开关和 3000 根电缆。每解决一个新问题都需要重新配置硬件。
</Callout>

## 1940 年代的语言

| 语言 | 年份 | 创造者 | 意义 |
|------|------|--------|------|
| Plankalkül | 1943 | Konrad Zuse | 最早的高级语言设计（直到 1998 年才实现） |
| ENIAC 编码系统 | 1945 | Eckert & Mauchly 团队 | 最早的操作程序记号 |
| Short Code | 1949 | John Mauchly | 数字计算机上第一个解释执行的高级语言 |

## 核心主题

**硬件与软件浑然一体。** ENIAC 没有存储程序——指令和数据存放在不同的系统中。1945 年提出的存储程序概念（冯·诺依曼架构）改变了一切：程序可以作为数据存在，机器可以修改自己的指令。

**数学驱动设计。** Konrad Zuse 的 Plankalkül 旨在表达数学算法和国际象棋逻辑。尽管在 1940 年代从未实现，它已经预见了几十年后才成为主流的许多概念：赋值、子程序、数组操作，甚至类型系统。

**女性程序员编写了最早的程序。** ENIAC 的六位原始程序员——Jean Jennings Bartik、Frances Bilas Spence、Betty Holberton、Marlyn Wescoff Meltzer 等——都是女性数学家，她们在没有任何手册或语言的情况下，将数学方程转化为机器指令。

## 展望未来

1940 年代播下的种子在下个十年迅速萌芽。存储程序模型催生了 Fortran、LISP 和 COBOL——人类终于可以用接近数学和英语的方式编写程序，由机器负责翻译成二进制。
