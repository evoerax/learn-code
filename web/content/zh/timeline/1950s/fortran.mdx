## 诞生背景

FORTRAN（公式翻译器）由约翰·巴克斯和 IBM 团队于 1957 年创建。它是第一门成功商业化的高级编程语言，彻底改变了科学计算领域。

在 FORTRAN 出现之前，科学家们必须用汇编语言或机器码编写程序，这既耗时又容易出错。巴克斯着手创建一门让科学家能够直接在程序中表达数学公式的语言。

1957 年，第一个 FORTRAN 编译器为 IBM 704 计算机发布。据报道，它生成的代码效率几乎与手工编写的汇编代码相当，这证明了高级语言的可行性。

## 设计理念

<Callout type="default" emoji="💡">
  "我主要是想创造一种语言，让科学家能够更轻松地为计算机编程。"——约翰·巴克斯
</Callout>

FORTRAN 的首要目标是效率：程序员的编码效率和编译后代码的效率。这门语言专为数值计算和科学编程而设计。

关键设计决策：
- 语法接近数学符号
- 编译为高效机器码
- 支持数组运算和矩阵代数
- DO 循环用于迭代计算

## 代码示例

```fortran
      PROGRAM HELLO
      PRINT *, '你好，世界！'
      END PROGRAM HELLO

      SUBROUTINE MATRIX_MULT(A, B, C, N)
      REAL A(N,N), B(N,N), C(N,N)
      DO I = 1, N
        DO J = 1, N
          C(I,J) = 0.0
          DO K = 1, N
            C(I,J) = C(I,J) + A(I,K) * B(K,J)
          END DO
        END DO
      END DO
      END SUBROUTINE
```

## 影响与遗产

**灵感来源：** ENIAC Short Code、Speedcoding

**影响语言：** ALGOL、BASIC、C、Pascal，所有后续命令式语言

FORTRAN 确立了许多成为编程基础的概念：
- 有意义的变量名
- 算术表达式
- 子程序和函数
- 用于迭代的 DO 循环
- 数组和矩阵运算

尽管已有 65 年历史，FORTRAN 仍然是气候建模、天气预报和计算物理领域的首选语言。今天运行在许多超级计算机上的代码，最初是几十年前用 FORTRAN 编写的。
