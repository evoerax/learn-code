## Origin Story

2003年，Mitchel Resnick及其MIT媒体实验室的团队——延续Seymour Papert的Logo遗产——创建了Scratch，作为"如何让编程对每个人都可及"这一问题的现代化答案。

关键创新是"积木"——五颜六色的、可拼接的视觉元素，彻底消除了语法错误。孩子们不再打字写代码，而是拖放积木来创建程序。在线社区于2007年启动，让全世界的孩子能够分享他们的作品。

Scratch的设计理念是"低门槛、高天花板、宽墙壁"：容易上手，能做复杂项目，足够宽广以支持多种类型的创作。到2023年，超过1亿个Scratch项目已被来自200多个国家的用户分享。

## Design Philosophy

Scratch消除了挫败感。没有语法错误意味着没有关于缺少分号或括号不匹配的调试信息。程序通过直接操作创建——即时看到结果强化了动作和结果之间的联系。

社交维度同样重要。Scratch网站是一个创意社区，年轻人展示项目、互相改编作品并接受反馈。这反映了真正的程序员如何学习：通过协作和迭代。

## Code Example

```scratch filename="接水果游戏"
// 用markdown表示的视觉积木
// [当绿旗被点击]
// 重复直到 <分数 = 10>
//   移到 (随机位置)
//   说 [你好！] (1) 秒
//   将 [分数 v] 增加 (1)
// end
// 说 [你赢了！] (2) 秒
```

## Did You Know?

<Callout type="info" emoji="🤔">
  "Scratch"这个名字来自DJ使用的声音混合技术——scratching。在这种技术中，DJ将不同的声音混合在一起。同样，Scratch让你把不同的媒体混合在一起：图形、声音和脚本，组合成你自己的创作。
