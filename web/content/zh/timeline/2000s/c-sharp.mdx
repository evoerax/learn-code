## Origin Story

1990年代末，微软面临一个战略挑战。Java 以"编写一次，到处运行"的理念迅速崛起，Windows 需要一门能与之竞争的现代语言。2000年，Turbo Pascal 和 Delphi 的创造者 Anders Hejlsberg 发布了 C#（发音"C 升"），作为 .NET 计划的一部分。

C# 从零开始设计，由公共语言运行时（CLR）直接支持。它借鉴了 C++、Java 和 Delphi 的最佳思想，同时引入了属性、事件和 LINQ 等创新功能。

## Design Philosophy

C# 的设计目标是成为一门简单、现代、面向对象的语言。微软希望创造出兼具 C++ 的强大和 Visual Basic 的简洁的语言。结果是一门强调类型安全、垃圾回收和组件化开发的语言。

C# 的演进令人瞩目——从 1.0 版本的 Java-like 语言，到如今功能丰富的强大语言。每个版本都增加了重要功能：泛型 (2.0)、LINQ (3.0)、async/await (5.0)、record (9.0)、模式匹配 (8.0+)。

## When to Use

**Good fit:**
- 企业应用和 Web 服务
- 使用 Unity 开发游戏
- 使用 .NET MAUI 的跨平台桌面应用
- Windows 专用应用（WPF、WinForms）

**Not a good fit:**
- 需要手动内存控制的内核编程
- 启动时间至关重要的轻量级脚本
- 前端 Web 开发（改用 TypeScript）
