# 编程语言大全（按年代）

> 所有主流及重要编程语言，按年代排列，每种语言一句话介绍。
> 作为语言介绍页面的编写依据。

---

## 1940s

### Plankalkül
最早的高级编程语言概念（1943），从未被实现，但已包含赋值、循环、条件等核心要素，奠定了理论基础。

### ENIAC 编码系统
为 ENIAC 计算机解决弹道计算而生（1943），早期计算机编程系统之一，程序通过物理接线和开关配置。

### ENIAC Short Code
第一个运行于计算机的非机器码语言（1946），用解释器执行简单数学表达式。

### Short Code / Brief Code
为 UNIVAC I 计算机提供更易用的编程方式（1949），早期解释型语言的奠基之作。

---

## 1950s

### Autocode
1952 年英国曼彻斯特大学开发的早期高级语言，首批编译型语言之一。

### FORTRAN
IBM 1957 年发布的第一门成功的高级语言，Formula Translation，科学计算至今仍在使用。

### FLOW-MATIC
1955 年 Grace Hopper 团队开发，首个面向商业数据处理的英语风格语言，直接影响了 COBOL 的诞生。

### IPL（Information Processing Language）
1956 年，早期 AI 符号处理语言，第一个使用列表结构和递归的语言，对 LISP 影响深远。

### LISP
1958 年诞生的第一门函数式语言，括号语法，符号处理，AI 研究的奠基语言。

### ALGOL 58
1958 年的国际算法语言标准，现代结构化编程语言（C、Pascal）的共同祖先。

### COBOL
1959 年为商业数据处理而生，英文式语法，全球银行和政府系统的核心语言。

---

## 1960s

### ALGOL 60
ALGOL 58 的改进版，引入块结构和词法作用域，深刻影响了 Pascal 和 C。

### COBOL 61
COBOL 的标准化完善版本，奠定了商业数据处理语言的行业标准。

### BCPL
1967 年 Martin Richards 设计，C 语言的直接祖先，无类型系统，引入了花括号代码块风格。

### B
Dennis Ritchie 在贝尔实验室基于 BCPL 开发（1969），C 语言的直接前身，UNIX 早期用它编写。

### SNOBOL
1962 年贝尔实验室开发，早期字符串处理与模式匹配语言，影响了后来的文本处理方向。

### Simula
1967 年世界第一门面向对象语言，类和对象概念的发源地，影响了 C++、Java、Python。

### APL
1966 年，用特殊数学符号操作数组的语言，代码极度简洁，影响了 J、K 等后继数组语言。

### BASIC
1964 年 Dartmouth 设计的教学语言，降低编程门槛，微软以此起家的第一个商业产品。

### PL/I
IBM 1964 年推出的多范式语言，试图统一科学与商业计算，第一门支持并发和异常处理的语言。

### RPG
IBM 1964 年为报表生成设计的语言，至今仍运行于 IBM i（AS/400）系统。

### Logo
1968 年通过海龟绘图教孩子编程，编程教育启蒙语言，Seymour Papert 的教育理念载体。

### Forth
1968 年为天文望远镜控制而生，栈式语言，逆波兰表示法，嵌入式和航天控制系统经典。

---

## 1970s

### Pascal
1970 年 Niklaus Wirth 设计的教学语言，结构化编程典范，Delphi 的前身。

### C
1972 年诞生于贝尔实验室，UNIX 的实现语言，50 年来系统编程的绝对基础。

### Smalltalk
1972 年 Xerox PARC 发明，纯面向对象，消息传递机制，GUI 编程和 IDE 的先驱。

### Prolog
1972 年第一门逻辑编程语言，声明式，用规则和事实表达知识，AI 专家系统基础。

### SQL
1974 年基于关系代数设计，结构化查询语言，所有关系型数据库的通用接口。

### ML
1973 年，第一门带类型推断的函数式语言，OCaml 和 F# 的直接祖先。

### Scheme
1975 年的 LISP 方言，极简设计，Lambda 演算最纯粹的实现，MIT 经典教材《SICP》配套语言。

### sed
1974 年 Unix 文本流编辑工具语言，与 AWK 并列构成 Unix 文本处理传统的两大支柱。

### AWK
1977 年专为文本处理和数据提取设计，Unix 工具链核心成员，模式匹配语言。

### PostScript
1982 年 Adobe 发明的页面描述语言，出版、打印和图形系统的行业标准，PDF 的前身。

### Modula-2
1979 年 Niklaus Wirth 在 Pascal 基础上引入模块和协程，系统编程语言。

---

## 1980s

### Ada
1983 年成为 ANSI 标准（设计始于 1977），美国国防部委托，强类型高可靠性，用于航空航天和军事关键系统。

### C++
1983 年在 C 上加入面向对象，多范式语言，游戏引擎、浏览器、数据库的首选。

### Objective-C
1983 年 C + Smalltalk 消息机制，苹果生态主力语言直到 Swift 出现。

### Common Lisp
1984 年 ANSI 标准化的 Lisp 方言，统一了众多 Lisp 分支，至今仍是工业级 Lisp 的参考实现。

### MATLAB
1984 年 MathWorks 推出，矩阵运算和数值分析的商业语言，工程和科研标配。

### Eiffel
1985 年，面向对象设计的理论典范，发明了"契约式设计"（Design by Contract）。

### Erlang
1986 年爱立信为电信系统开发，天生并发、容错、热更新，WhatsApp 的技术基础。

### ABAP
1983 年 SAP 开发，企业资源规划（ERP）系统的核心语言，全球企业软件生态中使用最广的语言之一。

### VHDL
1983 年美国国防部发起，硬件描述语言，用于 FPGA 和 ASIC 数字电路设计。

### Perl
1987 年，强大的正则表达式和文本处理，"有不止一种方法做到"，运维脚本经典。

### Tcl
1988 年，Tool Command Language，嵌入式脚本和 Tk GUI 工具的经典组合。

### Bash
1989 年 GNU 项目发布，Unix/Linux 系统最主流的 Shell 脚本语言，运维和 CI/CD 的第一语言。

---

## 1990s

### Haskell
1990 年，纯函数式、惰性求值、强大的类型系统，学术界函数式编程的标准语言。

### Python
1991 年，简洁优雅、多范式，从 Web 到 AI 无处不在，当今最流行的语言之一。

### Visual Basic
1991 年微软推出，可视化 Windows 应用开发，极大降低了桌面应用开发门槛。

### Lua
1993 年巴西发明，极轻量嵌入式脚本语言，World of Warcraft、Roblox 的脚本引擎。

### R
1993 年，统计计算和数据可视化的专用语言，ggplot2 是数据可视化的行业标准。

### Delphi / Object Pascal
1995 年 Borland 推出，Pascal 的面向对象扩展，Windows 时代桌面开发的代表性语言。

### Java
1995 年 Sun 推出，JVM 跨平台，Write Once Run Anywhere，企业级应用支柱。

### JavaScript
1995 年 Brendan Eich 仅用 10 天设计，浏览器唯一原生语言，全栈生态最广。

### PHP
1995 年，服务端脚本语言，驱动全球 43% 的网站，WordPress 的运行基础。

### Ruby
1995 年，注重开发者幸福感的纯面向对象脚本语言，Rails 框架改变了 Web 开发范式。

### OCaml
1996 年，ML 的面向对象方言，强类型推断，Jane Street 等金融公司的生产语言。

### Wolfram Language（Mathematica）
1988 年发布（1990 年代广泛使用），符号计算和知识引擎语言，Wolfram Alpha 的运行基础。

### PL/SQL
Oracle 的 SQL 过程扩展，存储过程、触发器、游标，企业数据库开发的主流语言。

### T-SQL
微软和 Sybase 的 SQL 扩展，SQL Server 的专用过程语言，Windows 企业生态核心。

---

## 2000s

### C#
2000 年微软推出，.NET 生态核心，Anders Hejlsberg 设计，现代语言特性丰富，Unity 游戏引擎首选。

### ActionScript
2000 年 Macromedia 为 Flash 动画和应用提供编程能力，基于 ECMAScript，RIA 时代的 Web 语言。

### D
2001 年，C++ 的现代化替代方案，垃圾回收可选、编译速度快、元编程强大。

### Processing
2001 年 MIT 媒体实验室，为艺术家和设计师提供的可视化编程环境，Java 方言。

### Scratch
2003 年 MIT 媒体实验室，可视化积木编程，全球最广泛使用的青少年编程入门工具。

### Scala
2003 年，融合面向对象和函数式编程，运行于 JVM，Apache Spark 的实现语言。

### Groovy
2003 年，JVM 上的动态语言，Gradle 构建脚本和 Jenkins Pipeline 的专用语言。

### F#
2005 年微软出品，.NET 平台的函数式语言，数据科学和金融建模的实用选择。

### Haxe
2005 年，强类型跨平台编译语言，可编译为 JavaScript、Python、C++、Java 等多个目标，游戏开发常用。

### PowerShell
2006 年微软推出，基于 .NET 的 Shell 脚本语言，Windows 系统管理和 DevOps 工具。

### Clojure
2007 年，JVM 上的现代 Lisp，不可变数据结构，函数式并发编程。

### Nim
2008 年，编译为 C 的高级语言，Python 般的语法配合接近 C 的运行性能。

### Go
2009 年 Google 三位传奇工程师设计，极简语法、goroutine 并发，Docker/K8s 的语言。

### CoffeeScript
2009 年，编译为 JavaScript 的语法糖语言，TypeScript 出现前的主流替代方案。

---

## 2010s

### Rust
2010 年 Mozilla 开发，用所有权系统在编译期保证内存安全，C++ 的现代挑战者。

### Kotlin
2011 年 JetBrains 开发，简洁空安全，与 Java 完全互操作，Android 官方首选。

### Elixir
2011 年，在 Erlang VM 上运行的现代函数式语言，Phoenix 框架支撑百万并发。

### Dart
2011 年 Google 推出，Flutter 框架的专用语言，编译为原生或 JavaScript。

### TypeScript
2012 年微软推出，JavaScript 的静态类型超集，大型前端项目的工程标准。

### Julia
2012 年，为科学计算设计的高性能动态语言，JIT 编译，速度接近 C。

### Elm
2012 年，纯函数式前端语言，零运行时错误，The Elm Architecture 影响了 Redux。

### PureScript
2013 年，强类型函数式语言，编译为 JavaScript，Haskell 风格的前端开发。

### Hack
2014 年 Facebook 为 PHP 添加静态类型，HHVM 虚拟机上运行。

### Swift
2014 年 Apple 推出，安全现代，替代 Objective-C，iOS/macOS 开发首选。

### Crystal
2014 年，Ruby 语法 + 静态类型 + 编译执行，兼顾开发效率和运行速度。

### Solidity
2014 年以太坊开发，智能合约主流语言，语法类 JavaScript，DeFi 生态基础。

### ReScript / ReasonML
2016 年（ReasonML）/ 2020 年（ReScript），OCaml 进入前端工程的重要尝试，强类型 JavaScript 替代。

### Pony
2015 年，基于 Actor 模型的并发语言，用能力系统（Capabilities）消除数据竞争。

### Ballerina
2017 年 WSO2 发布，专为云服务与网络集成场景设计，内置并发和网络原语的语言。

### Cairo
2020 年 StarkWare 发布，支持零知识证明可验证计算的智能合约语言，ZK Rollup 生态核心。

---

## 2020s

### Zig
2016 年起（广泛认知于 2020），比 C 更简单安全的系统语言，与 C 完全互操作，Bun 和 Ghostty 用它构建。

### Move
2019 年 Facebook（Diem）设计，资源导向语言，防止双重花费，用于 Aptos 和 Sui 区块链。

### V（Vlang）
2019 年，简洁的系统编程语言，编译速度极快，无 GC，语法类 Go。

### Gleam
2021 年，Erlang VM 上的强类型函数式语言，语法简洁，适合高并发 Web 服务。

### Carbon
2022 年 Google 推出，设计为 C++ 的现代继承者，与 C++ 双向互操作，实验阶段。

### Hare
2022 年，极简系统语言，C 的精神继承者，无 GC、无运行时，专注于稳定和简洁。

### Mojo
2023 年 Modular 推出，Python 超集，专为 AI 硬件加速设计，声称在 AI 计算场景比 Python 快万倍。

### Roc
函数式语言，专注于极快的编译速度和友好的错误信息，Elm 的精神继承者，面向生产环境。
